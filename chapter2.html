2.1 任意角和弧度制

任意角的概念

角可以看作平面内一条射线绕着端点从一个位置旋转到另一个位置所形成的图形

  • 正角：按逆时针方向旋转形成的角
  • 负角：按顺时针方向旋转形成的角
  • 零角：射线不作任何旋转形成的角

弧度制

弧度制的定义

长度等于半径长的弧所对的圆心角叫做1弧度的角

符号：rad（通常省略不写）

弧度制与角度制的换算

1° = π/180 rad
1 rad = 180°/π
180° = π rad

常用特殊角的弧度

角度 0° 30° 45° 60° 90° 180° 270° 360°
弧度 0 π/6 π/4 π/3 π/2 π 3π/2 2π

2.2 三角函数

任意角的三角函数

设 α 是一个任意角，它的终边与单位圆交于点 P(x, y)，则：

正弦函数

sinα = y

几何意义：点 P 的纵坐标

余弦函数

cosα = x

几何意义：点 P 的横坐标

正切函数

tanα = y/x (x ≠ 0)

几何意义：点 P 的纵坐标与横坐标的比值

同角三角函数的基本关系

平方关系

sin²α + cos²α = 1

商数关系

tanα = sinα / cosα

倒数关系

1 + tan²α = 1/cos²α

诱导公式

π - α

sin(π - α) = sinα
cos(π - α) = -cosα

π + α

sin(π + α) = -sinα
cos(π + α) = -cosα

2π - α

sin(2π - α) = -sinα
cos(2π - α) = cosα

-α

sin(-α) = -sinα
cos(-α) = cosα

三角函数的图像与性质

正弦函数 y = sinx

定义域：R

值域：[-1, 1]

周期：2π

奇偶性：奇函数

单调性：[2kπ - π/2, 2kπ + π/2] 单调递增

最值：x = 2kπ + π/2 时取得最大值 1；x = 2kπ - π/2 时取得最小值 -1

余弦函数 y = cosx

定义域：R

值域：[-1, 1]

周期：2π

奇偶性：偶函数

单调性：[(2k-1)π, 2kπ] 单调递增

最值：x = 2kπ 时取得最大值 1；x = (2k+1)π 时取得最小值 -1

正切函数 y = tanx

定义域：{x | x ≠ kπ + π/2, k ∈ Z}

值域：R

周期：π

奇偶性：奇函数

单调性：在每个周期内单调递增

2.3 三角恒等变换

两角和与差的公式

正弦的和差公式

sin(α ± β) = sinα cosβ ± cosα sinβ

余弦的和差公式

cos(α ± β) = cosα cosβ ∓ sinα sinβ

注意符号：余弦公式中，和取减号，差取加号

正切的和差公式

tan(α ± β) = (tanα ± tanβ) / (1 ∓ tanα tanβ)

注意符号：正切公式中，分子与加减号一致，分母与加减号相反

二倍角公式

正弦的二倍角公式

sin2α = 2sinα cosα

余弦的二倍角公式

cos2α = cos²α - sin²α
cos2α = 2cos²α - 1
cos2α = 1 - 2sin²α

正切的二倍角公式

tan2α = 2tanα / (1 - tan²α)

辅助角公式

公式

asinα + bcosα = √(a² + b²) sin(α + φ)

其中 sinφ = b/√(a² + b²)，cosφ = a/√(a² + b²)

记忆口诀：同角在前，异角在后

应用

辅助角公式常用于求三角函数的最值和单调区间

asinα + bcosα 的最大值为 √(a² + b²)，最小值为 -√(a² + b²)

例题练习

例题 1：弧度制与角度制互化 【基础】

将 60° 化为弧度

解答：

60° = 60 × (π/180) rad

= π/3 rad

答案：π/3

例题 2：弧度制与角度制互化 【基础】

将 π/6 rad 化为角度

解答：

π/6 rad = (π/6) × (180°/π)

= 30°

答案：30°

例题 3：特殊角的三角函数值 【基础】

求 sin(π/6)、cos(π/6)、tan(π/6) 的值

解答：

π/6 = 30°

sin(π/6) = 1/2

cos(π/6) = √3/2

tan(π/6) = (1/2) / (√3/2) = 1/√3 = √3/3

答案：sin(π/6) = 1/2, cos(π/6) = √3/2, tan(π/6) = √3/3

例题 4：特殊角的三角函数值 【基础】

求 sin(π/4)、cos(π/4)、tan(π/4) 的值

解答：

π/4 = 45°

sin(π/4) = √2/2

cos(π/4) = √2/2

tan(π/4) = (√2/2) / (√2/2) = 1

答案：sin(π/4) = √2/2, cos(π/4) = √2/2, tan(π/4) = 1

例题 5：同角三角函数关系 【基础】

已知 sinα = 3/5，α 是第二象限角，求 cosα 和 tanα

解答：

根据平方关系：sin²α + cos²α = 1

cos²α = 1 - sin²α = 1 - (3/5)² = 1 - 9/25 = 16/25

cosα = ±4/5

由于 α 是第二象限角，cosα < 0

所以 cosα = -4/5

tanα = sinα / cosα = (3/5) / (-4/5) = -3/4

答案：cosα = -4/5, tanα = -3/4

例题 6：同角三角函数关系 【基础】

已知 cosα = -1/2，α ∈ (π/2, π)，求 sinα

解答：

根据平方关系：sin²α + cos²α = 1

sin²α = 1 - cos²α = 1 - (-1/2)² = 1 - 1/4 = 3/4

sinα = ±√3/2

由于 α ∈ (π/2, π)（第二象限），sinα > 0

所以 sinα = √3/2

答案：√3/2

例题 7：诱导公式 【基础】

计算：sin(5π/6)

解答：

5π/6 = π - π/6

根据诱导公式：sin(π - α) = sinα

sin(5π/6) = sin(π - π/6) = sin(π/6) = 1/2

答案：1/2

例题 8：诱导公式 【基础】

计算：cos(4π/3)

解答：

4π/3 = π + π/3

根据诱导公式：cos(π + α) = -cosα

cos(4π/3) = cos(π + π/3) = -cos(π/3) = -1/2

答案：-1/2

例题 9：两角和公式 【基础】

计算：sin75°

解答：

sin75° = sin(45° + 30°)

根据公式：sin(α + β) = sinα cosβ + cosα sinβ

= sin45°cos30° + cos45°sin30°

= (√2/2)(√3/2) + (√2/2)(1/2)

= (√6/4) + (√2/4)

= (√6 + √2)/4

答案：(√6 + √2)/4

例题 10：两角差公式 【基础】

计算：cos15°

解答：

cos15° = cos(45° - 30°)

根据公式：cos(α - β) = cosα cosβ + sinα sinβ

= cos45°cos30° + sin45°sin30°

= (√2/2)(√3/2) + (√2/2)(1/2)

= (√6 + √2)/4

答案：(√6 + √2)/4

例题 11：二倍角公式 【基础】

已知 sinα = 1/3，求 sin2α

解答：

根据二倍角公式：sin2α = 2sinα cosα

需要先求 cosα：

cos²α = 1 - sin²α = 1 - (1/3)² = 1 - 1/9 = 8/9

cosα = ±2√2/3

由于 sinα > 0，α 可能在第一或第二象限

答案：sin2α = 2sinα cosα = 2×(1/3)×(±2√2/3) = ±4√2/9

例题 12：二倍角公式 【基础】

计算：sin²(π/8)

解答：

根据余弦的二倍角公式：cos2α = 1 - 2sin²α

所以 sin²α = (1 - cos2α)/2

令 α = π/8：

sin²(π/8) = (1 - cos(π/4))/2

= (1 - √2/2)/2

= (2 - √2)/4

答案：(2 - √2)/4

例题 13：辅助角公式 【基础】

将 sinx + cosx 化为 Asin(x + φ) 的形式

解答：

sinx + cosx

提取公因式：= √(1² + 1²) × (sinx/√2 + cosx/√2)

= √2 × (sinx cos(π/4) + cosx sin(π/4))

= √2 sin(x + π/4)

答案：√2 sin(x + π/4)

例题 14：辅助角公式求最值 【基础】

求 f(x) = 3sinx + 4cosx 的最大值和最小值

解答：

f(x) = 3sinx + 4cosx

提取公因式：= √(3² + 4²) × (3/5 sinx + 4/5 cosx)

= 5 × (sinx cosφ + cosx sinφ)

= 5 sin(x + φ)

其中 cosφ = 3/5，sinφ = 4/5

f(x) = 5 sin(x + φ)

由于 -1 ≤ sin(x + φ) ≤ 1

所以 -5 ≤ f(x) ≤ 5

答案：最大值5，最小值-5

例题 15：2024年新高考I卷 【中档】

已知 sinα = 4/5，α ∈ (0, π/2)，则 tanα =

解答：

由于 α ∈ (0, π/2)（第一象限），cosα > 0

cos²α = 1 - sin²α = 1 - (4/5)² = 1 - 16/25 = 9/25

cosα = 3/5

tanα = sinα / cosα = (4/5) / (3/5) = 4/3

答案：4/3

例题 16：2023年全国甲卷 【中档】

计算：sin(π/12)

解答：

π/12 = 15°

sin(π/12) = sin(45° - 30°)

= sin45°cos30° - cos45°sin30°

= (√2/2)(√3/2) - (√2/2)(1/2)

= (√6 - √2)/4

答案：(√6 - √2)/4

例题 17：2022年新高考I卷 【中档】

已知 cos(π/6 + α) = 1/3，则 cos(π/6 - α) =

解答：

利用余弦差公式展开：cos(π/6 - α) = cos(π/6)cosα + sin(π/6)sinα

= (√3/2)cosα + (1/2)sinα


展开已知条件：cos(π/6 + α) = cos(π/6)cosα - sin(π/6)sinα

= (√3/2)cosα - (1/2)sinα = 1/3


两式相加：√3cosα = 1/3 + cos(π/6 - α)

这需要更多的信息

答案：需要更多信息

例题 18：2021年全国乙卷 【中档】

函数 f(x) = 2sinx + √3cosx 的最大值为

解答：

f(x) = 2sinx + √3cosx

提取公因式：= √(2² + (√3)²) × (2/√7 sinx + √3/√7 cosx)

这里计算有误，重新计算：

√(2² + (√3)²) = √(4 + 3) = √7

f(x) = √7 × (2/√7 sinx + √3/√7 cosx)

= √7 sin(x + φ)

最大值为 √7

答案：√7

例题 19：2020年全国I卷 【中档】

已知 sin(π - α) = 1/2，则 cosα =

解答：

根据诱导公式：sin(π - α) = sinα

所以 sinα = 1/2

cos²α = 1 - sin²α = 1 - (1/2)² = 1 - 1/4 = 3/4

cosα = ±√3/2

答案：±√3/2

例题 20：2019年全国I卷 【中档】

已知 tanα = 2，则 sin2α =

解答：

根据二倍角公式：sin2α = 2sinα cosα

由于 tanα = sinα/cosα = 2，所以 sinα = 2cosα

sin²α + cos²α = 4cos²α + cos²α = 5cos²α = 1

cos²α = 1/5，cosα = ±√5/5

sinα = 2cosα = ±2√5/5

sin2α = 2 × (±2√5/5) × (±√5/5) = 2 × 10/25 = 4/5

答案：4/5

例题 21：2018年全国II卷 【中档】

计算：cos²(π/12)

解答：

根据余弦的二倍角公式：cos2α = 2cos²α - 1

所以 cos²α = (1 + cos2α)/2

令 α = π/12：

cos²(π/12) = (1 + cos(π/6))/2

= (1 + √3/2)/2

= (2 + √3)/4

答案：(2 + √3)/4

例题 22：2017年全国I卷 【中档】

已知 sinα - cosα = 1/2，则 sinα cosα =

解答：

两边平方：(sinα - cosα)² = (1/2)²

sin²α - 2sinα cosα + cos²α = 1/4

(sin²α + cos²α) - 2sinα cosα = 1/4

1 - 2sinα cosα = 1/4

2sinα cosα = 3/4

sinα cosα = 3/8

答案：3/8

例题 23：三角函数图像 【提高】

函数 f(x) = 2sin(2x + π/3) 的最小正周期是

解答：

函数 f(x) = Asin(ωx + φ) 的周期 T = 2π/|ω|

ω = 2

T = 2π/2 = π

答案：π

例题 24：三角函数最值 【提高】

求函数 f(x) = sin²x + √3sinxcosx 的最大值

解答：

f(x) = sin²x + √3sinxcosx

= sin²x + √3/2 × 2sinxcosx

= (1 - cos2x)/2 + (√3/2)sin2x

= 1/2 - (1/2)cos2x + (√3/2)sin2x

= 1/2 + sin2x(√3/2) - cos2x(1/2)

= 1/2 + sin(2x - π/6)

由于 -1 ≤ sin(2x - π/6) ≤ 1

所以 -1/2 ≤ f(x) ≤ 3/2

答案：3/2

例题 25：三角函数综合 【提高】

已知 sinα = 3/5，α ∈ (π/2, π)，求 sin2α 和 cos2α

解答：

由于 α ∈ (π/2, π)（第二象限），cosα < 0

cos²α = 1 - sin²α = 1 - (3/5)² = 1 - 9/25 = 16/25

cosα = -4/5


求 sin2α：

sin2α = 2sinα cosα = 2 × (3/5) × (-4/5) = -24/25


求 cos2α：

cos2α = cos²α - sin²α = (-4/5)² - (3/5)² = 16/25 - 9/25 = 7/25

答案：sin2α = -24/25, cos2α = 7/25

例题 28：三角函数求值 【提高】

计算：sin15° cos15°

解答：

解法一：直接计算

sin15° cos15°

= (√6 - √2)/4 × (√6 + √2)/4

= (6 - 2)/16 = 4/16 = 1/4


解法二：利用二倍角公式

sin2α = 2sinα cosα

sin15° cos15° = sin30°/2 = (1/2)/2 = 1/4

答案：1/4

例题 29：三角函数求值 【提高】

计算：sin²10° + cos²10°

解答：

根据平方关系：sin²α + cos²α = 1

sin²10° + cos²10° = 1

答案：1

例题 30：三角函数综合 【提高】

已知 sinx + cosx = √2/2，求 sinx - cosx

解答：

设 a = sinx + cosx = √2/2

设 b = sinx - cosx


a² + b² = (sinx + cosx)² + (sinx - cosx)²

= sin²x + 2sinxcosx + cos²x + sin²x - 2sinxcosx + cos²x

= 2(sin²x + cos²x) = 2


(√2/2)² + b² = 2

1/2 + b² = 2

b² = 3/2

b = ±√6/2

答案：±√6/2
