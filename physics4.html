4.1 牛顿第一定律

内容

一切物体总保持匀速直线运动状态或静止状态，直到有外力迫使它改变这种状态为止

惯性

物体保持原来匀速直线运动状态或静止状态的性质叫做惯性

  • 质量是物体惯性大小的唯一量度
  • 惯性是物体的固有属性，与运动状态无关

4.2 牛顿第二定律

公式

F = ma

物体的加速度跟作用力成正比，跟物体的质量成反比

理解要点

  • 矢量性：加速度方向与合外力方向相同
  • 瞬时性：a与F同时产生、同时变化、同时消失
  • 独立性：每个力独立产生加速度

4.3 牛顿第三定律

内容

两个物体之间的作用力和反作用力总是大小相等，方向相反，作用在同一条直线上

特点

  • 作用力与反作用力作用在不同物体上
  • 同时产生、同时消失
  • 性质相同的力

例题练习

例题 1：【基础】牛顿第二定律基本应用

质量为2kg的物体，在10N的合外力作用下，加速度为多少？

解答：

根据牛顿第二定律 F = ma

a = F/m = 10/2 = 5m/s²

知识点标签：牛顿第二定律

答案：5m/s²

例题 2：【2024·全国卷】惯性概念 【基础】

在足球运动中，足球入网。关于足球的运动，下列说法正确的是？足球在飞行和触网时惯性是否变化？

解答：

（1）惯性只与质量有关，是物体的固有属性

（2）足球在飞行和触网时质量不变，则惯性不变

（3）惯性大小与运动状态无关

知识点标签：惯性、牛顿第一定律

答案：惯性不变

例题 3：【2023·全国卷】火箭发射 【中档】

我国多次成功使用"冷发射"技术发射长征十一号系列运载火箭。发射仓内的高压气体先将火箭竖直向上推出，火箭速度接近零时再点火飞向太空。从火箭开始运动到点火的过程中，下列说法正确的是？

解答：

（1）火箭受高压气体推力F和重力mg作用，向上加速

（2）当F = mg时，加速度为零，动能最大

（3）高压气体释放的能量不是全部转化为火箭的动能（有重力势能变化）

（4）高压气体的推力和空气阻力对火箭做功之和等于火箭动能的增加量

知识点标签：牛顿第二定律、动能定理

答案：D（推力和阻力做功之和等于动能增加量）

例题 4：【2024·新课标卷】小球加速度与弹簧压缩量 【高档】

在星球M上将一轻弹簧竖直固定在水平桌面上，把物体P轻放在弹簧上端，P由静止向下运动，物体的加速度a与弹簧的压缩量x间的关系如图中实线所示。在另一星球N上用完全一样的弹簧，改用物体Q完成同样的过程，其a-x关系如图中虚线所示。假设两星球均为质量均匀分布的球体。星球M的半径是星球N的3倍，则下列说法正确的是？

解答：

（1）由牛顿第二定律：mg - kx = ma → a = g - (k/m)x

（2）图像斜率为k/m，截距为g

（3）斜率分析：k/m₁ : k/m_Q = 1:4 → m₁ : m_Q = 4:1

（4）截距分析：g_M : g_N = 2:1

（5）由g = GM/R² = G·(4πρR³/3)/R² = 4πGρR/3

（6）得ρ_M : ρ_N = (g_M/R_M) : (g_N/R_N) = 1:1

知识点标签：牛顿第二定律、万有引力、天体质量与密度

答案：AC（密度相等，质量之比4:1）

例题 5：【2023·全国卷】运动员起跳 【中档】

运动员原地垂直起跳扣篮，离地后重心上升的最大高度为H。上升第一个H/4所用的时间为t₁，第四个H/4所用的时间为t₂。不计空气阻力，则t₁/t₂满足什么关系？

解答：

（1）逆向思维法：将上升过程视为自由落体的逆过程

（2）自由落体各段相等位移的时间比为：1 : (√2-1) : (√3-√2) : (2-√3)

（3）t₁对应上升第一个H/4（逆向为下落第四个H/4）

（4）t₂对应上升第四个H/4（逆向为下落第一个H/4）

（5）故t₁/t₂ = (2-√3) : 1 ≈ 0.268

（6）3 < t₁/t₂ < 4

知识点标签：匀变速直线运动、逆向思维

答案：3 < t₁/t₂ < 4（选项C）

例题 6：【2022·山东卷】连接体问题 【高档】

一不可伸长轻绳两端各连接一质量为m的小球，初始时整个系统静置于光滑水平桌面上，两球间的距离等于绳长L。一大小为F的水平恒力作用在轻绳的中点，方向与两球连线垂直。当两球运动至二者相距3L/5时，它们加速度的大小均为多少？

解答：

（1）设两球间距离为d，绳中张力为T

（2）对任意一个小球，由几何关系得：sinθ = (d/2)/(L/2) = d/L

（3）当d = 3L/5时，sinθ = 3/5

（4）对小球应用牛顿第二定律：2Tcosθ = ma

（5）对绳中点：F = 2Tsinθ

（6）联立得：a = F/m · cosθ/sinθ = 4F/(3m)

知识点标签：牛顿第二定律、连接体、力的分解

答案：4F/(3m)

例题 7：【2023·河北卷】小车与滑轮 【中档】

某同学用轻绳跨过光滑定滑轮，一端连接质量为m₁的物块，另一端连接质量为m₂的物块（m₂ > m₁），系统由静止释放。求两物块的加速度大小和轻绳的拉力。（重力加速度为g）

解答：

（1）对m₁：T - m₁g = m₁a（向上为正）

（2）对m₂：m₂g - T = m₂a（向下为正）

（3）两式相加：(m₂ - m₁)g = (m₁ + m₂)a

（4）加速度：a = (m₂ - m₁)g/(m₁ + m₂)

（5）拉力：T = m₁(g + a) = 2m₁m₂g/(m₁ + m₂)

知识点标签：牛顿第二定律、连接体

答案：a = (m₂ - m₁)g/(m₁ + m₂)，T = 2m₁m₂g/(m₁ + m₂)

例题 8：【2024·江西卷】小鱼跃出水面 【高档】

一条河流某处存在高度差，小鱼从低处向上跃出水面，冲到高处。以小鱼跃出水面处为坐标原点，x轴沿水平方向建立坐标系，小鱼的初速度为v₀，末速度v沿x轴正方向。在此过程中，小鱼可视为质点且只受重力作用。关于小鱼的竖直位置y与时间t的关系，下列图像可能正确的是？

解答：

（1）小鱼只受重力作用，做斜抛运动

（2）竖直方向：做竖直上抛运动

（3）y = v_y₀t - ½gt²（抛物线）

（4）由于末速度沿水平方向，说明在最高点

（5）y-t图像应为开口向下的抛物线

知识点标签：斜抛运动、运动分解

答案：抛物线形状图像

例题 9：【2023·新课标卷】理想气体循环 【高档】

一定量理想气体的循环由四个过程组成：12为绝热过程，23为等压过程，34为绝热过程，41为等容过程。上述四个过程是四冲程柴油机工作循环的主要过程。下列说法正确的是？

解答：

（1）12为绝热过程：此过程中气体体积减小，外界对气体做功

（2）根据热力学第一定律ΔU = Q + W，气体内能增加，故A正确

（3）23为等压过程：气体体积增大时温度增加，内能增大

（4）气体体积增大时，气体对外界做功W < 0

（5）由ΔU = Q + W知气体吸收热量，故B错误

（6）34为绝热过程：气体内能减少，故C错误

知识点标签：热力学第一定律、理想气体状态方程

答案：AD

例题 10：【2022·全国卷】雪地转椅 【高档】

雪地转椅是一种游乐项目，其中心传动装置带动转椅在雪地上滑动。传动装置有一高度可调的水平圆盘，可绕通过中心O点的竖直轴匀速转动。圆盘边缘A处固定连接一轻绳，轻绳另一端B连接转椅（视为质点）。转椅与雪地之间的动摩擦因数为μ，重力加速度为g，不计空气阻力。求转椅做匀速圆周运动时AB与OB之间的夹角。

解答：

（1）对转椅受力分析：重力mg、绳子拉力T、摩擦力f、支持力N

（2）匀速圆周运动，合力提供向心力

（3）竖直方向：N = mg

（4）水平方向：Tcosθ - f = mω²r（θ为AB与OB夹角）

（5）摩擦力：f = μN = μmg

（6）绳中拉力由转动角速度和几何关系确定

知识点标签：匀速圆周运动、牛顿第二定律、摩擦力

答案：θ = arctan(mω²r/μmg)

例题 11：【2024·河北卷】小车加速刹车 【中档】

某公司在封闭公路上对一新型电动汽车进行直线加速和刹车性能测试，某次测试的速度-时间图像如图所示。已知0~3.0s和3.5~6.0s内图线为直线，3.0~3.5s内图线为曲线。求该车的平均速度和加速度大小。

解答：

（1）0~3.0s内，图线为直线，做匀加速运动

（2）平均速度 = 初速度+末速度/2 = v₀/2

（3）位移s = 平均速度 × 时间

（4）加速度a = Δv/Δt

（5）3.0~3.5s内，图线为曲线，做变加速运动

（6）3.5~6.0s内，图线为直线，做匀减速运动

知识点标签：匀变速直线运动、v-t图像

答案：需要根据图像具体数据计算

例题 12：【2023·浙江卷】卫星轨道转移 【高档】

"嫦娥六号"探测器于2024年5月8日进入环月轨道，后续经调整环月轨道高度和倾角，实施月球背面软着陆。当探测器的轨道半径从r₁调整到r₂时（两轨道均可视为圆形轨道），其动能和周期从Ek₁、T₁分别变为Ek₂、T₂。下列选项正确的是？

解答：

（1）探测器在环月轨道做匀速圆周运动

（2）月球对探测器的万有引力提供向心力

（3）GMm/r² = mv²/r = mr(2π/T)²

（4）动能Ek = ½mv² = GMm/(2r)

（5）Ek₁/Ek₂ = r₂/r₁

（6）周期T = 2π√(r³/GM)

（7）T₁/T₂ = √(r₁³/r₂³)

知识点标签：万有引力定律、圆周运动、动能

答案：Ek₁/Ek₂ = r₂/r₁，T₁/T₂ = √(r₁³/r₂³)

例题 13：【2024·新课标卷】力学综合 【高档】

水平面上固定两排平行的半圆柱体，重为G的光滑圆柱体静置其上。a、b为相切点，∠aOb=90°，半径Ob与重力的夹角为37°。求圆柱体受到的支持力Fa和Fb的大小。

解答：

（1）对光滑圆柱体受力分析

（2）圆柱体受重力G，支持力Fa和Fb

（3）由平衡条件：Fa = Gsin37° = 0.6G

（4）Fb = Gcos37° = 0.8G

知识点标签：力的平衡、力的分解

答案：Fa = 0.6G，Fb = 0.8G

例题 14：【2023·新课标卷】电磁阻尼 【高档】

一粗精斜面固定在地面上，斜面顶端装有一光滑定滑轮。一细绳跨过滑轮，其一端悬挂物块N，另一端与斜面上的物块M相连，系统处于静止状态。现用水平向左的拉力缓慢拉动N，直至悬挂N的细绳与竖直方向成45°。在此过程中，M所受斜面的摩擦力大小如何变化？

解答：

（1）对N受力分析：重力mg、拉力T、水平力F

（2）由矢量三角形可知，T随细绳偏角增大而增大

（3）对M受力分析：摩擦力f的方向取决于T与Mgsinθ的大小关系

（4）若初始T < Mgsinθ（f沿斜面向上），则T增大时f先减小至零，再反向增大

（5）若初始T > Mgsinθ（f沿斜面向下），则f随T增大而增大

知识点标签：摩擦力、动态平衡、力的分析

答案：可能先减小后增大，或一直增大

例题 15：【2022·全国卷】验证牛顿第二定律实验 【基础】

某小组利用打点计时器探究物体加速度与其所受合外力的关系。实验装置如图所示，水平轨道上安装两个光电门，小车上固定一遮光片，细线一端与小车连接，另一端跨过定滑轮挂上钩码。实验前需要如何平衡摩擦力？

解答：

（1）实验前调节轨道右端滑轮高度，使细线与轨道平行

（2）适当垫高轨道左端以平衡小车所受摩擦力

（3）在不挂钩码的情况下，轻推小车，观察小车是否匀速运动

（4）如果小车匀速运动，说明摩擦力已平衡

（5）这样可以保证细线的拉力等于小车所受合外力

知识点标签：实验、牛顿第二定律、摩擦力平衡

答案：垫高轨道左端，使小车匀速下滑
