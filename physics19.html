知识点1

基本概念

α粒子散射实验：卢瑟福用α粒子轰击金箔，发现大部分α粒子直线穿过，少数发生大角度偏转，极少数甚至反弹

原子核式结构模型：在原子的中心有一个很小的核，叫做原子核，集中了原子的全部正电荷和几乎全部质量，电子在核外空间绕核旋转

氢原子光谱：氢原子发射的光谱是分立的线状光谱，存在巴耳末系、帕邢系、布喇开系等线系

知识点2

重要规律

玻尔理论：原子只能处于一系列不连续的能量状态，原子从一种能量状态跃迁到另一种能量状态时，发射或吸收光子

玻尔半径：r₁ = 0.53×10⁻¹⁰m，是氢原子电子轨道的最小半径

能级公式：E_n = -13.6eV/n²，其中n为主量子数

电子跃迁：从高能级跃迁到低能级时发射光子，从低能级跃迁到高能级时吸收光子

例题练习

例题 1：【2024·新课标卷】 玻尔理论 【基础】

根据玻尔理论，氢原子的基态能量为-13.6eV。求氢原子的第一激发态能量和从基态跃迁到第一激发态需要吸收的光子能量。

解答：

（1）氢原子的能级公式：E_n = -13.6eV/n²

（2）基态：n = 1，E₁ = -13.6eV/1² = -13.6eV

（3）第一激发态：n = 2，E₂ = -13.6eV/2² = -3.4eV

（4）跃迁需要吸收的光子能量：ΔE = E₂ - E₁ = -3.4eV - (-13.6eV) = 10.2eV

知识点标签：玻尔理论、能级、电子跃迁

答案：第一激发态能量 = -3.4eV，吸收光子能量 = 10.2eV

例题 2：【2023·浙江卷】 玻尔半径 【中档】

根据玻尔理论，氢原子电子在第n条轨道上的轨道半径r_n与玻尔半径r₁的关系是什么？

解答：

（1）玻尔半径：r₁ = 0.53×10⁻¹⁰m

（2）根据玻尔理论，电子的轨道半径：r_n = n²r₁

（3）其中n是主量子数，n = 1, 2, 3, ...

（4）电子轨道半径与主量子数的平方成正比

（5）r₂ = 4r₁，r₃ = 9r₁，...

知识点标签：玻尔理论、玻尔半径、轨道半径

答案：r_n = n²r₁

例题 3：【2022·新课标卷】 氢原子光谱 【中档】

氢原子从n = 3能级跃迁到n = 2能级时，发射光子的波长是多少？（已知里德伯常量R = 1.097×10⁷m⁻¹）

解答：

（1）根据玻尔频率条件：hν = E₃ - E₂

（2）波恩公式：1/λ = R(1/n₂² - 1/n₁²)

（3）对于巴耳末系（n = 3, 2, 1），1/λ = R(1/2² - 1/3²) = R(1/4 - 1/9)

（4）1/λ = R(9/36 - 4/36) = 5R/36

（5）波长：λ = 36/(5R) = 36/(5×1.097×10⁷) = 36/(5.485×10⁷) ≈ 6.56×10⁻⁷m

（6）换算成纳米：λ ≈ 656nm

（7）这是可见光中的红光

知识点标签：氢原子光谱、波恩公式、波长

答案：λ ≈ 656nm（红光）

例题 4：【2024·河北卷】 电子跃迁 【中档】

氢原子从n = 4能级跃迁到n = 2能级时，发射光子的能量和频率各是多少？（已知普朗克常量h = 6.63×10⁻³⁴J·s）

解答：

（1）氢原子的能级公式：E_n = -13.6eV/n²

（2）E₄ = -13.6eV/4² = -13.6eV/16 = -0.85eV

（3）E₂ = -13.6eV/2² = -13.6eV/4 = -3.4eV

（4）发射光子的能量：ΔE = E₄ - E₂ = -0.85eV - (-3.4eV) = 2.55eV

（5）光子频率：ν = ΔE/h = 2.55×1.6×10⁻¹⁹/(6.63×10⁻³⁴) ≈ 6.15×10¹⁴Hz

（6）波长：λ = c/ν = 3×10⁸/(6.15×10¹⁴) ≈ 4.88×10⁻⁷m = 488nm（可见光）

（7）电子从高能级跃迁到低能级时发射光子

知识点标签：电子跃迁、光子能量、频率

答案：ΔE = 2.55eV，ν ≈ 6.15×10¹⁴Hz

例题 5：【2023·全国卷】 原子核式结构 【中档】

卢瑟福α粒子散射实验的结果是什么？这个结果支持了汤姆逊的枣糕模型吗？

解答：

（1）α粒子散射实验结果：

- 绝大多数α粒子穿过金箔后仍沿原来的方向前进

- 少数α粒子发生了较大的偏转

- 极少数α粒子的偏转角超过90°，甚至被弹回

（2）不支持汤姆逊的枣糕模型

（3）汤姆逊模型认为正电荷均匀分布在原子中，无法解释大角度散射

（4）卢瑟福提出了原子核式结构模型来解释这个实验结果

（5）原子的大部分是空虚的，正电荷和几乎全部质量集中在一个很小的核上

知识点标签：α粒子散射、原子核式结构、汤姆逊模型

答案：不支持汤姆逊模型，支持原子核式结构模型

例题 6：【2024·辽宁卷】 能级跃迁 【中档】

氢原子电子从n = 3能级跃迁到基态时，可能发射几种不同能量的光子？它们的波长各是多少？

解答：

（1）从n = 3跃迁到n = 1（基态）有两种途径：

- 直接从n = 3到n = 1

- 先从n = 3到n = 2，再从n = 2到n = 1

（2）可能的跃迁：3→1，3→2，2→1

（3）可能发射3种不同能量的光子

（4）各跃迁的波长：

- 3→1：1/λ₁ = R(1/1² - 1/3²) = R(1 - 1/9) = 8R/9，λ₁ = 9/(8R) ≈ 102.6nm（赖曼系，紫外）

- 3→2：1/λ₂ = R(1/2² - 1/3²) = R(1/4 - 1/9) = 5R/36，λ₂ = 36/(5R) ≈ 656nm（巴耳末系，可见）

- 2→1：1/λ₃ = R(1/1² - 1/2²) = R(1 - 1/4) = 3R/4，λ₃ = 4/(3R) ≈ 121.6nm（赖曼系，紫外）

（5）其中λ₂ = 656nm是可见光中的红光

知识点标签：能级跃迁、光谱线、波恩公式

答案：3种，波长分别为102.6nm、656nm、121.6nm

例题 7：【2022·浙江卷】 电离能 【中档】

氢原子的电离能是多少？若要使基态的氢原子电离，需要吸收的光子最小能量是多少？

解答：

（1）氢原子的电离能：|E_∞ - E₁|

（2）其中E_∞ = 0（电子完全脱离原子核束缚时的能量为零）

（3）E₁ = -13.6eV（基态能量）

（4）电离能：|E_∞ - E₁| = |0 - (-13.6eV)| = 13.6eV

（5）电离能的倒数等于电离电势：1/(13.6V)

（6）使基态氢原子电离的最小光子能量：13.6eV

（7）对应波长：λ = hc/E = 6.63×10⁻³⁴×3×10⁸/(13.6×1.6×10⁻¹⁹) ≈ 91.2nm（紫外）

知识点标签：电离能、基态、光子能量

答案：电离能 = 13.6eV，最小光子能量 = 13.6eV

例题 8：【2023·河北卷】 巴耳末系 【中档】

氢原子巴耳末系的光谱线中，波长最长的是哪一条？它的波长和频率各是多少？

解答：

（1）巴耳末系的跃迁：n = n→2，其中n = 3, 4, 5, ...

（2）波长最长对应能量最小，即从n = 3跃迁到n = 2

（3）波恩公式：1/λ = R(1/2² - 1/3²) = R(1/4 - 1/9) = 5R/36

（4）波长：λ = 36/(5R) = 36/(5×1.097×10⁷) ≈ 6.56×10⁻⁷m = 656nm

（5）频率：ν = c/λ = 3×10⁸/(656×10⁻⁹) ≈ 4.57×10¹⁴Hz

（6）这是可见光中的红光（Hα线）

（7）巴耳末系的其它谱线波长更短，在紫外区

知识点标签：巴耳末系、波恩公式、波长

答案：Hα线，λ ≈ 656nm，ν ≈ 4.57×10¹⁴Hz

例题 9：【2024·新课标Ⅱ卷】 能级差 【中档】

氢原子两个相邻能级之间的能量差ΔE随能级数n的增大如何变化？

解答：

（1）氢原子的能级公式：E_n = -13.6eV/n²

（2）相邻能级n和n+1之间的能量差：

（3）ΔE = E_n - E_{n+1} = -13.6eV(1/n² - 1/(n+1)²)

（4）化简：ΔE = -13.6eV[(n+1)² - n²]/[n²(n+1)²] = -13.6eV(2n+1)/[n²(n+1)²]

（5）绝对值：|ΔE| = 13.6eV(2n+1)/[n²(n+1)²]

（6）随着n的增大，|ΔE|减小

（7）能级越高，相邻能级的间距越小

（8）当n→∞时，|ΔE| → 0，能级趋于连续

知识点标签：能级差、玻尔理论、氢原子能级

答案：|ΔE|随n的增大而减小

例题 10：【2022·全国卷】 电子轨道 【中档】

根据玻尔理论，氢原子电子在第n条轨道上的速率v_n与玻尔速率v₁的关系是什么？（已知v₁ = 2.18×10⁶m/s）

解答：

（1）根据玻尔理论，电子做圆周运动：mv²/r_n = ke²/r_n²

（2）角动量量子化：m·v_n·r_n = n·h/(2π)

（3）解得：v_n = n·e²/(2ε₀h) = v₁/n

（4）其中v₁ = e²/(2ε₀h) ≈ 2.18×10⁶m/s是玻尔速率

（5）所以电子在第n条轨道上的速率：v_n = v₁/n

（6）电子速率与主量子数成反比

（7）v₁ = 2.18×10⁶m/s，v₂ = 1.09×10⁶m/s，v₃ = 0.73×10⁶m/s，...

知识点标签：玻尔理论、电子轨道、电子速率

答案：v_n = v₁/n

例题 11：【2023·江苏卷】 光谱分析 【中档】

某元素的特征光谱中，有一条波长为λ的光谱线。已知这是该元素原子的电子从第m能级跃迁到第n能级时发射的。求第(m+1)能级跃迁到第n能级时发射光子的波长。

解答：

（1）从第m能级跃迁到第n能级的能量：ΔE₁ = E_m - E_n

（2）发射光子的波长：λ = hc/ΔE₁

（3）从第(m+1)能级跃迁到第n能级的能量：ΔE₂ = E_{m+1} - E_n

（4）根据波恩公式：1/λ₁ = R(1/n² - 1/m²)，1/λ₂ = R(1/n² - 1/(m+1)²)

（5）由于1/n² - 1/(m+1)² > 1/n² - 1/m²

（6）所以：1/λ₂ > 1/λ₁，即 λ₂ < λ₁

（7）第(m+1)能级跃迁到第n能级时发射光子的波长更短，频率更高

（8）具体的波长关系取决于m和n的值

知识点标签：光谱分析、能级跃迁、波恩公式

答案：λ₂ < λ₁

例题 12：【2024·江西卷】 原子核 【中档】

关于原子核，下列说法正确的是：

(A) 原子核由质子和电子组成

(B) 原子核带正电

(C) 原子核的大小约为10⁻¹⁰m

(D) 原子核几乎集中了原子的全部质量

解答：

（A）错误。原子核由质子和中子组成（除普通氢核外），电子在核外

（B）正确。原子核由质子带正电，中子不带电，所以原子核带正电

（C）错误。原子核的大小约为10⁻¹⁴~10⁻¹⁵m，比10⁻¹⁰m小得多

（D）正确。原子核的质子数和中子数之和（质量数）代表原子的质量，几乎集中了原子的全部质量

（5）原子核的体积很小，但质量很大，密度极大

知识点标签：原子核、质子、中子、原子核大小

答案：B、D

例题 13：【2022·河北卷】 能级跃迁和光谱 【中档】

氢原子从高能级跃迁到低能级时，发射光子。若发射光子的能量越大，则电子跃迁的能级差如何变化？对应的光谱线在光谱中的位置如何变化？

解答：

（1）发射光子的能量：ΔE = E高 - E低

（2）光子能量越大，意味着能级差越大

（3）根据波恩公式：1/λ = R(1/n低² - 1/n高²)

（4）能级差越大，1/λ越大，λ越小

（5）所以光子能量越大，波长越短，频率越高

（6）光谱线在光谱中的位置：波长越短，越靠近光谱的短波端（紫外区）

（7）光谱线按波长从长到短排列：红外 → 可见 → 紫外

（8）不同能级跃迁产生不同波长的光谱线

知识点标签：能级跃迁、光谱线、波长

答案：光子能量越大，能级差越大，波长越短，光谱线越靠近短波端

例题 14：【2023·新课标卷】 普朗克常量 【基础】

普朗克常量h = 6.63×10⁻³⁴J·s。已知氢原子基态能量E₁ = -13.6eV，求氢原子从基态跃迁到n = 2能级需要吸收的光子能量和对应的波长。

解答：

（1）氢原子的能级公式：E_n = -13.6eV/n²

（2）E₂ = -13.6eV/2² = -13.6eV/4 = -3.4eV

（3）需要吸收的光子能量：ΔE = E₂ - E₁ = -3.4eV - (-13.6eV) = 10.2eV

（4）对应的波长：λ = hc/ΔE = 6.63×10⁻³⁴×3×10⁸/(10.2×1.6×10⁻¹⁹)

（5）λ = 1.989×10⁻²⁵/(1.632×10⁻¹⁸) ≈ 1.22×10⁻⁷m = 122nm

（6）这是氢原子赖曼系的Hβ线，在紫外区

（7）吸收光子的能量等于发射光子的能量

知识点标签：普朗克常量、能级跃迁、波长

答案：ΔE = 10.2eV，λ ≈ 122nm

例题 15：【2024·全国卷】 原子结构综合 【高档】

氢原子中，电子在量子数为n的能级上运动。若要使电子跃迁到n = 2能级，可能需要吸收的光子有哪些波长？

解答：

（1）电子从高能级n跃迁到n = 2能级

（2）可能的跃迁：n = 3, 4, 5, ... → 2

（3）根据波恩公式：1/λ_{n,2} = R(1/2² - 1/n²) = R(1/4 - 1/n²)

（4）各跃迁的波长：

- 3→2：1/λ₃ = R(1/4 - 1/9) = 5R/36，λ₃ = 36/(5R) ≈ 656nm（巴耳末系，可见）

- 4→2：1/λ₄ = R(1/4 - 1/16) = 3R/16，λ₄ = 16/(3R) ≈ 486nm（巴耳末系，可见）

- 5→2：1/λ₅ = R(1/4 - 1/25) = 21R/100，λ₅ = 100/(21R) ≈ 434nm（巴耳末系，可见）

- ...

（5）可能的吸收波长有无限多个，当n→∞时，λ→4/(3R) ≈ 486nm

（6）吸收的波长都在巴耳末系范围内

（7）这些波长对应的光子在可见光区域

知识点标签：原子结构、能级跃迁、波恩公式、吸收光谱

答案：无限多个波长，最小波长约为486nm，在可见光区域
