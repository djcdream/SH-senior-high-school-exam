6.1 直线与方程

直线的方程

点斜式

y - y₁ = k(x - x₁)

适用：已知一点和斜率

注意：不能表示垂直于 x 轴的直线

斜截式

y = kx + b

适用：已知斜率和 y 轴截距

注意：不能表示垂直于 x 轴的直线

一般式

Ax + By + C = 0

适用：可以表示任何直线

斜率：k = -A/B（当 B ≠ 0）

两条直线的位置关系

平行

两条直线平行，当且仅当它们的斜率相等

k₁ = k₂ 且 b₁ ≠ b₂

一般式：A₁B₂ - A₂B₁ = 0 且 A₁C₂ - A₂C₁ ≠ 0

重合

两条直线重合，当且仅当它们的方程成比例

k₁ = k₂ 且 b₁ = b₂

一般式：A₁/B₁ = A₂/B₂ = C₁/C₂

垂直

两条直线垂直，当且仅当它们的斜率乘积为 -1

k₁·k₂ = -1

一般式：A₁A₂ + B₁B₂ = 0

6.2 圆与方程

圆的标准方程

圆心在 (a, b)，半径为 r 的圆的方程为：

(x - a)² + (y - b)² = r²

特例：圆心在原点的圆：x² + y² = r²

圆的一般方程

展开后的圆的方程：

x² + y² + Dx + Ey + F = 0

条件：D² + E² - 4F > 0

圆心：(-D/2, -E/2)

半径：r = √(D² + E² - 4F)/2

直线与圆的位置关系

判定方法

设圆心到直线的距离为 d，圆的半径为 r

  • d < r：相交（有两个交点）
  • d = r：相切（有一个交点）
  • d > r：相离（无交点）

点到直线的距离公式

点 P(x₀, y₀) 到直线 Ax + By + C = 0 的距离：

d = |Ax₀ + By₀ + C| / √(A² + B²)

6.3 圆锥曲线

椭圆

标准方程

x²/a² + y²/b² = 1 (a > b > 0)

焦点：F₁(±c, 0)，其中 c² = a² - b²

焦距：2c

离心率：e = c/a（0 < e < 1）

顶点：(±a, 0)、(0, ±b)

双曲线

标准方程

x²/a² - y²/b² = 1 (a > 0, b > 0)

焦点：F₁(±c, 0)，其中 c² = a² + b²

焦距：2c

离心率：e = c/a（e > 1）

顶点：(±a, 0)

渐近线：y = ±(b/a)x

抛物线

标准方程

y² = 2px (p > 0)

焦点：F(p/2, 0)

准线：x = -p/2

顶点：(0, 0)

开口方向：向右

其他形式

y² = -2px：向左

x² = 2py：向上

x² = -2py：向下

例题 1：直线的点斜式方程 【基础】

求过点 (1, 2)，斜率为 3 的直线方程

解答：

使用点斜式：y - y₁ = k(x - x₁)

y - 2 = 3(x - 1)

y - 2 = 3x - 3

y = 3x - 1

答案：y = 3x - 1

例题 2：直线的斜截式方程 【基础】

已知直线 y = 2x + 1，求斜率和 y 轴截距

解答：

斜截式：y = kx + b

比较得：k = 2，b = 1

答案：斜率2，y轴截距1

例题 3：两条直线平行 【基础】

直线 y = 2x + 1 与 y = 2x + 3 的位置关系是

解答：

y = 2x + 1：斜率 k₁ = 2

y = 2x + 3：斜率 k₂ = 2

由于 k₁ = k₂ 且截距不同，所以两条直线平行

答案：平行

例题 4：两条直线垂直 【基础】

直线 y = 2x + 1 与 y = -x/2 + 3 的位置关系是

解答：

y = 2x + 1：斜率 k₁ = 2

y = -x/2 + 3：斜率 k₂ = -1/2

k₁ × k₂ = 2 × (-1/2) = -1

由于斜率乘积为 -1，所以两条直线垂直

答案：垂直

例题 8：圆的一般方程 【基础】

求圆 x² + y² - 4x + 6y - 3 = 0 的圆心和半径

解答：

一般式：x² + y² + Dx + Ey + F = 0

比较得：D = -4, E = 6, F = -3

圆心：(-D/2, -E/2) = (2, -3)

半径：r = √(D² + E² - 4F)/2

= √((-4)² + 6² - 4×(-3))/2

= √(16 + 36 + 12)/2 = √64/2 = 8/2 = 4

答案：圆心(2, -3)，半径4

例题 9：点到直线的距离 【基础】

求点 (1, 2) 到直线 x + y - 1 = 0 的距离

解答：

点到直线的距离公式：d = |Ax₀ + By₀ + C| / √(A² + B²)

d = |1 × 1 + 1 × 2 - 1| / √(1 + 1)

= |1 + 2 - 1| / √2 = |2| / √2 = 2/√2 = √2

答案：√2

例题 10：直线与圆的位置关系 【基础】

判断直线 x + y - 5 = 0 与圆 x² + y² = 4 的位置关系

解答：

圆心 (0, 0)，半径 r = 2

圆心到直线的距离：

d = |0 + 0 - 5| / √(1 + 1) = 5/√2 ≈ 3.54

由于 d > r，所以直线与圆相离

答案：相离

例题 28：直线与圆综合 【提高】

求过点 (1, 2) 且与圆 x² + y² = 1 相切的直线方程

解答：

圆心 (0, 0)，半径 r = 1

点 (1, 2) 到圆心的距离：√(1 + 4) = √5 > 1

点在圆外，有两条切线

设切线斜率为 k，则切线方程为：y - 2 = k(x - 1)

化为一般式：kx - y + (2 - k) = 0

圆心到切线的距离等于半径：

|k×0 - 0 + (2 - k)| / √(k² + 1) = 1

|2 - k| = √(k² + 1)

两边平方：(2 - k)² = k² + 1

4 - 4k + k² = k² + 1

3 - 4k = 0

k = 3/4

但这样只得到一个斜率，说明另一条切线垂直于 x 轴

切线1：y - 2 = (3/4)(x - 1)

切线2：x = 1

答案：y - 2 = (3/4)(x - 1) 或 x = 1

例题 35：圆的综合 【提高】

已知圆 C 过点 A(1, 3)、B(5, 1)，且圆心在直线 y = x 上，求圆的方程

解答：

设圆心 C(a, a)（因为圆心在 y = x 上）

由于圆过 A、B，所以 |CA| = |CB|

√[(1 - a)² + (3 - a)²] = √[(5 - a)² + (1 - a)²]

两边平方：(1 - a)² + (3 - a)² = (5 - a)² + (1 - a)²

(3 - a)² = (5 - a)²

9 - 6a + a² = 25 - 10a + a²

4a = 16

a = 4

所以圆心 C(4, 4)

半径 r = √[(1 - 4)² + (3 - 4)²] = √(9 + 1) = √10

圆的方程：(x - 4)² + (y - 4)² = 10

答案：(x - 4)² + (y - 4)² = 10

例题 38：两圆位置关系 【提高】

圆 C₁：x² + y² = 1 与圆 C₂：(x - 3)² + (y - 4)² = 16 的位置关系是

圆 C₂：圆心 O₂(3, 4)，半径 r₂ = 4

圆心距：d = √[(3 - 0)² + (4 - 0)²] = √(9 + 16) = √25 = 5

半径和：r₁ + r₂ = 1 + 4 = 5

由于 d = r₁ + r₂，所以两圆外切

答案：外切

例题 39：圆与椭圆相切 【提高】

圆 x² + y² = 1 与椭圆 x²/4 + y²/1 = 1 在第一象限相切，求切点坐标

解答：

由于相切，圆心与椭圆中心连线通过切点

椭圆长轴在 x 轴上，a = 2, b = 1

第一象限的切点在椭圆上：(2cosθ, sinθ)

由于切点也在圆上：(2cosθ, sinθ) 满足 x² + y² = 1

(2cosθ)² + (sinθ)² = 1

4cos²θ + sin²θ = 1

4cos²θ + (1 - cos²θ) = 1

3cos²θ = 0

cosθ = 0

θ = π/2

切点：(2cos(π/2), sin(π/2)) = (0, 1)

答案：(0, 1)

例题 40：直线与椭圆相交 【提高】

直线 y = x + 1 截椭圆 x²/4 + y² = 1 所得的弦长是

解答：

将直线代入椭圆方程：x²/4 + (x + 1)² = 1

x²/4 + x² + 2x + 1 = 1

两边乘以4：x² + 4x² + 8x + 4 = 4

5x² + 8x = 0

x(5x + 8) = 0

x = 0 或 x = -8/5

当 x = 0 时，y = 1

当 x = -8/5 时，y = -8/5 + 1 = -3/5

弦长：√[(-8/5 - 0)² + (-3/5 - 1)²] = √(64/25 + 64/25) = √(128/25) = 8√2/5

答案：8√2/5

例题 41：圆与抛物线相交 【提高】

圆 x² + y² = 4 与抛物线 y² = 4x 的交点个数是

解答：

抛物线：y² = 4x

代入圆方程：x² + 4x = 4

x² + 4x - 4 = 0

x = [-4 ± √(16 + 16)]/2 = [-4 ± √32]/2 = -2 ± 2√2

x₁ = -2 - 2√2 < 0（舍去，因为抛物线 x ≥ 0）

x₂ = -2 + 2√2 > 0

只有 x₂ 有效

当 x = -2 + 2√2 时，y² = 4(-2 + 2√2)

= -8 + 8√2

由于 -8 + 8√2 ≈ -8 + 11.31 = 3.31 > 0，所以 y 有两个值

因此有 2 个交点

答案：2个

例题 42：椭圆焦点弦 【提高】

椭圆 x²/25 + y²/9 = 1 过右焦点 F 的弦的中点为 M(3, 0)，求弦的方程

解答：

椭圆：a = 5, b = 3

c = √(a² - b²) = √(25 - 9) = √16 = 4

右焦点：F(4, 0)

弦的中点为 M(3, 0)

由于弦的中点在 x 轴上，所以弦垂直于 x 轴

弦的方程：x = 3

答案：x = 3

例题 43：双曲线渐近线夹角 【提高】

双曲线 x²/9 - y²/16 = 1 的两条渐近线的夹角是

解答：

双曲线：a = 3, b = 4

渐近线：y = ±(b/a)x = ±(4/3)x

设两条渐近线的斜率分别为 k₁ = 4/3，k₂ = -4/3

夹角 α 满足：tanα = |(k₂ - k₁)/(1 + k₁k₂)|

= |(-4/3 - 4/3)/(1 + (-16/9))|

= |(-8/3)/(-7/9)| = |-8/3 × (-9/7)| = |24/7| = 24/7

α = arctan(24/7)

答案：arctan(24/7)

例题 44：抛物线动点 【提高】

抛物线 y² = 4x 上的一点 P 到焦点 F 的距离等于到准线 l 的距离，求点 P 的坐标

解答：

抛物线：y² = 4x

2p = 4，p = 2

焦点：F(p/2, 0) = (1, 0)

准线：x = -p/2 = -1

设点 P(x, y)

点 P 到焦点 F 的距离：d₁ = √[(x - 1)² + y²]

点 P 到准线的距离：d₂ = |x + 1|

由于 d₁ = d₂，所以 √[(x - 1)² + y²] = |x + 1|

平方：(x - 1)² + y² = (x + 1)²

x² - 2x + 1 + y² = x² + 2x + 1

y² = 4x

这与抛物线方程一致

由于点 P 在抛物线上，所以满足这个条件

点 P 的坐标：(t², 2t)，其中 t 为参数

答案：(t², 2t)，t为参数

例题 45：椭圆面积最大 【提高】

点 P 在椭圆 x²/4 + y²/3 = 1 上移动，求 △POA 面积的最大值（其中 O 为原点，A 为右顶点）

解答：

椭圆：a = 2, b = √3

右顶点：A(2, 0)

椭圆上点：P(2cosθ, √3sinθ)

△POA 的面积：S = (1/2)×|OA|×|y_P| = (1/2)×2×|√3sinθ| = √3|sinθ|

当 sinθ = ±1 时，面积最大

S_max = √3

此时 θ = π/2 或 θ = 3π/2

点 P：(0, √3) 或 (0, -√3)

答案：最大值√3

例题 46：圆与直线最值 【提高】

圆 x² + y² = 4 上的动点 P 到直线 y = x + 2 的距离的最大值是

解答：

圆：x² + y² = 4，圆心 O(0, 0)，半径 r = 2

圆心到直线的距离：d = |0 - 0 + 2|/√(1 + 1) = 2/√2 = √2

圆上点到直线的距离：d_P = |d ± r|

d_max = |√2 + 2| = √2 + 2 ≈ 1.414 + 2 = 3.414

答案：√2 + 2

例题 47：椭圆离心率范围 【提高】

椭圆 x²/a² + y²/(a² - 1) = 1（a > 1）的离心率的取值范围是

c = 1

例题 48：双曲线离心率 【提高】

已知双曲线的离心率 e = 2，焦距 2c = 4√3，则双曲线的标准方程是

解答：

2c = 4√3，c = 2√3

e = c/a = 2

a = c/2 = √3

c² = a² + b²

(2√3)² = 3 + b²

12 = 3 + b²

b² = 9，b = 3

双曲线方程：x²/3 - y²/9 = 1

答案：x²/3 - y²/9 = 1

例题 49：抛物线范围 【提高】

抛物线 y² = 4x 上的点 P 到点 Q(2, 2) 的距离的最小值是

解答：

设抛物线上点 P(t², 2t)

距离平方：d² = (t² - 2)² + (2t - 2)²

= (t² - 2)² + 4(t - 1)²

设 u = t - 1

d² = ((u + 1)² - 3)² + 4u²

= (u² + 2u - 2)² + 4u²

= u⁴ + 4u³ + 4u² - 4u² - 8u + 4 + 4u²

= u⁴ + 4u³ + 4u² - 8u + 4

求导：d'(u²) = 2u³ + 12u² + 8u - 8

令 d'(u²) = 0：2u³ + 12u² + 8u - 8 = 0

u³ + 6u² + 4u - 4 = 0

(u - 1)(u² + 7u + 4) = 0

u = 1

t = u + 1 = 2

点 P(4, 4)

d_min = √[(4 - 2)² + (4 - 2)²] = √(4 + 4) = √8 = 2√2

答案：2√2

例题 50：圆锥曲线综合 【提高】

已知椭圆 C₁ 和双曲线 C₂ 有相同的焦点 F₁(±√3, 0) 和 F₂(±√3, 0)，且椭圆的离心率 e₁ = √3/2，双曲线的离心率 e₂ = 2/√3，则椭圆 C₁ 和双曲线 C₂ 的标准方程分别是

解答：

对于椭圆：e₁ = c/a = √3/2

c = √3，a = c/e₁ = √3 × 2/√3 = 2

c² = a² - b²

3 = 4 - b²

b² = 1，b = 1

椭圆方程：x²/4 + y²/1 = 1


对于双曲线：e₂ = c/a = 2/√3

c = √3，a = c/e₂ = √3 × √3/2 = 3/2

c² = a² + b²

3 = 9/4 + b²

b² = 3 - 9/4 = 3/4，b = √3/2

双曲线方程：x²/(9/4) - y²/(3/4) = 1

即：4x²/9 - 4y²/3 = 1

答案：x²/4 + y²/1 = 1 和 4x²/9 - 4y²/3 = 1
