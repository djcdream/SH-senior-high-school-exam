知识点1

基本概念

光的直线传播：光在同种均匀介质中沿直线传播

光的反射：光从一种介质射到界面时，一部分光返回原介质

光的折射：光从一种介质射入另一种介质时，传播方向改变

全反射：光从光密介质射向光疏介质，入射角大于临界角时，全部反射

知识点2

重要规律

反射定律：反射光线、入射光线和法线在同一平面内，反射角等于入射角

折射定律：入射光线、折射光线和法线在同一平面内，入射角的正弦与折射角的正弦之比等于两种介质的相对折射率

色散：不同频率的光在同一种介质中的折射率不同，造成白光分解为色光

光的干涉：两列相干光叠加，形成明暗条纹

光的衍射：光遇到障碍物时，能绕过障碍物继续传播

例题练习

例题 1：【2024·新课标卷】 光的反射 【基础】

一光线以入射角i射到平面镜上，求反射光线的反射角和反射光线与入射光线的夹角。

解答：

（1）根据光的反射定律：反射角等于入射角，r = i

（2）反射光线与入射光线的夹角：θ = r + i = i + i = 2i

（3）反射光线与镜面的夹角：α = 90° - r = 90° - i

（4）入射光线与镜面的夹角：β = 90° - i

（5）反射光线与入射光线的夹角是入射角的两倍

知识点标签：光的反射、反射定律、平面镜

答案：r = i，θ = 2i

例题 2：【2023·浙江卷】 光的折射 【基础】

一光线从空气射入水中，入射角为i，折射角为r。若空气中的光速为c₁，水中的光速为c₂（c₁ > c₂），求空气对水的折射率。

解答：

（1）根据光的折射定律：n₁₂ = sin i/sin r

（2）空气对水的折射率：n₁₂ = c₁/c₂

（3）所以：sin i/sin r = c₁/c₂

（4）折射率也可以表示为速度之比

（5）光从光疏介质进入光密介质时，折射角小于入射角

知识点标签：光的折射、折射定律、折射率

答案：n₁₂ = c₁/c₂

例题 3：【2022·新课标卷】 全反射 【中档】

一光线从水中射向空气，水的折射率为n = 4/3。求发生全反射的临界角。

解答：

（1）全反射的条件：光从光密介质射向光疏介质，入射角大于临界角

（2）临界角C的定义：折射角r = 90°时的入射角

（3）根据折射定律：sin C/sin 90° = n空气/n水

（4）由于n空气 ≈ 1，所以：sin C = 1/n = 3/4

（5）临界角：C = arcsin(3/4) ≈ 48.6°

（6）当入射角i > 48.6°时，发生全反射

知识点标签：全反射、临界角、折射定律

答案：C = arcsin(3/4) ≈ 48.6°

例题 4：【2024·河北卷】 光的色散 【中档】

白光通过棱镜分解为七色光，其中红光的折射率最小，紫光的折射率最大。比较红光和紫光的折射角和偏折角。

解答：

（1）根据折射定律：n = sin i/sin r

（2）入射角i相同，折射率n越大，折射角r越小

（3）红光的折射率最小，所以红光的折射角最大

（4）紫光的折射率最大，所以紫光的折射角最小

（5）偏折角δ = i - r，所以红光的偏折角最小，紫光的偏折角最大

（6）这形成了彩虹光谱，红光在外，紫光在内

知识点标签：光的色散、棱镜、折射率

答案：红光折射角最大，紫光偏折角最大

例题 5：【2023·全国卷】 光的干涉 【中档】

一单色光通过双缝干涉装置，缝间距为d，双缝到屏幕的距离为L，屏幕上相邻亮纹的间距为Δx。求该单色光的波长。

解答：

（1）双缝干涉的亮纹条件：Δr = kλ，k = 0, 1, 2, ...

（2）相邻亮纹的波程差：Δr = d·sinθ ≈ d·tanθ = d·(Δx/L)

（3）所以：λ = Δr = d·(Δx/L)

（4）波长：λ = d·Δx/L

（5）其中d是缝间距，L是缝到屏幕的距离，Δx是相邻亮纹的间距

（6）通过测量Δx可以计算光的波长

知识点标签：光的干涉、双缝干涉、波长测量

答案：λ = d·Δx/L

例题 6：【2024·辽宁卷】 光的衍射 【中档】

一单色光通过宽缝，缝宽为a，光的波长为λ。若缝宽a与波长λ的关系满足a = 5λ，在缝后多远处能观察到明显的衍射现象？

解答：

（1）光的衍射现象与缝宽a和波长λ的比值有关

（2）当a ≈ λ时，衍射现象最明显

（3）当a >> λ时，衍射现象不明显，近似沿直线传播

（4）当a << λ时，衍射现象非常明显

（5）本题中a = 5λ，属于"缝宽略大于波长"的情况

（6）在缝后一定距离（远大于缝宽）处都能观察到衍射现象，但不是非常明显

（7）当a = λ时，衍射现象最明显

知识点标签：光的衍射、缝宽、波长

答案：a = 5λ时，在缝后较远处能观察到一定的衍射现象

例题 7：【2022·浙江卷】 凸透镜成像 【中档】

一物体放在焦距为f的凸透镜前2f处。求像的位置、大小和性质。

解答：

（1）透镜成像公式：1/u + 1/v = 1/f

（2）物体距离：u = 2f

（3）代入公式：1/(2f) + 1/v = 1/f

（4）解得：1/v = 1/f - 1/(2f) = 1/(2f)

（5）像距：v = 2f

（6）放大率：m = v/u = (2f)/(2f) = 1

（7）像的大小与物体相同，像为实像、倒立

（8）这是凸透镜成像的特殊点，当u = 2f时，v = 2f，m = 1

知识点标签：凸透镜、透镜成像、放大率

答案：v = 2f，m = 1，实像、倒立、等大

例题 8：【2023·河北卷】 光的反射和折射 【中档】

一光线以入射角i从空气射入水中，一部分反射，一部分折射。若空气中的光速为c₁，水中的光速为c₂，求反射角和折射角。

解答：

（1）反射角：根据反射定律，r = i

（2）折射角：根据折射定律，n₁₂ = sin i/sin r = c₁/c₂

（3）所以：sin r = (c₂/c₁)·sin i

（4）折射角：r = arcsin[(c₂/c₁)·sin i]

（5）由于c₁ > c₂（光在空气中比水中快），所以c₂/c₁ < 1

（6）因此sin r < sin i，折射角r小于入射角i

（7）反射光线和折射光线分别在界面两侧

知识点标签：光的反射、折射、反射角、折射角

答案：r = i，折射角r = arcsin[(c₂/c₁)·sin i]

例题 9：【2024·新课标Ⅱ卷】 光的反射成像 【中档】

一物体放在平面镜前距离d处，求像的位置和像与物体的距离。

解答：

（1）平面镜成像特点：物距等于像距

（2）物体距离：u = d

（3）像距：v = d

（4）像在平面镜后方距离d处

（5）像与物体的距离：L = u + v = d + d = 2d

（6）像为虚像、正立、与物体大小相同

（7）平面镜成的像是左右对称的

知识点标签：平面镜成像、像距、物距

答案：像距v = d，像与物体的距离L = 2d

例题 10：【2022·全国卷】 光的折射率 【基础】

一光线从介质A射入介质B，入射角为i，折射角为r。若介质A中的光速为v_A，介质B中的光速为v_B，求介质B对介质A的折射率。

解答：

（1）介质B对介质A的折射率：n_BA = sin i/sin r

（2）折射率与光速的关系：n_BA = v_A/v_B

（3）所以：sin i/sin r = v_A/v_B

（4）若v_A > v_B，则n_BA > 1，介质B比介质A光密

（5）若v_A < v_B，则n_BA < 1，介质B比介质A光疏

（6）折射率描述了光在不同介质中的传播特性

知识点标签：折射率、光速、折射定律

答案：n_BA = v_A/v_B

例题 11：【2023·江苏卷】 凹透镜成像 【中档】

一物体放在焦距为f的凹透镜前f处。求像的位置、大小和性质。

解答：

（1）透镜成像公式：1/u + 1/v = 1/f

（2）凹透镜的焦距f为负值

（3）物体距离：u = f（f为负，所以u为正）

（4）代入公式：1/f + 1/v = 1/f

（5）解得：1/v = 0，v → ∞（平行光）

（6）放大率：m = v/u → ∞

（7）像在无穷远处，为虚像、正立

（8）这是凹透镜成像的特殊情况

知识点标签：凹透镜、透镜成像、焦距

答案：v → ∞，像在无穷远处，虚像

例题 12：【2024·江西卷】 光的干涉条纹 【中档】

一双缝干涉装置，缝间距为d，双缝到屏幕的距离为L，光的波长为λ。求屏幕上第k级亮纹的位置。

解答：

（1）双缝干涉的亮纹条件：Δr = kλ

（2）波程差：Δr = d·sinθ ≈ d·tanθ = d·(x/L)

（3）其中x是亮纹到中央亮纹的距离

（4）所以：d·(x/L) = kλ

（5）解得：x = kλL/d

（6）k = 0时为中央亮纹，k = 1, 2, ...时为两侧的第1、2、...级亮纹

（7）相邻亮纹的间距：Δx = λL/d

知识点标签：光的干涉、双缝干涉、亮纹位置

答案：x = kλL/d

例题 13：【2022·河北卷】 凸透镜成像公式 【中档】

一凸透镜的焦距为f，物体放在透镜前距离u处。求像的距离v和放大率m。

解答：

（1）透镜成像公式：1/u + 1/v = 1/f

（2）解得像距：1/v = 1/f - 1/u = (u - f)/(uf)

（3）所以：v = uf/(u - f)

（4）放大率：m = v/u = [uf/(u - f)]/u = f/(u - f)

（5）当u > f时，v > 0，成实像、倒立

（6）当u < f时，v < 0，成虚像、正立

（7）当u = 2f时，v = 2f，m = 1

知识点标签：凸透镜、透镜成像、放大率

答案：v = uf/(u - f)，m = f/(u - f)

例题 14：【2023·新课标卷】 光的反射定律 【基础】

一光线以入射角i射到平面镜上，若将平面镜旋转角度α，入射光线方向不变，求反射光线与原反射光线的夹角。

解答：

（1）原反射角：r₁ = i

（2）原反射光线与镜面的夹角：θ₁ = 90° - i

（3）平面镜旋转角度α后，法线也旋转角度α

（4）新的入射角：i' = i + α

（5）新的反射角：r' = i' = i + α

（6）反射光线的转角：Δθ = 2α

（7）反射光线与原反射光线的夹角：2α

（8）平面镜旋转α角，反射光线旋转2α角

知识点标签：光的反射、平面镜旋转、反射角

答案：反射光线与原反射光线的夹角为2α

例题 15：【2024·全国卷】 光的全反射应用 【高档】

一光纤的纤芯折射率为n₁，包层折射率为n₂（n₁ > n₂），光线从空气射入纤芯端面。求光线能在光纤中发生全反射的最大入射角。

解答：

（1）光纤发生全反射的条件：光线从纤芯射向包层时，入射角大于临界角

（2）纤芯对包层的临界角：C = arcsin(n₂/n₁)

（3）光纤端面的数值孔径：NA = n₁·sinθ_max

（4）其中θ_max是最大入射角

（5）数值孔径也可以表示为：NA = √(n₁² - n₂²)

（6）所以：sinθ_max = NA/n₁ = √(n₁² - n₂²)/n₁ = √(1 - (n₂/n₁)²)

（7）最大入射角：θ_max = arcsin[√(n₁² - n₂²)/n₁]

（8）光纤利用全反射原理传输光信号，损耗极小

知识点标签：全反射、光纤、数值孔径、临界角

答案：θ_max = arcsin[√(n₁² - n₂²)/n₁]
