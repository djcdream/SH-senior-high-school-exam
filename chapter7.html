7.1 空间几何体的表面积与体积

柱体

棱柱

体积 = Sh

说明：S 为底面积，h 为高

圆柱

表面积 = 2πr² + 2πrh
体积 = πr²h

说明：r 为底面半径，h 为高

锥体

棱锥

体积 = (1/3)Sh

说明：S 为底面积，h 为高

圆锥

表面积 = πr² + πrl
体积 = (1/3)πr²h

说明：r 为底面半径，l 为母线长，h 为高

l = √(r² + h²)

球

表面积与体积公式

表面积 = 4πR²
体积 = (4/3)πR³

说明：R 为球的半径

7.2 点、直线、平面之间的位置关系

直线与平面的位置关系

直线在平面内

直线上的所有点都在平面内

直线与平面平行

直线与平面没有公共点

直线与平面相交

直线与平面有且只有一个公共点

平面与平面的位置关系

平面与平面平行

两个平面没有公共点

平面与平面相交

两个平面有一条公共直线

二面角

定义

从一条直线出发的两个半平面所组成的图形叫做二面角

二面角的平面角：过二面角的棱上任意一点，在两个面内分别作垂直于棱的射线，这两条射线所成的角

直线与平面所成的角

定义

平面的一条斜线和它在平面内的射影所成的锐角，叫做这条直线和这个平面所成的角

说明：直线与平面平行时，所成的角为 0°

例题练习

例题 1：圆柱的表面积 【基础】

已知圆柱的底面半径为 2，高为 3，求圆柱的表面积

解答：

表面积 = 2πr² + 2πrh

= 2π×2² + 2π×2×3

= 8π + 12π = 20π

答案：20π

例题 2：圆锥的体积 【基础】

已知圆锥的底面半径为 3，高为 4，求圆锥的体积

解答：

体积 = (1/3)πr²h

= (1/3)π×3²×4

= (1/3)π×9×4

= 12π

答案：12π

例题 3：球的表面积 【基础】

已知球的半径为 2，求球的表面积

解答：

表面积 = 4πR²

= 4π×2² = 16π

答案：16π

例题 4：球的体积 【基础】

已知球的半径为 3，求球的体积

解答：

体积 = (4/3)πR³

= (4/3)π×3³

= (4/3)π×27 = 36π

答案：36π

例题 5：棱锥的体积 【基础】

已知棱锥的底面积为 6，高为 4，求棱锥的体积

解答：

体积 = (1/3)Sh

= (1/3)×6×4 = 8

答案：8

例题 6：棱柱的体积 【基础】

已知棱柱的底面积为 5，高为 3，求棱柱的体积

解答：

体积 = Sh

= 5×3 = 15

答案：15

例题 7：2024年新高考I卷 【中档】

已知圆锥的底面半径为 1，母线长为 √5，求圆锥的高

解答：

l² = r² + h²

h² = l² - r²

= (√5)² - 1² = 5 - 1 = 4

h = 2

答案：2

例题 8：2023年全国甲卷 【中档】

已知球的体积为 36π，求球的半径

解答：

体积 = (4/3)πR³ = 36π

R³ = 36π × (3/4) / π = 27

R = 3

答案：3

例题 9：2022年新高考I卷 【中档】

已知圆柱的体积为 16π，高为 4，求圆柱的底面半径

解答：

体积 = πr²h = 16π

r²h = 16

r²×4 = 16

r² = 4

r = 2

答案：2

例题 10：2021年全国乙卷 【中档】

已知球的表面积为 16π，求球的体积

解答：

表面积 = 4πR² = 16π

R² = 4

R = 2

体积 = (4/3)πR³ = (4/3)π×8 = (32/3)π

答案：(32/3)π

例题 11：2020年全国I卷 【中档】

已知圆锥的体积为 4π，高为 3，求圆锥的底面半径

解答：

体积 = (1/3)πr²h = 4π

(1/3)r²×3 = 4

r² = 4

r = 2

答案：2

例题 12：2019年全国I卷 【中档】

已知球的半径为 R，球的表面积增加 25%，则球的半径增加

解答：

原表面积：S₁ = 4πR²

新表面积：S₂ = S₁(1 + 25%) = 1.25S₁ = 5πR²

4πr² = 5πR²

r² = (5/4)R²

r = (√5/2)R

半径增加：r - R = ((√5/2) - 1)R = ((√5 - 2)/2)R

答案：((√5 - 2)/2)R

例题 13：2018年全国II卷 【中档】

已知圆柱的底面周长为 4π，高为 2，求圆柱的体积

解答：

周长 = 2πr = 4π

r = 2

体积 = πr²h = π×2²×2 = 8π

答案：8π

例题 14：直线与平面平行 【基础】

已知直线 l 与平面 α 平行，且直线 m 与平面 α 垂直，则直线 l 与直线 m 的位置关系是

解答：

由于 l ∥ α，m ⟂ α

所以 l ⟂ m（直线 l 与直线 m 垂直）

答案：垂直

例题 15：平面与平面平行 【基础】

已知平面 α ∥ β，平面 β ∥ γ，则平面 α 与平面 γ 的位置关系是

解答：

由于 α ∥ β，β ∥ γ

所以 α ∥ γ（平面 α 与平面 γ 平行）

答案：平行

例题 16：圆锥的综合 【提高】

已知圆锥的底面半径为 3，母线长为 5，求圆锥的表面积

解答：

先求圆锥的高：

h² = l² - r² = 5² - 3² = 25 - 9 = 16

h = 4

圆锥的表面积：

= πr² + πrl

= π×3² + π×3×5

= 9π + 15π = 24π

答案：24π

例题 17：球的综合 【提高】

已知球的半径增加 50%，求球的体积增加的百分比

解答：

原半径：R

新半径：r = R(1 + 50%) = 1.5R

原体积：V₁ = (4/3)πR³

新体积：V₂ = (4/3)π(1.5R)³ = (4/3)π×3.375R³ = 4.5πR³

体积增加：V₂ - V₁ = 4.5πR³ - (4/3)πR³ = (4.5 - 1.333)πR³ = 3.167πR³

增加百分比：3.167/(4/3) = 3.167×3/4 = 2.375

= 237.5%

答案：237.5%

例题 18：圆柱与球的体积比 【提高】

已知圆柱的底面半径为 r，球的半径也为 r，且圆柱的高为球的直径，求圆柱与球的体积比

解答：

球的直径：d = 2r

圆柱的高：h = 2r

圆柱的体积：V_圆柱 = πr²h = πr²×2r = 2πr³

球的体积：V_球 = (4/3)πr³

体积比：V_圆柱 : V_球 = 2πr³ : (4/3)πr³ = 2 : 4/3 = 3 : 2

答案：3:2

例题 19：圆锥与球的体积比 【提高】

已知圆锥的底面半径为 r，球的半径也为 r，且圆锥的高为球的直径，母线与底面成 60° 角，求圆锥与球的体积比

解答：

球的直径：d = 2r

圆锥的高：h = 2r

由于母线与底面成 60° 角，所以 cos60° = r/l

1/2 = r/l，l = 2r

圆锥的体积：V_圆锥 = (1/3)πr²h = (1/3)πr²×2r = (2/3)πr³

球的体积：V_球 = (4/3)πr³

体积比：V_圆锥 : V_球 = (2/3)πr³ : (4/3)πr³ = 2 : 4 = 1 : 2

答案：1:2

例题 20：球的体积计算 【提高】

已知球的体积为 32π/3，求球的表面积

解答：

体积 = (4/3)πR³ = 32π/3

R³ = 8

R = 2

表面积 = 4πR² = 4π×4 = 16π

答案：16π

例题 21：2017年全国I卷 【提高】

已知球的表面积为 100π，求球的体积

解答：

表面积 = 4πR² = 100π

R² = 25

R = 5

体积 = (4/3)πR³ = (4/3)π×125 = (500/3)π

答案：(500/3)π

例题 22：2016年全国I卷 【提高】

已知圆柱的表面积为 24π，高为 4，求圆柱的体积

解答：

表面积 = 2πr² + 2πrh = 24π

2πr² + 2πr×4 = 24π

2πr² + 8πr = 24π

r² + 4r - 12 = 0

(r + 6)(r - 2) = 0

r = 2（舍去 r = -6）

体积 = πr²h = π×2²×4 = 16π

答案：16π

例题 23：圆锥的综合计算 【提高】

已知圆锥的底面周长为 6π，母线长为 5，求圆锥的体积

解答：

周长 = 2πr = 6π

r = 3

圆锥的高：h² = l² - r² = 5² - 3² = 25 - 9 = 16

h = 4

体积 = (1/3)πr²h = (1/3)π×3²×4 = 12π

答案：12π

例题 24：球的体积变化 【提高】

已知球的半径增加 100%，求球的体积增加的百分比

解答：

原半径：R

新半径：r = R(1 + 100%) = 2R

原体积：V₁ = (4/3)πR³

新体积：V₂ = (4/3)π(2R)³ = (4/3)π×8R³ = (32/3)πR³

体积增加：V₂ - V₁ = (32/3 - 4/3)πR³ = 28πR³/3

增加百分比：(28/3)/(4/3) = 7

= 700%

答案：700%

例题 25：几何体的综合 【提高】

已知圆柱的体积为 24π，高为 6，求圆柱的表面积

解答：

体积 = πr²h = 24π

r²×6 = 24

r² = 4

r = 2

表面积 = 2πr² + 2πrh = 2π×4 + 2π×2×6 = 8π + 24π = 32π

答案：32π
