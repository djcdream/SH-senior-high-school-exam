# 高中数学 - 第十章 复数

这是一个讲解高中复数概念的互动式教学网站。

## 📚 内容结构

### 10.1 复数的概念
- 复数的定义：形如 a + bi 的数
- 实部和虚部的概念
- 复数的分类（纯实数、纯虚数）
- 复数的几何表示

### 10.2 复数的四则运算
- 加法：(a + bi) + (c + di) = (a + c) + (b + d)i
- 减法：(a + bi) - (c + di) = (a - c) + (b - d)i
- 乘法：(a + bi)(c + di) = (ac - bd) + (ad + bc)i
- 模：|a + bi| = √(a² + b²)

### 例题练习
- 4道精选例题，包含详细的解答过程
- 交互式解答显示/隐藏功能
- 内置复数计算器

## 🔧 自定义和扩展

### 修改主题颜色
在 `styles.css` 中修改 CSS 变量：
```css
:root {
    --accent-primary: #6366f1;  /* 主色调 */
    --accent-secondary: #8b5cf6; /* 次色调 */
    /* ... */
}
```

### 添加更多例题
在 `index.html` 的 `examples` 部分添加新的 `example-box`：
```html
<div class="example-box">
    <h3>例题 X</h3>
    <p class="question">题目内容...</p>
    <button class="solution-btn" onclick="toggleSolution('solX')">查看解答</button>
    <div id="solX" class="solution">
        <p><strong>解答：</strong></p>
        <p>详细解答过程...</p>
    </div>
</div>
```

### 扩展计算器功能
在 `script.js` 中的 `calculate()` 函数中添加更多运算，如：
- 除法
- 共轭复数
- 模运算
- 幂运算
