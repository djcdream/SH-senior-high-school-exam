# 高中理科互动学习平台

这是一个现代化的、涵盖高中数学与物理核心知识点的互动式学习平台。旨在通过结构化的课程体系、交互式的学习工具和实时的进度追踪，帮助学生更高效地掌握理科知识。

## 📚 学科模块

平台包含两大核心学科模块，共计 30+ 个章节：

### 📐 高中数学
入口：`math.html`
- **涵盖内容**：集合与常用逻辑用语、一元二次函数/方程/不等式、函数的概念与性质、指数函数与对数函数、三角函数、平面向量、复数、立体几何、解析几何、概率与统计等。
- **特色章节**：
    - **第十章 复数** (`chapter10.html`)：包含复数计算器、几何意义演示。
    - **第六章 解析几何** (`chapter6.html`)：包含直线与圆的方程、圆锥曲线详解。
    - **第一章 集合** (`chapter1.html`)：集合的基本概念与运算。

### ⚡ 高中物理
入口：`physics.html`
- **涵盖内容**：运动学、力学、牛顿运动定律、功与能、动量、曲线运动、万有引力、电场、磁场、电磁感应、交变电流、热学、光学、原子物理等。
- **特色**：
    - 覆盖高中物理全阶段知识点。
    - 结构清晰的知识点梳理。

## � 核心功能

1.  **全局进度追踪**
    - 主页 (`index.html`) 提供学习仪表盘，实时显示总进度、已完成章节数和学习中状态。
    - 利用 `localStorage` 本地存储学习记录，刷新页面不丢失进度。

2.  **互动式学习体验**
    - **动态解答**：例题解答默认隐藏，点击按钮或按 `H` 键可一键切换显示/隐藏，促进独立思考。
    - **专用工具**：内置复数计算器（仅在相关章节加载），支持加减乘除运算。
    - **平滑导航**：章节内部包含平滑滚动导航，提升阅读体验。

3.  **现代化 UI 设计**
    - **深色模式**：默认采用深色主题，长时间学习护眼舒适。
    - **响应式布局**：完美适配桌面端、平板和移动端设备。
    - **视觉反馈**：包含卡片悬停效果、加载动画和操作反馈。

## 🚀 快速开始

### 本地运行（推荐）

由于使用了 ES6 模块和部分浏览器安全策略，建议通过本地服务器运行：

```bash
# 使用 Python (macOS/Linux/Windows)
python3 -m http.server 8000

# 或者使用 Node.js http-server
npx http-server
```

然后在浏览器中访问：`http://localhost:8000`

### 直接打开
也可以直接双击 `index.html` 文件在浏览器中打开（部分功能可能会受限于浏览器的本地文件安全策略）。

## 📁 项目结构

```text
project/
├── index.html          # 平台主入口（仪表盘）
├── math.html           # 数学学科主页
├── physics.html        # 物理学科主页
├── chapter*.html       # 数学各章节详情页（如 chapter10.html）
├── physics*.html       # 物理各章节详情页（如 physics1.html）
├── styles.css          # 全局样式表
├── script.js           # 全局交互逻辑（UI、计算器、事件处理）
├── data.js             # 数学数据结构与进度管理逻辑
├── physics-data.js     # 物理数据结构与进度管理逻辑
├── start.py            # Python 启动脚本（自动处理端口占用）
└── README.md           # 项目说明文档
```

## 🛠️ 技术栈

*   **HTML5**: 语义化标签构建清晰文档结构。
*   **CSS3**: Flexbox/Grid 布局，CSS 变量管理主题，CSS 动画。
*   **JavaScript (ES6+)**: 原生 JS 实现，无第三方库依赖。
    *   `IntersectionObserver` 实现滚动动画。
    *   `localStorage` 实现状态持久化。
    *   `Event Delegation` 优化事件处理。

## 📝 学习建议

1.  从 **主页** 查看整体学习规划。
2.  选择 **数学** 或 **物理** 学科进入目录。
3.  点击章节卡片进入详情页学习，完成阅读和例题后点击底部的“**完成本章学习**”按钮。
4.  随时返回主页查看更新后的学习进度。
