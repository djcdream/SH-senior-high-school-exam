6.1 行星的运动

地心说与日心说

  • 地心说：地球是宇宙的中心，太阳和其他行星都绕地球运动
  • 日心说：太阳是宇宙的中心，地球和其他行星都绕太阳运动
  • 日心说更接近实际，但也不完全正确（太阳只是太阳系的中心）

开普勒第一定律（轨道定律）

所有行星绕太阳运动的轨道都是椭圆，太阳处在椭圆的一个焦点上

开普勒第二定律（面积定律）

对任意一个行星来说，它与太阳的连线在相等的时间内扫过相等的面积

这意味着：行星在近日点速度大，在远日点速度小

开普勒第三定律（周期定律）

所有行星轨道的半长轴的三次方跟它的公转周期的二次方的比都相等

a³/T² = k（常量）

其中k的值只与中心天体有关，与行星无关

圆轨道近似

大多数行星轨道都很接近圆，因此在高中阶段通常将行星轨道视为圆

此时：a³/T² = r³/T² = k

6.2 万有引力定律

万有引力定律的发现

  • 牛顿在开普勒等前人研究的基础上发现了万有引力定律
  • 月地检验：验证了地面物体受的重力与地球吸引月球的力是同一性质的力

万有引力定律的内容

自然界中任何两个物体都相互吸引，引力的方向在它们的连线上，引力的大小与物体的质量m₁和m₂的乘积成正比，与它们之间距离r的二次方成反比

F = G(m₁m₂)/r²

G：万有引力常量，G = 6.67×10⁻¹¹ N·m²/kg²（卡文迪许测定）

适用条件

  • 质点：两物体距离远大于物体本身大小时，可视为质点
  • 均匀球体：质量分布均匀的球体，可视为质量集中在球心
  • 不适用于形状不规则或质量分布不均匀的物体

万有引力的特点

  • 普遍性：任何两个物体之间都存在
  • 相互性：两个物体之间的引力是一对作用力与反作用力
  • 宏观性：通常只有质量巨大的物体间才有明显效果
  • 特殊性：两物体间的距离趋近于零时，公式不再适用

6.3 万有引力理论的成就

计算天体质量

以地球绕太阳运动为例（将轨道视为圆）：

万有引力提供向心力：G(Mm)/r² = m(4π²r)/T²

解得中心天体质量：

M = (4π²r³)/(GT²)

其中：M为中心天体质量，r为轨道半径，T为公转周期

计算天体密度

若已知天体半径R和质量M，则密度：

ρ = M/V = M/(4πR³/3) = 3M/(4πR³)

若卫星在天体表面附近运行，r ≈ R：

ρ = 3π/(GT²)

发现未知天体

  • 海王星的发现：根据万有引力定律计算的天王星轨道与实际观测不符，推测存在未知行星，后被观测证实
  • 冥王星的发现：同样利用万有引力定律

6.4 宇宙航行

第一宇宙速度（环绕速度）

物体在地面附近绕地球做匀速圆周运动的速度叫做第一宇宙速度

由 mg = mv²/R 得：v = √(gR) = √(9.8×6.4×10⁶) ≈ 7.9km/s

这是人造地球卫星的最小发射速度，也是卫星最大的环绕速度

第二宇宙速度（脱离速度）

使物体挣脱地球引力束缚，飞离地球的最小发射速度

v₂ = 11.2km/s

第三宇宙速度（逃逸速度）

使物体挣脱太阳引力束缚，飞出太阳系的最小发射速度

v₃ = 16.7km/s

卫星的运行规律

万有引力提供向心力：G(Mm)/r² = m(v²)/r = mω²r = m(4π²r)/T²

线速度：v = √(GM/r)，轨道半径越大，线速度越小

角速度：ω = √(GM/r³)，轨道半径越大，角速度越小

周期：T = √(4π²r³/GM)，轨道半径越大，周期越大

同步卫星（地球同步轨道卫星）

  • 周期：与地球自转周期相同（T = 24h）
  • 轨道：赤道上方，轨道平面与赤道平面重合
  • 高度：固定高度（约36000km）
  • 应用：通信、气象观测等

卫星的超重与失重

  • 发射过程（加速上升）：卫星处于超重状态
  • 在轨运行：卫星处于完全失重状态
  • 返回过程（减速下降）：卫星处于超重状态

例题练习

例题 1：开普勒第三定律的应用 【基础】

两行星绕太阳做圆周运动，它们的轨道半径之比为r₁:r₂=1:4，则它们的周期之比为多少？

解答：

根据开普勒第三定律：r³/T² = k（常量）

对于两行星：r₁³/T₁² = r₂³/T₂²

T₁²/T₂² = r₁³/r₂³ = (1/4)³ = 1/64

T₁/T₂ = 1/8

答案：1:8

例题 2：万有引力的计算 【基础】

两个质量均为50kg的人相距1m，他们之间的万有引力约为多大？（G取6.67×10⁻¹¹ N·m²/kg²）

解答：

根据万有引力定律：F = G(m₁m₂)/r²

F = 6.67×10⁻¹¹ × (50×50)/(1²)

F = 6.67×10⁻¹¹ × 2500 = 1.67×10⁻⁷ N

答案：约1.67×10⁻⁷ N（非常小，可忽略）

例题 3：第一宇宙速度 【基础】

已知地球半径R=6.4×10⁶m，地面重力加速度g=10m/s²，求地球的第一宇宙速度

解答：

第一宇宙速度是物体在地面附近绕地球做匀速圆周运动的速度

由 mg = mv²/R 得：v = √(gR)

v = √(10 × 6.4×10⁶) = √(6.4×10⁷) = 8×10³ m/s = 8km/s

答案：8km/s（通常取7.9km/s）

例题 4：卫星周期与轨道半径 【中档】

人造地球卫星绕地球做匀速圆周运动，若轨道半径增大到原来的2倍，则卫星的周期变为原来的几倍？

解答：

万有引力提供向心力：G(Mm)/r² = m(4π²r)/T²

解得：T² = (4π²r³)/(GM)，即 T ∝ r^(3/2)

当r变为2r时：T' = √(2³) × T = 2√2 × T

答案：2√2倍

例题 5：【2024·新课标卷】 天体质量计算 【中档】

"嫦娥六号"探测器成功着陆月球背面，完成了人类历史首次月球背面采样返回任务。已知月球半径为R，月球表面的重力加速度为g，引力常量为G。若"嫦娥六号"在月球表面附近绕月球做匀速圆周运动，求月球的质量

解答：

在月球表面附近，万有引力近似等于重力：G(Mm)/R² = mg

解得：M = (gR²)/G

知识点标签：万有引力定律、重力与万有引力的关系

答案：M = gR²/G

例题 6：【2023·全国卷】 同步卫星 【中档】

地球同步卫星离地高度约为地球半径的6倍。若有一卫星在离地高度为地球半径的3倍的轨道上运行，则该卫星的周期与同步卫星周期的比值约为多少？

解答：

根据开普勒第三定律：r³/T² = k

同步卫星轨道半径：r₁ = R + 6R = 7R

另一卫星轨道半径：r₂ = R + 3R = 4R

r₁³/T₁² = r₂³/T₂²

T₂²/T₁² = r₂³/r₁³ = (4R)³/(7R)³ = 64/343

T₂/T₁ = √(64/343) = 8/(7√7) ≈ 8/18.5 ≈ 0.43

知识点标签：开普勒第三定律、卫星运动

答案：约0.43

例题 7：【2024·浙江卷】 天体密度计算 【中档】

某行星的半径为R，一颗卫星绕该行星做匀速圆周运动的周期为T，轨道半径为r。若已知引力常量为G，求该行星的平均密度

解答：

（1）卫星做圆周运动，万有引力提供向心力：

G(Mm)/r² = m(4π²r)/T²

（2）解得行星质量：M = (4π²r³)/(GT²)

（3）行星体积：V = (4πR³)/3

（4）行星密度：ρ = M/V = (4π²r³)/(GT²) × (3)/(4πR³) = (3πr³)/(GT²R³)

知识点标签：万有引力定律、天体密度

答案：ρ = 3πr³/(GT²R³)

例题 8：【2022·新课标卷】 双星系统 【高档】

宇宙中存在双星系统，两恒星质量分别为m₁和m₂，相距L，绕它们连线上的某点做匀速圆周运动。求两恒星的轨道半径r₁和r₂，以及运动周期T

解答：

（1）两恒星间的万有引力提供向心力，且角速度相同：

G(m₁m₂)/L² = m₁ω²r₁ = m₂ω²r₂

（2）由 r₁ + r₂ = L，且 m₁r₁ = m₂r₂，解得：

r₁ = (m₂L)/(m₁+m₂)，r₂ = (m₁L)/(m₁+m₂)

（3）由 G(m₁m₂)/L² = m₁(4π²r₁)/T²，解得：

T = √[4π²L³/(G(m₁+m₂))]

知识点标签：双星系统、万有引力、圆周运动

答案：r₁ = m₂L/(m₁+m₂)，r₂ = m₁L/(m₁+m₂)，T = √[4π²L³/G(m₁+m₂)]

例题 9：【2023·河北卷】 卫星变轨 【高档】

卫星在圆轨道I上运行，半径为r₁，速度为v₁。若要使卫星进入半径为r₂（r₂ > r₁）的圆轨道II，需要在A点通过加速实现。求在圆轨道II上的速度v₂

解答：

（1）在圆轨道I上：G(Mm)/r₁² = mv₁²/r₁，v₁ = √(GM/r₁)

（2）在圆轨道II上：G(Mm)/r₂² = mv₂²/r₂

（3）解得：v₂ = √(GM/r₂)

（4）由于r₂ > r₁，所以v₂ < v₁

（5）从轨道I进入轨道II需要在A点加速，但加速后的椭圆轨道在远地点的速度会小于圆轨道II的速度，需要再次加速才能进入圆轨道II

知识点标签：卫星变轨、圆周运动、能量守恒

答案：v₂ = √(GM/r₂)

例题 10：【2024·新课标卷】 火星探测 【中档】

"天问一号"火星探测器成功着陆火星。已知火星质量为M，半径为R，引力常量为G。求火星表面的重力加速度g和火星的第一宇宙速度v

解答：

（1）在火星表面，万有引力等于重力：

G(Mm)/R² = mg

解得：g = (GM)/R²

（2）第一宇宙速度是卫星在火星表面附近绕火星做匀速圆周运动的速度：

mg = mv²/R

解得：v = √(gR) = √(GM/R)

知识点标签：万有引力定律、第一宇宙速度

答案：g = GM/R²，v = √(GM/R)

例题 11：【2023·新课标卷】 卫星加速度比较 【基础】

两颗卫星P、Q绕地球做匀速圆周运动，轨道半径r_P < r_Q。下列说法正确的是：

(A) P的周期大于Q的周期

(B) P的线速度大于Q的线速度

(C) P的角速度大于Q的角速度

(D) P的向心加速度大于Q的向心加速度

解答：

根据 G(Mm)/r² = m(4π²r)/T² = mv²/r = mω²r = ma，得：

（A）错误。T = √(4π²r³/GM)，r_P < r_Q，所以T_P < T_Q

（B）正确。v = √(GM/r)，r_P < r_Q，所以v_P > v_Q

（C）正确。ω = √(GM/r³)，r_P < r_Q，所以ω_P > ω_Q

（D）正确。a = GM/r²，r_P < r_Q，所以a_P > a_Q

知识点标签：卫星运动、圆周运动规律

答案：B、C、D

例题 12：【2024·江西卷】 航天飞机的能量 【高档】

航天飞机在半径为r的轨道上绕地球飞行。若要将航天飞机转移到半径为2r的轨道上，需要通过加速实现。已知地球质量为M，引力常量为G，航天飞机质量为m。求两次加速所需的总能量

解答：

（1）在半径为r的轨道上，速度v₁ = √(GM/r)，总能量E₁ = -GMm/(2r)

（2）在半径为2r的轨道上，速度v₂ = √(GM/2r)，总能量E₂ = -GMm/(4r)

（3）能量变化：ΔE = E₂ - E₁ = -GMm/(4r) - (-GMm/(2r)) = GMm/(4r)

（4）实际操作中需要先加速进入椭圆轨道，再加速进入圆轨道，总能量等于两轨道能量之差

知识点标签：航天工程、能量守恒、轨道变换

答案：GMm/(4r)

例题 13：【2022·浙江卷】 月球引力波探测 【中档】

"鹊桥"中继星在地月拉格朗日L2点附近运行，为月球背面的着陆器提供中继通信服务。已知地球质量为M，月球质量为m，地月距离为L。若"L2点"位于月球外侧距离月球x处，在此处地球和月球对中继星的引力合力为零，求x的值

解答：

（1）在L2点，中继星受地球引力和月球引力，合力为零：

G(Mm_卫)/(L+x)² = G(mm_卫)/x²

（2）化简得：M/(L+x)² = m/x²

（3）展开：Mx² = m(L+x)² = m(L² + 2Lx + x²)

（4）整理：(M-m)x² - 2mLx - mL² = 0

（5）解方程（取正值）得近似值（当M≫m时）：x ≈ L√(m/M)

知识点标签：拉格朗日点、万有引力平衡

答案：x满足(M-m)x² - 2mLx - mL² = 0

例题 14：【2023·全国卷】 行星宜居带 【中档】

恒星宜居带是指恒星周围存在液态水的区域。若某恒星质量为太阳质量的2倍，宜居带范围与轨道半径有关。已知地球绕太阳公转周期为1年，轨道半径为R。求该恒星宜居带内一颗行星绕该恒星公转的周期与地球绕太阳公转周期的比值（设行星轨道半径为1.5R）

解答：

（1）地球绕太阳：GM_日m/R² = m(4π²R)/T_日²

（2）行星绕恒星：G(2M_日)m'/(1.5R)² = m'(4π²×1.5R)/T_行²

（3）两式相除：M/R³ : 2M/(1.5R)³ = 1/T_日² : 1/T_行²

（4）化简：T_行²/T_日² = (1.5)³/2 = 3.375/2 = 1.6875

（5）T_行/T_日 = √1.6875 ≈ 1.3

知识点标签：开普勒第三定律、宜居带

答案：约1.3

例题 15：【2024·河北卷】 空间站 【基础】

中国空间站绕地球做匀速圆周运动，轨道高度约为400km。已知地球半径R=6400km，地面重力加速度g=10m/s²。求空间站绕地球运行的周期

解答：

（1）空间站轨道半径：r = R + h = 6400km + 400km = 6800km = 6.8×10⁶m

（2）在空间站轨道上：G(Mm)/r² = m(4π²r)/T²

（3）在地球表面：GM = gR²

（4）联立得：(gR²)/r² = (4π²r)/T²

（5）解得：T² = (4π²r³)/(gR²) = (4π²×6.8³×10¹⁸)/(10×6.4²×10¹²)

T ≈ √(3.05×10⁷) ≈ 5525s ≈ 92min

知识点标签：空间站、卫星周期

答案：约92分钟
