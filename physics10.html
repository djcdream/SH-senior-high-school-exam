第十章 机械波

知识点1

基本概念

机械波：机械振动在介质中传播形成的波

横波：质点的振动方向与波的传播方向垂直的波，如绳波

纵波：质点的振动方向与波的传播方向平行的波，如声波

波长：λ，波在一个周期内传播的距离，单位是米(m)

波速：v，波在介质中传播的速度，单位是米每秒(m/s)

知识点2

重要规律

波长、频率和波速的关系：v = λf = λ/T

波的叠加原理：两列波相遇时，介质中质点的位移等于两列波单独传播时引起的位移的矢量和

波的干涉：频率相同的两列波叠加时，形成稳定的振动加强区和减弱区

波的衍射：波遇到障碍物时，能绕过障碍物继续传播的现象

声波：频率在20Hz-20000Hz之间的纵波

例题练习

例题 1：【2024·新课标卷】 波长、频率和波速 【基础】

一列波在空气中传播，波速为340m/s，频率为500Hz。求这列波的波长。

解答：

（1）波长、频率和波速的关系：v = λf

（2）解得：λ = v/f = 340/500 = 0.68m

知识点标签：波长、频率、波速

答案：0.68m

例题 2：【2023·浙江卷】 横波和纵波 【基础】

下列波中属于横波的是：

(A) 水波

(B) 声波

(C) 地震波

(D) 绳波

解答：

（A）水波既有横波成分也有纵波成分，不是纯粹的横波

（B）错误。声波是纵波，质点的振动方向与波的传播方向平行

（C）错误。地震波既有横波也有纵波

（D）正确。绳波是典型的横波，质点的振动方向与波的传播方向垂直

知识点标签：横波、纵波

答案：D

例题 3：【2022·新课标卷】 波的图像 【中档】

一列波沿x轴传播，某时刻的波形图如图所示，若波的传播速度为v，求该波的波长和周期。

解答：

（1）从波形图中读出波长：两个相邻的波峰或波谷之间的距离为λ

（2）波长、周期和波速的关系：λ = vT

（3）周期：T = λ/v

（4）频率：f = 1/T = v/λ

（5）波形图反映的是某一时刻各个质点的位移情况

知识点标签：波的图像、波长、周期

答案：波长 = λ，周期 = T = λ/v

例题 4：【2024·河北卷】 波的叠加 【中档】

两列频率相同、振幅相同的波在介质中传播，在相遇点形成振动加强区。求该点到两波源的距离之差。

解答：

（1）波的叠加原理：介质中质点的位移等于两列波单独传播时引起的位移的矢量和

（2）振动加强区：两列波在该点引起的振动同相，即相位差为2π的整数倍

（3）相位差与波程差的关系：Δφ = 2π·Δr/λ

（4）振动加强条件：Δφ = 2π·n（n = 0, 1, 2, ...）

（5）所以波程差：Δr = nλ（n为整数）

知识点标签：波的叠加、干涉、振动加强

答案：Δr = nλ（n = 0, 1, 2, ...）

例题 5：【2023·全国卷】 波的干涉 【高档】

两个相干波源S₁和S₂，频率相同，初相相同，相距2d，在它们连线的垂直平分线上一点P，求该点的振动情况。

解答：

（1）设S₁和S₂到P点的距离分别为r₁和r₂

（2）由对称性，r₁ = r₂

（3）波程差：Δr = r₂ - r₁ = 0

（4）相位差：Δφ = 2π·Δr/λ = 0

（5）两列波在P点同相到达，位移相加，振动加强

（6）P点是振动加强点，振幅为两列波振幅之和

知识点标签：波的干涉、相干波源、振动加强

答案：P点是振动加强点，振幅最大

例题 6：【2024·辽宁卷】 波的衍射 【中档】

关于波的衍射，下列说法正确的是：

(A) 波遇到障碍物时，一定能绕过障碍物继续传播

(B) 波的衍射现象只有在波长与障碍物尺寸相当时才明显

(C) 波的衍射现象只有在波长远大于障碍物尺寸时才明显

(D) 波的衍射现象与波长和障碍物尺寸无关

解答：

（A）错误。波的衍射是否明显与波长和障碍物尺寸有关

（B）正确。波的衍射现象在波长与障碍物尺寸相当时比较明显

（C）错误。不是波长远大于障碍物尺寸时明显

（D）错误。波的衍射现象与波长和障碍物尺寸有关

（5）当波长远大于障碍物尺寸时，衍射现象非常明显；当波长远小于障碍物尺寸时，衍射现象不明显

知识点标签：波的衍射、波长、障碍物

答案：B

例题 7：【2022·浙江卷】 声波 【中档】

一列声波在空气中传播，波长为0.5m，波速为340m/s。求该声波的频率。

解答：

（1）波长、频率和波速的关系：v = λf

（2）解得：f = v/λ = 340/0.5 = 680Hz

（3）声波的频率在20Hz-20000Hz之间，人耳可以听到

（4）680Hz在人耳的听觉范围内，可以听到

知识点标签：声波、频率、波长

答案：680Hz

例题 8：【2023·河北卷】 波的多普勒效应 【高档】

一辆警车以速度v₁向静止的观察者驶来，警车发出的警报声频率为f₀。若声波的速度为v，求观察者接收到的频率f'。

解答：

（1）波源向观察者运动时，波长缩短：λ' = (v - v₁)/f₀

（2）波速仍为v，观察者接收到的频率：f' = v/λ'

（3）代入λ'得：f' = v/[(v - v₁)/f₀] = v·f₀/(v - v₁)

（4）由于v - v₁ < v，所以f' > f₀，接收到的频率升高

（5）这称为多普勒效应

知识点标签：多普勒效应、波源运动、频率变化

答案：f' = v·f₀/(v - v₁)

例题 9：【2024·新课标Ⅱ卷】 驻波 【中档】

一列波沿x轴正方向传播，波形图如图所示。若要使该波形成驻波，遇到障碍物反射后的波与入射波的叠加条件是什么？

解答：

（1）驻波是两列振幅相同、频率相同、传播方向相反的波叠加形成的

（2）驻波的波节（振幅为零的点）和波腹（振幅最大的点）的位置固定不动

（3）相邻波节之间的距离为λ/2

（4）相邻波腹之间的距离也是λ/2

（5）形成驻波的条件：两列波频率相同、振幅相同、传播方向相反

（6）反射波与入射波在介质中形成驻波

知识点标签：驻波、波反射、波的叠加

答案：两列波频率相同、振幅相同、传播方向相反

例题 10：【2022·全国卷】 波的能量传递 【中档】

一列波在介质中传播，波的振幅为A，周期为T。求一个周期内波传播的距离和波传递的能量。

解答：

（1）波在一个周期内传播的距离等于波长：Δx = λ

（2）根据波长、周期和波速的关系：λ = vT

（3）所以：Δx = vT

（4）波传播时伴随能量的传递，一个周期内传递的能量E = ½kA²T（k是等效劲度系数）

（5）波的能量密度与振幅的平方成正比

知识点标签：波的传播、波长、能量传递

答案：Δx = vT，E ∝ A²

例题 11：【2023·江苏卷】 横波图像 【中档】

一列横波沿x轴正方向传播，t=0时刻的波形图如图所示。若该波的周期为T，求t=T/4时刻的波形图。

解答：

（1）波沿x轴正方向传播，经过时间t，波形沿传播方向移动距离：Δx = vt

（2）t = T/4时，Δx = v·(T/4) = (λ/T)·(T/4) = λ/4

（3）波形图沿x轴正方向平移λ/4

（4）质点的振动方向：若沿x轴正方向传播，则波形图右侧的质点向上运动，左侧的质点向下运动

（5）根据波的传播方向和质点振动方向确定波形的变化

知识点标签：横波图像、波的传播、周期

答案：波形图沿x轴正方向平移λ/4

例题 12：【2024·江西卷】 波的反射 【中档】

一列波垂直射向两种介质的界面，入射波从波速较小的介质射向波速较大的介质。求反射后波的相位变化。

解答：

（1）波从波速较小的介质射向波速较大的介质，称为"波疏介质"

（2）反射时，发生相位突变，相位改变π（相当于波峰变为波谷）

（3）这种现象称为"半波损失"

（4）反射波与入射波在界面处反相

（5）若波从波速较大的介质射向波速较小的介质，反射波不发生相位突变

知识点标签：波的反射、相位变化、半波损失

答案：反射波的相位改变π

例题 13：【2022·河北卷】 声音 【基础】

在音乐厅中，为什么在座位上也能清楚地听到舞台上的乐器声？

解答：

（1）声波是纵波，可以在空气中传播

（2）声波的传播方式：舞台上的乐器振动产生声波，声波向四周传播

（3）音乐厅的设计有利于声波的传播和反射，使声音均匀分布

（4）声波的反射和衍射现象使得座位上也能听到乐器声

（5）声波的干涉和驻波现象也会影响声音的分布

知识点标签：声波、声音的传播、声学

答案：声波在空气中传播，通过反射和衍射到达座位

例题 14：【2023·新课标卷】 波的强度 【中档】

一列波的振幅为A，波速为v。若将振幅增大为原来的2倍，其他条件不变，则波的强度变为原来的几倍？

解答：

（1）波的强度I与振幅A的平方成正比：I ∝ A²

（2）原强度：I₁ ∝ A²

（3）振幅变为2A后，新强度：I₂ ∝ (2A)² = 4A²

（4）强度之比：I₂/I₁ = 4A²/A² = 4

（5）波的强度还与波速、频率有关，但本例中只有振幅变化

知识点标签：波的强度、振幅

答案：4倍

例题 15：【2024·全国卷】 波的衍射和干涉 【高档】

一列平面波通过宽缝，缝宽为d，波长为λ。若缝宽d与波长λ的关系满足d = λ，在缝后多远处能观察到明显的衍射现象？

解答：

（1）波的衍射现象与缝宽d和波长λ的比值有关

（2）当d ≈ λ时，衍射现象最明显

（3）当d >> λ时，衍射现象不明显，近似沿直线传播

（4）当d << λ时，衍射现象非常明显，波可以绕过缝向各个方向传播

（5）本题中d = λ，属于"缝宽与波长相当"的情况，衍射现象比较明显

（6）在缝后一定距离（远大于缝宽）处都能观察到明显的衍射现象

知识点标签：波的衍射、缝宽、波长

答案：当d = λ时，在缝后较远处都能观察到明显的衍射现象
