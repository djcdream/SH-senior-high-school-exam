3.1 重力

重力的定义

由于地球的吸引而使物体受到的力叫做重力

重力公式

G = mg

G：重力，m：质量，g：重力加速度（约9.8N/kg）

重力的方向

  • 竖直向下（不是垂直向下）
  • 重垂线：利用重力的方向竖直向下的原理制成

重心

物体各部分所受重力的等效作用点叫做重心

  • 质量分布均匀、形状规则的物体，重心在几何中心
  • 质量分布不均匀或形状不规则的物体，重心可用悬挂法确定
  • 重心可能在物体上，也可能在物体外

3.2 弹力

弹力的定义

物体发生弹性形变后，由于要恢复原状，对与它接触的物体产生的力叫做弹力

弹力产生的条件

  • 物体直接接触
  • 发生弹性形变

胡克定律

F = kx

F：弹力，k：劲度系数，x：形变量

在弹性限度内，弹力的大小与形变量成正比

弹力的方向

  • 压力和支持力：垂直于接触面，指向被压或被支持的物体
  • 绳子的拉力：沿绳子方向，指向绳子收缩的方向
  • 弹簧的弹力：沿弹簧轴线，与形变方向相反

3.3 摩擦力

静摩擦力

两个相互接触的物体，当它们发生相对运动趋势时，在接触面上产生的阻碍相对运动趋势的力叫做静摩擦力

  • 产生条件：接触面粗糙、有正压力、有相对运动趋势
  • 方向：与相对运动趋势方向相反
  • 大小：0 ≤ f ≤ fmax，根据平衡条件或牛顿第二定律计算

滑动摩擦力

两个相互接触的物体，当它们发生相对滑动时，在接触面上产生的阻碍相对运动的力叫做滑动摩擦力

  • 产生条件：接触面粗糙、有正压力、发生相对滑动
  • 方向：与相对运动方向相反

滑动摩擦力公式

f = μN

f：滑动摩擦力，μ：动摩擦因数，N：正压力

3.4 力的合成与分解

合力与分力

如果一个力产生的效果跟几个力共同产生的效果相同，这个力就叫做那几个力的合力，那几个力就叫做这个力的分力

平行四边形定则

求两个互成角度的力的合力，可以用表示这两个力的线段为邻边，作平行四边形，这两个邻边之间的对角线就表示合力的大小和方向

力的合成的特殊情况

同向

F合 = F₁ + F₂

方向与两个力相同

反向

F合 = |F₁ - F₂|

方向与较大的力相同

力的分解

力的分解是力的合成的逆运算，同样遵守平行四边形定则

  • 如果没有限制，一个力可以分解为无数对分力
  • 在实际问题中，要根据力的实际作用效果进行分解
  • 常用方法：正交分解法

正交分解法

将力沿两个互相垂直的方向分解

Fx = Fcosθ
Fy = Fsinθ

合力大小：F = √(Fx² + Fy²)

例题练习

例题 1：重心的位置 【基础】

关于重心，下列说法正确的是：

(A) 重心就是物体的几何中心

(B) 重心一定在物体上

(C) 物体的重心位置与质量分布和形状有关

(D) 重心位置随物体放置方式不同而改变

解答：

(A) 错误。只有质量分布均匀、形状规则的物体，重心才在几何中心

(B) 错误。重心可能在物体外（如圆环）

(C) 正确。重心位置确实与质量分布和形状有关

(D) 错误。重心位置不随放置方式改变

答案：C

例题 2：胡克定律的应用 【基础】

一根弹簧原长20cm，挂5N重物时长25cm，挂10N重物时长多少？（在弹性限度内）

解答：

根据胡克定律 F = kx

挂5N时：x₁ = 25cm - 20cm = 5cm = 0.05m

k = F₁/x₁ = 5/0.05 = 100N/m

挂10N时：x₂ = F₂/k = 10/100 = 0.1m = 10cm

弹簧长度 = 原长 + 形变量 = 20 + 10 = 30cm

答案：30cm

例题 3：静摩擦力的判断 【基础】

用手握住瓶子竖直悬挂，瓶子和手都保持静止。当增大握力时，瓶子受到的摩擦力如何变化？

解答：

瓶子静止，受力平衡

竖直方向：静摩擦力 = 重力

增大握力，只是增大了最大静摩擦力

由于瓶子仍静止，静摩擦力仍等于重力，不变

答案：摩擦力不变

例题 4：滑动摩擦力的计算 【基础】

质量为2kg的物体在水平面上滑行，已知动摩擦因数μ = 0.2，求滑动摩擦力大小（g取10m/s²）

解答：

物体在水平面上，正压力 N = mg = 2×10 = 20N

滑动摩擦力 f = μN = 0.2×20 = 4N

答案：4N

例题 5：力的合成 【中档】

两个共点力F₁=3N，F₂=4N，求合力的最大值和最小值

解答：

合力最大：当两个力同向时

Fmax = F₁ + F₂ = 3 + 4 = 7N

合力最小：当两个力反向时

Fmin = |F₁ - F₂| = |3 - 4| = 1N

答案：最大值7N，最小值1N

例题 6：力的正交分解 【中档】

一个力F=10N，方向与水平方向成30°角斜向上，求这个力的水平分量和竖直分量

解答：

水平分量：Fx = Fcos30° = 10 × (√3/2) ≈ 8.66N

竖直分量：Fy = Fsin30° = 10 × 0.5 = 5N

答案：水平分量约8.66N，竖直分量5N

例题 7：【2024·新课标卷】 平抛运动与动能 【中档】

福建舰是我国自主设计建造的首艘弹射型航空母舰。借助配重小车可以进行弹射测试，测试时配重小车被弹射器从甲板上水平弹出后，落到海面上。调整弹射装置，使小车水平离开甲板时的动能变为调整前的4倍。忽略空气阻力，则小车在海面上的落点与其离开甲板处的水平距离为调整前的几倍？

解答：

（1）动能表达式为Ek = ½mv²

（2）由题意可知小车水平离开甲板时的动能变为调整前的4倍，则小车离开甲板时速度变为调整前的2倍

（3）小车离开甲板后做平抛运动，从离开甲板到到达海面上时间不变（h = ½gt²，高度不变）

（4）根据x = vt，可知小车在海面上的落点与其离开甲板处的水平距离为调整前的2倍

知识点标签：平抛运动、动能定理、运动合成与分解

答案：2倍

例题 8：【2023·浙江卷】 光滑圆柱体平衡 【中档】

水平面上固定两排平行的半圆柱体，重为G的光滑圆柱体静置其上，a、b为相切点，∠aOb=90°，半径Ob与重力的夹角为37°。已知sin37°=0.6，cos37°=0.8，则圆柱体受到的支持力Fa、Fb大小为多少？

解答：

对光滑圆柱体受力分析，圆柱体受重力G和两个支持力Fa、Fb

由题意有：Fa = Gsin37° = 0.6G

Fb = Gcos37° = 0.8G

知识点标签：力的合成与分解、平衡条件

答案：Fa = 0.6G，Fb = 0.8G

例题 9：【2022·全国卷】 滑雪大跳台 【高档】

北京2022年冬奥会首钢滑雪大跳台，运动员从a处由静止自由滑下，到b处起跳，c点为a、b之间的最低点，a、c两处的高度差为h。要求运动员经过c点时对滑雪板的压力不大于自身所受重力的k倍，运动过程中将运动员视为质点并忽略所有阻力，则c点处这一段圆弧雪道的半径不应小于多少？

解答：

（1）运动员从a到c过程，由动能定理得：mgh = ½mv²

（2）在c点，由牛顿第二定律得：FN - mg = mv²/R

（3）要求FN ≤ kmg，即压力不大于k倍重力

（4）联立得：R ≥ h/(k-1)

知识点标签：牛顿第二定律、动能定理、圆周运动

答案：R ≥ h/(k-1)

例题 10：【2024·江西卷】 带电小球平衡 【高档】

垂直于水平桌面固定一根轻质绝缘细直杆，质量均为m、带同种电荷的绝缘小球甲和乙穿过直杆，两小球均可视为点电荷，带电荷量分别为q和Q。在图示的坐标系中，小球乙静止在坐标原点，初始时刻小球甲从x=x₀处由静止释放，开始向下运动。关于小球甲的运动情况，下列说法正确的是？

解答：

（1）两球带同种电荷，存在库仑斥力

（2）小球甲受重力、库仑斥力、杆的弹力作用

（3）释放后，甲在重力作用下加速下滑

（4）随着距离减小，库仑斥力增大，加速度减小

（5）当mg = F库时，速度最大

（6）之后减速，最终在平衡位置附近振动

知识点标签：库仑力、力的平衡、动力学分析

答案：先加速后减速，存在最大速度

例题 11：【2023·河北卷】 摩擦力判断 【基础】

一粗糙斜面固定在地面上，斜面顶端装有一光滑定滑轮。一细绳跨过滑轮，其一端悬挂物块N，另一端与斜面上的物块M相连，系统处于静止状态。现用水平向左的拉力缓慢拉动N，直至悬挂N的细绳与竖直方向成45°，M始终保持静止。在此过程中，M所受斜面的摩擦力大小如何变化？

解答：

（1）对N受力分析：受重力mg、拉力T、水平力F，三力平衡

（2）由矢量三角形可知，T随细绳偏角增大而增大

（3）对M受力分析：摩擦力f的方向取决于T与Mgsinθ的大小关系

（4）若初始T < Mgsinθ（f沿斜面向上），则T增大时f先减小至零，再反向增大

（5）若初始T > Mgsinθ（f沿斜面向下），则f随T增大而增大

知识点标签：静摩擦力、动态平衡、力的分析

答案：可能先减小后增大，或一直增大

例题 12：【2024·河北卷】 核电池衰变 【基础】

2024年我国研发出一款安全性高、稳定发电时间长的新微型原子能电池。该电池将⁶³Ni衰变释放的能量转化为电能，衰变方程为⁶³Ni → ⁶³Cu + X + v̅e，式中v̅e是质量可忽略不计的中性粒子，则X表示的是什么粒子？

解答：

（1）根据质量数守恒：63 = 63 + 质量数(X) + 0，故质量数(X) = 0

（2）根据电荷数守恒：28 = 29 + 电荷数(X) + 0，故电荷数(X) = -1

（3）质量数为0、电荷数为-1的粒子是电子（⁰₋₁e）

知识点标签：核反应、质量数与电荷数守恒

答案：电子（⁰₋₁e）

例题 13：【2023·全国卷】 氢原子能级 【基础】

氢原子能级示意图如图所示，光子能量在1.63eV～3.10eV的光为可见光。要使处于基态（n=1）的氢原子被激发后可辐射出可见光光子，最少应给氢原子提供的能量为多少？

解答：

（1）光子能量在1.63eV～3.10eV的光为可见光

（2）基态（n=1）能量为-13.60eV

（3）激发至n=3：能量为-1.51eV，需要能量E = (-1.51) - (-13.60) = 12.09eV

（4）从n=3跃迁到n=2时释放能量ΔE = (-1.51) - (-3.40) = 1.89eV（可见光）

（5）激发至n=2只能释放10.20eV（紫外光），无可见光

知识点标签：能级跃迁、光子能量

答案：12.09eV（选项A）

例题 14：【2024·新课标卷】 电磁感应 【高档】

电动汽车制动时可利用车轮转动将其动能转换成电能储存起来。车轮转动时带动磁极绕固定的线圈旋转，在线圈中产生电流。磁极匀速转动的某瞬间，磁场方向恰与线圈平面垂直。将两磁极间的磁场视为匀强磁场，则磁极再转过90°时，线圈中电流如何变化？

解答：

（1）题图中线圈处于中性面位置，磁通量最大，磁通量变化率为零，感应电动势为零

（2）当磁极再转过90°时，穿过线圈的磁通量为0，磁通量变化率最大

（3）感应电动势最大，电流最大

（4）磁极转过90°时，根据右手定则，感应电流方向由Q指向P

知识点标签：电磁感应、楞次定律、交流电

答案：电流最大，方向由Q指向P

例题 15：【2023·新课标卷】 动量定理 【高档】

长征九号研制的大推力型火箭发动机联试成功。假设某次试验中该发动机向后喷射的气体速度约为3km/s，产生的推力约为4.8×10⁸N，则它在1s时间内喷射的气体质量约为多少？

解答：

（1）火箭推力F等于单位时间内喷射气体的动量变化率

（2）F = Δp/Δt = (Δm·v)/Δt

（3）Δm/Δt = F/v = 4.8×10⁸N / (3×10³m/s) = 1.6×10⁵kg/s

（4）1s时间内喷射的气体质量为1.6×10⁵kg = 1.6×10²t

知识点标签：动量定理、冲量、航天工程

答案：1.6×10⁵kg
