知识点1

基本概念

波粒二象性：光和实物粒子既具有波动性，又具有粒子性

光子说：爱因斯坦提出，光是光子流，每个光子的能量E = hν，动量p = h/λ

德布罗意波：实物粒子也具有波动性，波长λ = h/p，其中h是普朗克常量

物质波：实物粒子的波，描述微观粒子的波动特性

知识点2

重要规律

不确定性原理：海森堡提出，不可能同时精确测量粒子的位置和动量

海森堡不等式：Δx·Δp ≥ h/(4π)

波函数：描述微观粒子状态的函数，|ψ|²表示粒子在某处出现的概率

薛定谔方程：量子力学的基本方程，描述波函数随时间的演化

隧穿效应：粒子能够穿透势垒，这是量子效应

例题练习

例题 1：【2024·新课标卷】 光子说 【基础】

一单色光的频率为ν，求该光子的能量和动量。已知普朗克常量h。

解答：

（1）光子能量：E = hν

（2）光子动量：p = h/λ = hν/c

（3）其中c是真空中的光速

（4）光子的能量与频率成正比

（5）光子的动量与能量成正比

知识点标签：光子说、光子能量、光子动量

答案：E = hν，p = hν/c

例题 2：【2023·浙江卷】 德布罗意波 【中档】

一电子的质量为m，速度为v。求该电子的德布罗意波长。

解答：

（1）电子的动量：p = mv

（2）德布罗意波长：λ = h/p = h/(mv)

（3）其中h = 6.63×10⁻³⁴J·s是普朗克常量

（4）德布罗意波长与动量成反比

（5）电子具有波动性，称为物质波

知识点标签：德布罗意波、物质波、电子波动性

答案：λ = h/(mv)

例题 3：【2022·新课标卷】 不确定性原理 【中档】

一电子的位置不确定量为Δx，求该电子的动量不确定量的最小值。

解答：

（1）根据海森堡不确定性原理：Δx·Δp ≥ h/(4π)

（2）动量不确定量的最小值：Δp_min = h/(4πΔx)

（3）位置测量越精确（Δx越小），动量不确定量越大

（4）无法同时精确测量位置和动量

（5）这是微观粒子的基本特性

知识点标签：不确定性原理、海森堡不等式

答案：Δp_min = h/(4πΔx)

例题 4：【2024·河北卷】 波粒二象性 【中档】

光具有波粒二象性，说明光既有波动性又有粒子性。举出光的波动性和粒子性的实验证据。

解答：

（1）光的波动性实验证据：干涉、衍射、偏振

（2）光的粒子性实验证据：光电效应、康普顿效应

（3）干涉和衍射证明光具有波动性

（4）光电效应和康普顿效应证明光具有粒子性（光子）

（5）波粒二象性是光的基本特性

（6）不同实验条件下，光表现不同的性质

知识点标签：波粒二象性、光的波动性、光的粒子性

答案：波动性：干涉、衍射；粒子性：光电效应、康普顿效应

例题 5：【2023·全国卷】 光电效应 【中档】

一金属的截止频率为ν₀，当用频率为ν（ν > ν₀）的光照射时，光电子的最大初动能是多少？

解答：

（1）根据爱因斯坦光电效应方程：hν = W + Ek_max

（2）其中W = hν₀是逸出功，Ek_max是光电子的最大初动能

（3）代入得：Ek_max = hν - hν₀ = h(ν - ν₀)

（4）当ν > ν₀时，Ek_max > 0，有光电子发射

（5）当ν = ν₀时，Ek_max = 0，刚好没有光电子

（6）当ν < ν₀时，没有光电子发射

知识点标签：光电效应、爱因斯坦方程、截止频率

答案：Ek_max = h(ν - ν₀)

例题 6：【2024·辽宁卷】 德布罗意波长 【中档】

一质子的质量为m_p，动能为Ek。求该质子的德布罗意波长。

解答：

（1）质子的动能：Ek = ½m_p² = p²/(2m_p)

（2）解得动量：p = √(2m_pEk)

（3）德布罗意波长：λ = h/p = h/√(2m_pEk)

（4）其中h = 6.63×10⁻³⁴J·s是普朗克常量

（5）动能越大，德布罗意波长越短

（6）质子具有波动性，可以用德布罗意波长描述

知识点标签：德布罗意波、物质波、质子

答案：λ = h/√(2m_pEk)

例题 7：【2022·浙江卷】 不确定度计算 【中档】

一电子被限制在长度为L的无限深势阱中。求该电子的位置不确定量和动量不确定量的最小乘积。

解答：

（1）电子被限制在长度为L的势阱中

（2）位置不确定量：Δx ≈ L

（3）根据不确定性原理：Δx·Δp ≥ h/(4π)

（4）动量不确定量：Δp ≥ h/(4πL)

（5）不确定量乘积：Δx·Δp ≥ h/(4π)

（6）这是不确定度最小值，实际不确定度可能更大

（7）限制在势阱中，位置相对确定，动量不确定度较大

知识点标签：不确定性原理、势阱、不确定度

答案：Δx·Δp ≥ h/(4π)

例题 8：【2023·河北卷】 物质波 【中档】

一质量为m的粒子，以速度v运动。若要使该粒子的德布罗意波长为λ，需要多大的速度？

解答：

（1）德布罗意波长：λ = h/p = h/(mv)

（2）解得速度：v = h/(mλ)

（3）质量越大，需要越大的速度才能达到相同的德布罗意波长

（4）波长越短，需要的速度越大

（5）这是实物粒子波动性的体现

（6）宏观粒子的德布罗意波长非常短，波动性不明显

知识点标签：德布罗意波、物质波、速度

答案：v = h/(mλ)

例题 9：【2024·新课标Ⅱ卷】 康普顿效应 【中档】

X射线光子与静止的自由电子碰撞，光子被散射，电子获得动能。求散射光子波长的变化。

解答：

（1）康普顿散射：光子与电子碰撞，光子被散射，电子获得动能

（2）能量守恒：hν₀ + m_ec² = hν' + E_e

（3）动量守恒：hν₀/c = hν'/c·cosθ + p_e·cosφ

（4）波长变化：Δλ = λ' - λ₀ = h/(m_ec)(1 - cosθ)

（5）其中θ是散射角

（6）当θ = 0时，Δλ = 0（散射波长不变）

（7）当θ = π时，Δλ = 2h/(m_ec)（波长变化最大）

（8）康普顿效应是光的粒子性的重要证据

知识点标签：康普顿效应、散射、波长变化

答案：Δλ = h/(m_ec)(1 - cosθ)

例题 10：【2022·全国卷】 光子动量 【中档】

一光子的能量为E，求该光子的动量和波长。

解答：

（1）光子能量：E = hν = hc/λ

（2）光子动量：p = E/c

（3）波长：λ = hc/E

（4）光子具有能量和动量，是光的粒子性的体现

（5）光子的能量与频率成正比，与波长成反比

（6）光子的动量与能量成正比

（7）光子静质量为零，总是以光速运动

知识点标签：光子、光子能量、光子动量、波长

答案：p = E/c，λ = hc/E

例题 11：【2023·江苏卷】 波函数 【中档】

关于波函数，下列说法正确的是：

(A) 波函数Ψ描述粒子的位置

(B) |Ψ|²表示粒子在某处出现的概率

(C) 波函数Ψ必须是实数

(D) 波函数Ψ可以是复数

解答：

（A）错误。波函数Ψ描述粒子的量子状态，不是直接描述位置

（B）正确。|Ψ|²表示粒子在某处出现的概率密度

（C）错误。波函数Ψ可以是复数

（D）正确。波函数Ψ可以是复数，只有|Ψ|²才有物理意义

（5）波函数满足归一化条件：∫|Ψ|²dx = 1

（6）波函数包含粒子的全部量子信息

知识点标签：波函数、概率密度、量子态

答案：B、D

例题 12：【2024·江西卷】 隧穿效应 【中档】

关于隧穿效应，下列说法正确的是：

(A) 粒子的能量大于势垒高度时才能发生隧穿

(B) 隧穿效应是经典现象

(C) 隧穿效应是量子效应

(D) 隧穿效应是概率性的

解答：

（A）错误。粒子的能量可以小于势垒高度，也有一定概率隧穿

（B）错误。隧穿效应是量子效应，不是经典现象

（C）正确。隧穿效应是量子效应，经典力学无法解释

（D）正确。隧穿效应是概率性的，不是确定的

（5）隧穿效应广泛应用于半导体器件

（6）扫描隧道显微镜利用隧穿效应成像

知识点标签：隧穿效应、量子效应、概率性

答案：C、D

例题 13：【2022·河北卷】 薛定谔方程 【高档】

一维无限深势阱中，粒子质量为m，势阱宽度为a。求粒子的能级和波函数。

解答：

（1）薛定谔方程：-(ℏ²/2m)·(d²ψ/dx²) + V(x)ψ = Eψ

（2）对于无限深势阱：V(x) = 0（0 < x < a），V(x) = ∞（其他区域）

（3）边界条件：ψ(0) = ψ(a) = 0

（4）解薛定谔方程得：ψ_n(x) = √(2/a)·sin(nπx/a)

（5）能级：E_n = n²π²ℏ²/(2ma²)，n = 1, 2, 3, ...

（6）基态：n = 1，E₁ = π²ℏ²/(2ma²)

（7）波函数描述了粒子在势阱中的概率分布

（8）能级是分立的，不是连续的

知识点标签：薛定谔方程、无限深势阱、能级、波函数

答案：E_n = n²π²ℏ²/(2ma²)，ψ_n(x) = √(2/a)·sin(nπx/a)

例题 14：【2023·新课标卷】 量子数 【基础】

关于量子数，下列说法正确的是：

(A) 主量子数n描述电子的能量

(B) 角量子数l描述电子的角动量

(C) 磁量子数m描述电子的自旋角动量

(D) 所有量子数都是分立的

解答：

（A）正确。主量子数n描述电子的能量，n = 1, 2, 3, ...

（B）正确。角量子数l描述电子的角动量，l = 0, 1, 2, ..., n-1

（C）正确。磁量子数m描述电子的自旋角动量，m = -l, ..., +l

（D）正确。所有量子数都是分立的，不是连续的

（5）自旋量子数s描述电子的内禀自旋

（6）量子数决定了电子的量子态

（7）泡利不相容原理：一个量子态最多容纳一个电子

知识点标签：量子数、主量子数、角量子数、磁量子数

答案：A、B、C、D

例题 15：【2024·全国卷】 量子力学综合 【高档】

一粒子在一维势场中运动，势函数为V(x) = ½kx²（谐振子）。求该粒子的能级和零点能。

解答：

（1）薛定谔方程：-(ℏ²/2m)·(d²ψ/dx²) + ½kx²ψ = Eψ

（2）引入角频率ω = √(k/m)

（3）解薛定谔方程得：ψ_n(x)和能级E_n

（4）能级：E_n = (n + ½)ℏω，n = 0, 1, 2, ...

（5）零点能：E₀ = (0 + ½)ℏω = ½ℏω

（6）零点能不为零，这是量子效应

（7）经典谐振子能量是连续的，量子谐振子能量是分立的

（8）基态能量：E₀ = ½ℏω，称为零点能

（9）即使在绝对零度，谐振子仍有零点能

知识点标签：薛定谔方程、谐振子、能级、零点能

答案：E_n = (n + ½)ℏω，E₀ = ½ℏω
