5.1 一元二次不等式

一元二次不等式的概念

形如 ax² + bx + c > 0（或 <, ≥, ≤）的不等式叫做一元二次不等式，其中 a ≠ 0

解集的判定

当 Δ > 0 时

方程 ax² + bx + c = 0 有两个不等实根 x₁, x₂（x₁ < x₂）

a > 0 时：

  • x < x₁ 或 x > x₂

a < 0 时：

  • x₁ < x < x₂

当 Δ = 0 时

方程 ax² + bx + c = 0 有两个相等实根 x₀

a > 0 时：

  • x ≠ x₀

a < 0 时：

  • ∅（空集）

当 Δ < 0 时

方程 ax² + bx + c = 0 无实根

a > 0 时：

  • R（全体实数）

a < 0 时：

  • ∅（空集）

解一元二次不等式的步骤

  1. 计算判别式 Δ = b² - 4ac
  2. 求出方程的根（如果 Δ ≥ 0）
  3. 根据判别式和二次项系数确定解集

5.2 基本不等式

基本不等式

定理

√(ab) ≤ (a + b)/2 (a, b > 0)

条件：a > 0, b > 0

等号成立条件：当且仅当 a = b 时，等号成立

几何意义

两个正数的几何平均数不大于它们的算术平均数

当且仅当两个数相等时，几何平均数等于算术平均数

基本不等式的应用

求最值

如果两个正数的积为定值，则当它们相等时，和有最小值

如果两个正数的和为定值，则当它们相等时，积有最大值

证明不等式

利用基本不等式可以证明许多重要的不等式

例如：a² + b² ≥ 2ab

常用的变形

变形1

a + b ≥ 2√(ab)

变形2

a + 1/a ≥ 2 (a > 0)

变形3

a² + 1 ≥ 2a

变形4

a/b + b/a ≥ 2 (a, b > 0)

例题练习

例题 1：一元二次不等式 【基础】

解不等式：x² - 3x + 2 > 0

解答：

步骤1：计算判别式

Δ = (-3)² - 4×1×2 = 9 - 8 = 1 > 0


步骤2：求根

x = (3 ± √1)/2 = (3 ± 1)/2

x₁ = 1，x₂ = 2


步骤3：确定解集

由于 a = 1 > 0，且 Δ > 0

解集：x < 1 或 x > 2

答案：(-∞, 1) ∪ (2, +∞)

例题 2：一元二次不等式 【基础】

解不等式：x² - 4 < 0

解答：

x² - 4 = (x - 2)(x + 2) < 0

-2 < x < 2

答案：(-2, 2)

例题 3：一元二次不等式 【基础】

解不等式：-x² + 4x - 3 > 0

解答：

两边乘以 -1：x² - 4x + 3 < 0

(x - 1)(x - 3) < 0

由于二次项系数 a = 1 > 0

解集：1 < x < 3

答案：(1, 3)

例题 4：一元二次不等式 【基础】

解不等式：2x² - 5x + 2 ≥ 0

解答：

步骤1：计算判别式

Δ = (-5)² - 4×2×2 = 25 - 16 = 9 > 0


步骤2：求根

x = (5 ± √9)/(2×2) = (5 ± 3)/4

x₁ = 2/4 = 1/2，x₂ = 8/4 = 2


步骤3：确定解集

由于 a = 2 > 0，且 Δ > 0

解集：x ≤ 1/2 或 x ≥ 2

答案：(-∞, 1/2] ∪ [2, +∞)

例题 5：一元二次不等式 【基础】

解不等式：x² + 2x + 1 < 0

解答：

步骤1：计算判别式

Δ = 2² - 4×1×1 = 4 - 4 = 0


步骤2：求根

x = -2/2 = -1


步骤3：确定解集

由于 a = 1 > 0，且 Δ = 0

解集：x ≠ -1

答案：(-∞, -1) ∪ (-1, +∞)

例题 6：一元二次不等式 【基础】

解不等式：x² + 4 < 0

解答：

步骤1：计算判别式

Δ = 0² - 4×1×4 = 0 - 16 = -16 < 0


步骤2：确定解集

由于 a = 1 > 0，且 Δ < 0

解集：∅（空集）

答案：∅

例题 7：基本不等式 【基础】

已知 a > 0，证明：a + 1/a ≥ 2

解答：

根据基本不等式：√(ab) ≤ (a + b)/2

令 b = 1/a：

√(a×1/a) ≤ (a + 1/a)/2

√1 ≤ (a + 1/a)/2

1 ≤ (a + 1/a)/2

a + 1/a ≥ 2

当且仅当 a = 1/a，即 a² = 1，a = 1 时，等号成立

证毕。

例题 8：基本不等式求最值 【基础】

已知 x > 0，且 x + 1/x = 3，求 x² + 1/x² 的值

解答：

(x + 1/x)² = x² + 2 + 1/x² = 9

x² + 1/x² = 7

答案：7

例题 9：基本不等式求最值 【基础】

已知 a > 0, b > 0，且 a + b = 4，求 ab 的最大值

解答：

根据基本不等式：√(ab) ≤ (a + b)/2

√(ab) ≤ 4/2 = 2

ab ≤ 4

当且仅当 a = b = 2 时，ab 取得最大值 4

答案：最大值4

例题 10：基本不等式求最值 【基础】

已知 x > 0，求 x + 4/x 的最小值

解答：

根据基本不等式：√(ab) ≤ (a + b)/2

令 a = x，b = 4/x：

√(x×4/x) ≤ (x + 4/x)/2

√4 ≤ (x + 4/x)/2

2 ≤ (x + 4/x)/2

x + 4/x ≥ 4

当且仅当 x = 4/x，即 x² = 4，x = 2 时，x + 4/x 取得最小值 4

答案：最小值4

例题 11：2024年新高考I卷 【中档】

解不等式：x² - x - 2 < 0

解答：

x² - x - 2 = (x - 2)(x + 1) < 0

由于二次项系数 a = 1 > 0

解集：-1 < x < 2

答案：(-1, 2)

例题 12：2023年全国甲卷 【中档】

解不等式：-2x² + 5x + 3 > 0

解答：

两边乘以 -1：2x² - 5x - 3 < 0

2x² - 5x - 3 = 0 的判别式：

Δ = (-5)² - 4×2×(-3) = 25 + 24 = 49 > 0

x = (5 ± √49)/(2×2) = (5 ± 7)/4

x₁ = (5 - 7)/4 = -2/4 = -1/2

x₂ = (5 + 7)/4 = 12/4 = 3

由于 a = 2 > 0

解集：-1/2 < x < 3

答案：(-1/2, 3)

例题 13：2022年新高考I卷 【中档】

√(ab) ≤ 6/2 = 3

ab ≤ 9

当且仅当 a = b = 3 时，ab 取得最大值 9

答案：9

例题 16：2019年全国I卷 【中档】

解不等式：-x² + 4x - 4 > 0

解答：

两边乘以 -1：x² - 4x + 4 < 0

(x - 2)² < 0

由于 (x - 2)² ≥ 0 恒成立，且 (x - 2)² < 0 永远不成立

例题 17：2018年全国II卷 【中档】

√4 ≤ (a + b)/2

2 ≤ (a + b)/2

a + b ≥ 4

当且仅当 a = b = 2 时，a + b 取得最小值 4

例题 18：2017年全国I卷 【中档】

已知 x > 0，则 2x + 8/x 的最小值是

解答：

根据基本不等式：√(ab) ≤ (a + b)/2

令 a = 2x，b = 8/x：

√(2x×8/x) ≤ (2x + 8/x)/2

√16 ≤ (2x + 8/x)/2

4 ≤ (2x + 8/x)/2

2x + 8/x ≥ 8

当且仅当 2x = 8/x，即 2x² = 8，x² = 4，x = 2 时，2x + 8/x 取得最小值 8

答案：最小值8

例题 19：不等式综合 【提高】

已知 a > 0, b > 0，且 a + b = 8，求 a² + b² 的最小值

解答：

a² + b² = (a + b)² - 2ab = 64 - 2ab

根据基本不等式：ab ≤ (a + b)²/4 = 64/4 = 16

a² + b² = 64 - 2ab ≥ 64 - 2×16 = 64 - 32 = 32

当且仅当 a = b = 4 时，a² + b² 取得最小值 32

答案：最小值32

例题 20：不等式综合 【提高】

已知 x > 0，求 x³ + 1/x³ 的最小值

解答：

(x + 1/x)³ = x³ + 3x + 3/x + 1/x³

x³ + 1/x³ = (x + 1/x)³ - 3(x + 1/x)

根据基本不等式：x + 1/x ≥ 2

令 t = x + 1/x ≥ 2

x³ + 1/x³ = t³ - 3t

求 f(t) = t³ - 3t（t ≥ 2）的最小值

f'(t) = 3t² - 3

令 f'(t) = 0：3t² - 3 = 0

t² = 1，t = ±1

由于 t ≥ 2，所以 f'(t) > 0 在 t ≥ 2 上恒成立

因此 f(t) 在 t ≥ 2 上单调递增

f(t) 的最小值为 f(2) = 2³ - 3×2 = 8 - 6 = 2

当且仅当 x = 1 时，x³ + 1/x³ 取得最小值 2

答案：最小值2

例题 21：不等式综合 【提高】

已知 a > 0, b > 0，且 a + b = 10，求 1/a + 1/b 的最小值

解答：

1/a + 1/b = (a + b)/(ab) = 10/(ab)

根据基本不等式：ab ≤ (a + b)²/4 = 100/4 = 25

1/a + 1/b = 10/(ab) ≥ 10/25 = 2/5

当且仅当 a = b = 5 时，1/a + 1/b 取得最小值 2/5

答案：最小值2/5

例题 22：不等式综合 【提高】

已知 x > 0，求 x + 16/x 的最小值

解答：

根据基本不等式：√(ab) ≤ (a + b)/2

令 a = x，b = 16/x：

√(x×16/x) ≤ (x + 16/x)/2

√16 ≤ (x + 16/x)/2

4 ≤ (x + 16/x)/2

x + 16/x ≥ 8

当且仅当 x = 16/x，即 x² = 16，x = 4 时，x + 16/x 取得最小值 8

答案：最小值8

例题 23：不等式证明 【提高】

证明：a² + b² + c² ≥ ab + bc + ca

解答：

2(a² + b² + c² - ab - bc - ca)

= (a² - 2ab + b²) + (b² - 2bc + c²) + (c² - 2ca + a²)

= (a - b)² + (b - c)² + (c - a)²

= (a - b)² + (b - c)² + (c - a)² ≥ 0

a² + b² + c² - ab - bc - ca ≥ 0

a² + b² + c² ≥ ab + bc + ca

证毕。

例题 24：不等式证明 【提高】

证明：a³ + b³ + c³ ≥ 3abc（当 a, b, c > 0）

解答：

根据基本不等式：

a³ + b³ ≥ 2a√(ab²)

b³ + c³ ≥ 2b√(bc²)

c³ + a³ ≥ 2c√(ca²)

三式相加：2(a³ + b³ + c³) ≥ 2(a√(ab²) + b√(bc²) + c√(ca²))

a³ + b³ + c³ ≥ a√(ab²) + b√(bc²) + c√(ca²)

由均值不等式：a√(ab²) + b√(bc²) + c√(ca²) ≥ 3∛(a√(ab²)×b√(bc²)×c√(ca²))

= 3∛(a³b³c³) = 3abc

a³ + b³ + c³ ≥ 3abc

证毕。
