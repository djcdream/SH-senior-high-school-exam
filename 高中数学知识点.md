📚 第一章 集合与函数

1.1 集合

集合的表示方法


列举法: 把集合中的元素一一列举出来

描述法: 用集合所含元素的共同特征表示集合

图示法: 用韦恩图表示集合


集合间的关系


子集: A⊆B，A的任何元素都是B的元素

真子集: A⊂B，A⊆B且A≠B

相等: A=B，A⊆B且B⊆A

空集: ∅，不含任何元素


集合的运算


交集: A∩B = {x|x∈A且x∈B}

并集: A∪B = {x|x∈A或x∈B}

补集: ∁UA = {x|x∈U且x∉A}


重要性质


A∩A = A, A∪A = A

A∩∅ = ∅, A∪∅ = A

德摩根定律: ∁U(A∪B) = ∁UA∩∁UB



1.2 函数及其表示

函数的三要素


定义域: 自变量x的取值范围

对应法则: f(x)的表达式

值域: 函数值f(x)的集合


函数的单调性


增函数: x₁<x₂ ⇒ f(x₁)<f(x₂)

减函数: x₁<x₂ ⇒ f(x₁)>f(x₂)


函数的奇偶性


偶函数: f(-x)=f(x)，图像关于y轴对称

奇函数: f(-x)=-f(x)，图像关于原点对称



1.3 基本初等函数

幂函数

函数y=x^a叫做幂函数，其中x是自变量，a是常数


指数函数

函数y=a^x (a>0, a≠1)叫做指数函数



定义域: R

值域: (0, +∞)

过点(0, 1)

a>1时单调递增；0<a<1时单调递减


对数函数

函数y=log_a x (a>0, a≠1)叫做对数函数



定义域: (0, +∞)

值域: R

过点(1, 0)

a>1时单调递增；0<a<1时单调递减



📐 第二章 三角函数

2.1 任意角和弧度制

弧度制


1° = π/180 rad

1 rad = 180°/π



2.2 三角函数

任意角的三角函数

设α是一个任意角，它的终边与单位圆交于点P(x, y)，则:



sinα = y

cosα = x

tanα = y/x (x≠0)


同角三角函数的基本关系


sin²α + cos²α = 1

tanα = sinα / cosα

1 + tan²α = 1/cos²α


诱导公式


sin(π - α) = sinα

cos(π - α) = -cosα

sin(π + α) = -sinα

cos(π + α) = -cosα



2.3 三角恒等变换

两角和与差的公式


sin(α±β) = sinα cosβ ± cosα sinβ

cos(α±β) = cosα cosβ ∓ sinα sinβ

tan(α±β) = (tanα ± tanβ)/(1 ∓ tanα tanβ)


二倍角公式


sin2α = 2sinα cosα

cos2α = cos²α - sin²α = 2cos²α - 1

tan2α = 2tanα/(1 - tan²α)



📊 第三章 平面向量

3.1 平面向量的线性运算

向量的加法


三角形法则: AB + BC = AC

平行四边形法则


向量的减法


AB - AC = CB


向量的数乘


|λa| = |λ|·|a|

当λ>0时，λa与a方向相同；当λ<0时，λa与a方向相反



3.2 平面向量的数量积

数量积的定义

a·b = |a||b|cosθ


数量积的性质


a·a = |a|²

a·b = b·a

(λa)·b = λ(a·b)


数量积的坐标运算

若a = (x₁, y₁), b = (x₂, y₂)，则:



a·b = x₁x₂ + y₁y₂

|a| = √(x₁²+y₁²)

a⊥b ⇔ x₁x₂ + y₁y₂ = 0



🔢 第四章 数列

4.1 等差数列

等差数列的定义

每一项与它的前一项的差等于同一个常数d


等差数列的通项公式

aₙ = a₁ + (n-1)d


等差数列的前n项和

Sₙ = n(a₁ + aₙ)/2 = na₁ + n(n-1)d/2



4.2 等比数列

等比数列的定义

每一项与它的前一项的比等于同一个常数q


等比数列的通项公式

aₙ = a₁·q^(n-1)


等比数列的前n项和

当q≠1时: Sₙ = a₁(1 - q^n)/(1 - q)
当q=1时: Sₙ = na₁



⚖️ 第五章 不等式

5.1 一元二次不等式

不等式的解集


Δ > 0: ax²+bx+c=0有两个不等实根x₁, x₂
a > 0: x < x₁或x > x₂
a < 0: x₁ < x < x₂

Δ = 0: ax²+bx+c=0有两个相等实根x₀
a > 0: x ≠ x₀
a < 0: ∅

Δ < 0: ax²+bx+c=0无实根
a > 0: R
a < 0: ∅



5.2 基本不等式

基本不等式

√(ab) ≤ (a+b)/2 (a, b > 0)


当且仅当a = b时，等号成立



📏 第六章 解析几何

6.1 直线与方程

直线的方程


点斜式: y - y₁ = k(x - x₁)

斜截式: y = kx + b

一般式: Ax + By + C = 0


两条直线的位置关系


平行: k₁ = k₂且b₁ ≠ b₂

重合: k₁ = k₂且b₁ = b₂

垂直: k₁·k₂ = -1



6.2 圆与方程

圆的标准方程

(x - a)² + (y - b)² = r²


圆的一般方程

x² + y² + Dx + Ey + F = 0



6.3 圆锥曲线

椭圆


标准方程: x²/a² + y²/b² = 1 (a > b > 0)

性质: 焦距2c (c² = a² - b²)，离心率e = c/a


双曲线


标准方程: x²/a² - y²/b² = 1 (a > 0, b > 0)

性质: 焦距2c (c² = a² + b²)，离心率e = c/a


抛物线


标准方程: y² = 2px (p > 0)

性质: 焦点F(p/2, 0)，准线x = -p/2

📈 第九章 导数及其应用

9.1 导数的定义

导数

f'(x₀) = lim[Δx→0] [f(x₀ + Δx) - f(x₀)]/Δx


导数的几何意义

函数f(x)在x = x₀处的导数是曲线y = f(x)在点(x₀, f(x₀))处的切线的斜率



9.2 导数的计算

基本初等函数的导数公式


C' = 0

(xⁿ)' = nxⁿ⁻¹

(sinx)' = cosx

(cosx)' = -sinx

(eˣ)' = eˣ

(ln x)' = 1/x


导数的四则运算法则


(u ± v)' = u' ± v'

(uv)' = u'v + uv'

(u/v)' = (u'v - uv')/v²



9.3 导数在研究函数中的应用

函数的单调性


f'(x) > 0 ⇒ f(x)单调递增

f'(x) < 0 ⇒ f(x)单调递减


函数的极值


极大值: f'(x₀) = 0且f'(x₀⁻) > 0, f'(x₀⁺) < 0

极小值: f'(x₀) = 0且f'(x₀⁻) < 0, f'(x₀⁺) > 0

10.1 复数的概念

复数

形如a + bi的数叫做复数，其中a是实部，b是虚部，i² = -1



10.2 复数的四则运算

复数的运算


(a + bi) ± (c + di) = (a ± c) + (b ± d)i

(a + bi)(c + di) = (ac - bd) + (ad + bc)i

|a + bi| = √(a² + b²)



📐 第十一章 参数方程与极坐标

11.1 参数方程

常见曲线的参数方程


直线: {x = x₀ + tcosθ, y = y₀ + tsinθ}

圆: {x = a + rcosφ, y = b + rsinφ}

椭圆: {x = acosφ, y = bsinφ}



11.2 极坐标

极坐标与直角坐标的互化


极坐标→直角坐标: x = ρcosθ, y = ρsinθ

直角坐标→极坐标: ρ = √(x² + y²), tanθ = y/x
