1.1 集合

集合的表示方法

列举法

把集合中的元素一一列举出来，写在花括号内

例：{1, 2, 3, 4}

描述法

用集合所含元素的共同特征表示集合

例：{x | x² - 1 = 0}

图示法

用韦恩图表示集合

用封闭曲线的内部表示集合

集合间的关系

子集

如果集合 A 的任何元素都是集合 B 的元素，则称 A 是 B 的子集，记作 A ⊆ B

A ⊆ B ⇔ ∀x ∈ A, x ∈ B

真子集

如果 A ⊆ B 且 A ≠ B，则称 A 是 B 的真子集，记作 A ⊂ B

A ⊂ B ⇔ A ⊆ B 且 A ≠ B

集合相等

如果 A ⊆ B 且 B ⊆ A，则称 A = B

A = B ⇔ A ⊆ B 且 B ⊆ A

空集

不含任何元素的集合叫做空集，记作 ∅

重要性质：空集是任何集合的子集

∅ ⊆ A

集合的运算

交集

A 与 B 的交集是由 A 和 B 的所有公共元素组成的集合

A ∩ B = {x | x ∈ A 且 x ∈ B}

并集

A 与 B 的并集是由 A 和 B 的所有元素组成的集合

A ∪ B = {x | x ∈ A 或 x ∈ B}

补集

设 U 是全集，A ⊆ U，则 A 的补集是 U 中所有不属于 A 的元素组成的集合

∁UA = {x | x ∈ U 且 x ∉ A}

集合运算的重要性质

基本性质

A ∩ A = A

A ∪ A = A

A ∩ ∅ = ∅

A ∪ ∅ = A

德摩根定律

∁U(A ∪ B) = ∁UA ∩ ∁UB

∁U(A ∩ B) = ∁UA ∪ ∁UB

1.2 函数及其表示

函数的三要素

定义域

自变量 x 的取值范围

分母不为零、偶次根号下非负、对数的真数大于零

对应法则

f(x) 的表达式

函数的运算规则

值域

函数值 f(x) 的集合

函数值的所有可能取值

函数的单调性

增函数

对于定义域内任意 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) < f(x₂)

x₁ < x₂ ⇒ f(x₁) < f(x₂)

图像特征：从左到右呈上升趋势

减函数

对于定义域内任意 x₁, x₂，当 x₁ < x₂ 时，都有 f(x₁) > f(x₂)

x₁ < x₂ ⇒ f(x₁) > f(x₂)

图像特征：从左到右呈下降趋势

函数的奇偶性

偶函数

对于定义域内任意 x，都有 f(-x) = f(x)

f(-x) = f(x)

图像特征：关于 y 轴对称

例：f(x) = x²

奇函数

对于定义域内任意 x，都有 f(-x) = -f(x)

f(-x) = -f(x)

图像特征：关于原点对称

例：f(x) = x³

奇偶性的判定

  1. 首先判断函数的定义域是否关于原点对称
  2. 如果定义域不关于原点对称，则函数既不是奇函数也不是偶函数
  3. 如果定义域关于原点对称，则验证 f(-x) = f(x) 或 f(-x) = -f(x)

1.3 基本初等函数

幂函数

函数 y = x^a 叫做幂函数，其中 x 是自变量，a 是常数

常见幂函数的性质

a > 0 时：在第一象限内图像过点 (0, 0) 和 (1, 1)，单调递增

a < 0 时：在第一象限内图像过点 (1, 1)，单调递减

指数函数

函数 y = a^x (a > 0, a ≠ 1) 叫做指数函数

基本性质

  • 定义域：R
  • 值域：(0, +∞)
  • 过点：(0, 1)

单调性

当 a > 1 时，函数单调递增

当 0 < a < 1 时，函数单调递减

对数函数

函数 y = log_a x (a > 0, a ≠ 1) 叫做对数函数

基本性质

  • 定义域：(0, +∞)
  • 值域：R
  • 过点：(1, 0)

单调性

当 a > 1 时，函数单调递增

当 0 < a < 1 时，函数单调递减

重要公式

log_a a = 1
log_a 1 = 0
a^log_a x = x
log_a x^n = n·log_a x

例题练习

例题 1：集合的表示 【基础】

用列举法表示集合 A = {x | x² - 3x + 2 = 0}

解答：

解方程 x² - 3x + 2 = 0

(x - 1)(x - 2) = 0

x = 1 或 x = 2

用列举法表示：A = {1, 2}

答案：{1, 2}

例题 2：集合的运算 【基础】

已知 A = {1, 2, 3, 4}，B = {2, 4, 6, 8}，求 A ∩ B 和 A ∪ B

解答：

交集 A ∩ B：

A ∩ B 是 A 和 B 的公共元素

A ∩ B = {2, 4}


并集 A ∪ B：

A ∪ B 是 A 和 B 的所有元素

A ∪ B = {1, 2, 3, 4, 6, 8}

答案：A ∩ B = {2, 4}，A ∪ B = {1, 2, 3, 4, 6, 8}

例题 3：子集的个数 【基础】

集合 A = {1, 2, 3}，求 A 的子集个数

解答：

含 n 个元素的集合，其子集个数为 2^n

集合 A 有 3 个元素

子集个数 = 2³ = 8

答案：8个

例题 4：补集运算 【基础】

设全集 U = {1, 2, 3, 4, 5, 6}，A = {1, 3, 5}，求 ∁UA

解答：

∁UA 是 U 中不属于 A 的元素

∁UA = {2, 4, 6}

答案：{2, 4, 6}

例题 5：德摩根定律 【基础】

设全集 U = {1, 2, 3, 4, 5}，A = {1, 2}，B = {2, 3}，验证 ∁U(A ∪ B) = ∁UA ∩ ∁UB

解答：

左边：∁U(A ∪ B)

A ∪ B = {1, 2} ∪ {2, 3} = {1, 2, 3}

∁U(A ∪ B) = {4, 5}


右边：∁UA ∩ ∁UB

∁UA = {3, 4, 5}

∁UB = {1, 4, 5}

∁UA ∩ ∁UB = {4, 5}


左边 = 右边，验证成立

答案：验证成立，两边都等于{4, 5}

例题 6：2024年新高考I卷 【中档】

已知集合 A = {x | x² - x - 2 < 0}，B = {x | 0 < x < 3}，则 A ∩ B =

解答：

先求集合 A：

x² - x - 2 < 0

(x - 2)(x + 1) < 0

所以 -1 < x < 2

A = {x | -1 < x < 2}


集合 B = {x | 0 < x < 3}

求交集：

A ∩ B = {x | 0 < x < 2}

答案：(0, 2)

例题 7：2023年全国甲卷 【中档】

设集合 A = {1, 2, a}，B = {a², a + 1}，若 A = B，则实数 a 的值为

解答：

由于 A = B，所以 A 和 B 的元素完全相同

集合 B 有两个元素，而 A 有三个元素

这意味着 A 中有一个元素与 B 中的元素重复


情况1：a = 1

则 A = {1, 2, 1} = {1, 2}

B = {1², 1 + 1} = {1, 2}

此时 A = B，满足条件


情况2：a = 2

则 A = {1, 2, 2} = {1, 2}

B = {2², 2 + 1} = {4, 3}

此时 A ≠ B，不满足条件


情况3：a 与 a² 或 a + 1 相等

a = a² ⇒ a(a - 1) = 0 ⇒ a = 0 或 a = 1

a = a + 1 ⇒ 0 = 1，无解

答案：a = 0 或 a = 1

例题 8：函数的定义域 【基础】

求函数 f(x) = √(x - 2) / (x - 3) 的定义域

解答：

函数有意义的条件：

  1. 偶次根号下非负：x - 2 ≥ 0 ⇒ x ≥ 2
  2. 分母不为零：x - 3 ≠ 0 ⇒ x ≠ 3

综合得：x ≥ 2 且 x ≠ 3

用区间表示：[2, 3) ∪ (3, +∞)

答案：[2, 3) ∪ (3, +∞)

例题 9：函数的奇偶性 【基础】

判断函数 f(x) = x² + 1 的奇偶性

解答：

步骤1：判断定义域

f(x) = x² + 1 的定义域为 R，关于原点对称


步骤2：验证奇偶性

f(-x) = (-x)² + 1 = x² + 1 = f(x)

由于 f(-x) = f(x)，所以 f(x) 是偶函数

答案：偶函数

例题 10：函数的单调性 【基础】

证明函数 f(x) = x + 1 在 R 上单调递增

解答：

设 x₁, x₂ 是 R 上任意两个数，且 x₁ < x₂

f(x₁) - f(x₂) = (x₁ + 1) - (x₂ + 1) = x₁ - x₂

由于 x₁ < x₂，所以 x₁ - x₂ < 0

即 f(x₁) < f(x₂)

所以 f(x) = x + 1 在 R 上单调递增

证毕。

例题 13：指数函数 【基础】

例题 17：函数综合 【提高】

已知函数 f(x) 是定义在 R 上的奇函数，当 x > 0 时，f(x) = x² + 1，求 f(-2)

解答：

由于 f(x) 是奇函数，满足 f(-x) = -f(x)

f(-2) = -f(2)

由于 2 > 0，所以 f(2) = 2² + 1 = 4 + 1 = 5

f(-2) = -f(2) = -5

答案：-5

例题 18：复合函数 【提高】

已知 f(x) = 2x + 1，求 f(f(x))

解答：

f(f(x)) = f(2x + 1)

= 2(2x + 1) + 1

= 4x + 2 + 1

= 4x + 3

答案：4x + 3

答案：x = 3
