9.1 导数的定义

导数的定义

设函数 y = f(x) 在点 x₀ 的某个邻域内有定义，当自变量 x 在点 x₀ 处取得增量 Δx 时，如果极限

f'(x₀) = lim[Δx→0] [f(x₀ + Δx) - f(x₀)]/Δx

存在，则称函数在点 x₀ 处可导，并称此极限值为函数在该点处的导数，记作 f'(x₀) 或 y'|x=x₀

导数的几何意义

函数 f(x) 在点 x₀ 处的导数 f'(x₀) 是曲线 y = f(x) 在点 (x₀, f(x₀)) 处的切线的斜率

切线方程：y - f(x₀) = f'(x₀)(x - x₀)

法线方程：y - f(x₀) = -[1/f'(x₀)](x - x₀)（当 f'(x₀) ≠ 0）

导数的物理意义

如果物体运动规律为 s = s(t)，则

  • s'(t₀) 表示物体在时刻 t₀ 的瞬时速度
  • s''(t₀) 表示物体在时刻 t₀ 的瞬时加速度

可导与连续的关系

定理：如果函数 f(x) 在点 x₀ 处可导，则函数在该点处一定连续

注意：连续不一定可导

例如：f(x) = |x| 在 x = 0 处连续但不可导

9.2 导数的计算

基本初等函数的导数公式

常数函数

C' = 0

幂函数

(xⁿ)' = nxⁿ⁻¹

正弦函数

(sinx)' = cosx

余弦函数

(cosx)' = -sinx

指数函数

(eˣ)' = eˣ

对数函数

(ln x)' = 1/x

导数的四则运算法则

和差的导数

(u ± v)' = u' ± v'

积的导数

(uv)' = u'v + uv'

商的导数

(u/v)' = (u'v - uv')/v²

复合函数的求导法则

设 y = f(u)，u = g(x)，则复合函数 y = f(g(x)) 的导数为：

y' = f'(u)·g'(x)

即：外层函数的导数 × 内层函数的导数

9.3 导数在研究函数中的应用

函数的单调性

设函数 f(x) 在某个区间内可导

增函数

f'(x) > 0 ⇒ f(x) 单调递增

减函数

f'(x) < 0 ⇒ f(x) 单调递减

常数

f'(x) = 0 ⇒ f(x) 为常数

函数的极值

设函数 f(x) 在点 x₀ 的某个邻域内有定义

极大值

对于该邻域内任意 x ≠ x₀，都有 f(x) < f(x₀)

函数在该点处取得局部最大值

极小值

对于该邻域内任意 x ≠ x₀，都有 f(x) > f(x₀)

函数在该点处取得局部最小值

极值的判定方法

第一充分条件：

  • 极大值：f'(x₀) = 0 且 f'(x₀⁻) > 0, f'(x₀⁺) < 0
  • 极小值：f'(x₀) = 0 且 f'(x₀⁻) < 0, f'(x₀⁺) > 0

第二充分条件：

  • 若 f'(x₀) = 0 且 f''(x₀) > 0，则 x₀ 为极小值点
  • 若 f'(x₀) = 0 且 f''(x₀) < 0，则 x₀ 为极大值点

函数的最值

闭区间上连续函数的最值求法：

  1. 求导数 f'(x)
  2. 求导数为0的点（驻点）和不可导点
  3. 求驻点和区间端点的函数值
  4. 比较这些函数值，最大的为最大值，最小的为最小值

例题练习

例题 1：幂函数的导数 【基础】

求下列函数的导数：y = x³

解答：

根据幂函数求导公式：(xⁿ)' = nxⁿ⁻¹

y' = (x³)' = 3x²

答案：3x²

例题 2：积的导数 【基础】

求 y = x²sinx 的导数

解答：

根据积的求导法则：(uv)' = u'v + uv'

y' = (x²)'sinx + x²(sinx)'

= 2x·sinx + x²·cosx

= 2xsinx + x²cosx

答案：2xsinx + x²cosx

例题 3：商的导数 【基础】

求 y = (x + 1)/x 的导数

解答：

解法一：使用商的求导法则

设 u = x + 1，v = x

u' = 1，v' = 1

y' = (u'v - uv')/v²

= (1·x - (x + 1)·1)/x²

= (x - x - 1)/x² = -1/x²


解法二：先化简

y = (x + 1)/x = 1 + 1/x

y' = 0 + (-1/x²) = -1/x²

答案：-1/x²

例题 4：复合函数的导数 【基础】

求 y = sin(2x) 的导数

解答：

设 u = 2x，则 y = sinu

u' = 2，y' = cosu

根据复合函数求导法则：y' = f'(u)·g'(x)

y' = cosu·u' = cos(2x)·2 = 2cos(2x)

答案：2cos(2x)

例题 5：切线方程 【基础】

求曲线 y = x² 在点 (1, 1) 处的切线方程

解答：

首先求导数：y' = (x²)' = 2x

在点 x = 1 处的导数：y'|x=1 = 2×1 = 2

切线斜率 k = 2

切线方程：y - 1 = 2(x - 1)

y = 2x - 2 + 1

y = 2x - 1

答案：y = 2x - 1

例题 6：函数的单调性 【基础】

讨论函数 y = x² - 2x 的单调性

解答：

求导数：y' = 2x - 2

令 y' > 0：2x - 2 > 0，x > 1

令 y' < 0：2x - 2 < 0，x < 1

结论：

  • 当 x < 1 时，y' < 0，函数单调递减
  • 当 x > 1 时，y' > 0，函数单调递增

例题 7：函数的极值 【基础】

求函数 y = x³ - 3x 的极值

解答：

求导数：y' = 3x² - 3

令 y' = 0：3x² - 3 = 0

x² = 1，x = ±1


极值判定：

  • 当 x = -1 时，y = (-1)³ - 3(-1) = -1 + 3 = 2

    • 在 x = -1 附近：y' 由正变负，故 x = -1 是极大值点，极大值为 2

  • 当 x = 1 时，y = 1³ - 3×1 = 1 - 3 = -2

    • 在 x = 1 附近：y' 由负变正，故 x = 1 是极小值点，极小值为 -2

答案：极大值为2，极小值为-2

例题 8：2024年新高考I卷 【中档】

已知函数 f(x) = x³ - 3x² + 2，求函数的单调区间

解答：

求导数：f'(x) = 3x² - 6x

令 f'(x) = 0：3x² - 6x = 0

3x(x - 2) = 0

x = 0 或 x = 2


列表讨论：

x (-∞, 0) 0 (0, 2) 2 (2, +∞)
f'(x) + 0 - 0 +
f(x) 增 极大值 减 极小值 增

答案：单调增区间为 (-∞, 0) 和 (2, +∞)，单调减区间为 (0, 2)

例题 9：2023年全国甲卷 【中档】

已知函数 f(x) = xeˣ，求 f(x) 在区间 [0, 1] 上的最大值和最小值

解答：

求导数：f'(x) = (x)'eˣ + x(eˣ)' = 1·eˣ + x·eˣ = eˣ(1 + x)

令 f'(x) = 0：eˣ(1 + x) = 0

由于 eˣ > 0 恒成立，所以 1 + x = 0，x = -1

但 x = -1 不在区间 [0, 1] 内


在区间 [0, 1] 内，f'(x) = eˣ(1 + x) > 0

所以 f(x) 在区间 [0, 1] 上单调递增


f(0) = 0·e⁰ = 0

f(1) = 1·e¹ = e

答案：最大值为 e，最小值为 0

例题 10：2022年新高考I卷 【中档】

曲线 y = x³ - 3x 在点 (1, -2) 处的切线斜率为

解答：

求导数：y' = 3x² - 3

在点 x = 1 处的切线斜率：y'|x=1 = 3×1² - 3 = 3 - 3 = 0

答案：0

例题 11：2021年全国乙卷 【中档】

函数 f(x) = x³ - 3x² 的极小值点的横坐标是

解答：

求导数：f'(x) = 3x² - 6x

令 f'(x) = 0：3x² - 6x = 0

3x(x - 2) = 0

x = 0 或 x = 2


极值判定：

  • 当 x = 0 时，f'(x) 由正变负，故 x = 0 是极大值点
  • 当 x = 2 时，f'(x) 由负变正，故 x = 2 是极小值点

答案：2

例题 12：2020年全国I卷 【中档】

若曲线 y = ax² + 1 在点 (1, 2) 处的切线与直线 y = 2x + 1 平行，则实数 a 的值为

解答：

首先，点 (1, 2) 在曲线上：

2 = a×1² + 1

a = 1


验证切线斜率：

y' = 2ax = 2×1×1 = 2

直线 y = 2x + 1 的斜率为 2，与切线斜率相等

答案：1

例题 13：2019年全国I卷 【中档】

函数 f(x) = ln x - x 的单调递减区间是

解答：

求导数：f'(x) = (ln x)' - (x)' = 1/x - 1

令 f'(x) < 0：1/x - 1 < 0

1/x < 1

由于 x > 0（ln x 的定义域），所以 x > 1

答案：(1, +∞)

例题 14：2018年全国II卷 【中档】

已知函数 f(x) = x³ - 3x，求 f(x) 在区间 [-2, 2] 上的最大值

解答：

求导数：f'(x) = 3x² - 3

令 f'(x) = 0：3x² - 3 = 0

x² = 1，x = ±1


计算各点函数值：

f(-2) = (-2)³ - 3(-2) = -8 + 6 = -2

f(-1) = (-1)³ - 3(-1) = -1 + 3 = 2

f(1) = 1³ - 3×1 = 1 - 3 = -2

f(2) = 2³ - 3×2 = 8 - 6 = 2


比较：max{-2, 2, -2, 2} = 2

答案：2

例题 15：含参数的函数单调性 【提高】

已知函数 f(x) = x³ - 3ax 在 R 上单调递增，求实数 a 的取值范围

解答：

求导数：f'(x) = 3x² - 3a

函数在 R 上单调递增，则 f'(x) ≥ 0 对任意 x ∈ R 成立

3x² - 3a ≥ 0

x² ≥ a

由于 x² ≥ 0 恒成立，要使 x² ≥ a 对所有 x 成立，需要 a ≤ 0

答案：a ≤ 0

例题 16：证明不等式 【提高】

证明：当 x > 0 时，ln x < x

解答：

构造函数：f(x) = x - ln x（x > 0）

求导数：f'(x) = 1 - 1/x = (x - 1)/x


讨论：

  • 当 0 < x < 1 时，x - 1 < 0，x > 0，所以 f'(x) < 0，函数递减
  • 当 x = 1 时，f'(x) = 0
  • 当 x > 1 时，x - 1 > 0，x > 0，所以 f'(x) > 0，函数递增

所以 x = 1 是最小值点

f(1) = 1 - ln1 = 1 - 0 = 1 > 0

因此，当 x > 0 时，f(x) = x - ln x > 0

即 ln x < x

证毕。

例题 17：三次函数的图像分析 【提高】

已知函数 f(x) = x³ - 3x² + 1，求：

(1) 函数的单调区间

(2) 函数的极值

解答：

(1) 求单调区间：

f'(x) = 3x² - 6x = 3x(x - 2)

令 f'(x) = 0：x = 0 或 x = 2

  • 当 x < 0 时，f'(x) > 0，函数递增
  • 当 0 < x < 2 时，f'(x) < 0，函数递减
  • 当 x > 2 时，f'(x) > 0，函数递增

单调增区间：(-∞, 0) 和 (2, +∞)

单调减区间：(0, 2)


(2) 求极值：

当 x = 0 时，f(0) = 1，是极大值

当 x = 2 时，f(2) = 8 - 12 + 1 = -3，是极小值

答案：(1) 增区间(-∞,0)和(2,+∞)，减区间(0,2) (2) 极大值1，极小值-3

例题 18：利用导数求最值 【提高】

求函数 f(x) = x³ - 3x² 在区间 [0, 3] 上的最大值和最小值

解答：

求导数：f'(x) = 3x² - 6x = 3x(x - 2)

令 f'(x) = 0：x = 0 或 x = 2

都在区间 [0, 3] 内


计算各点函数值：

f(0) = 0 - 0 = 0

f(2) = 8 - 12 = -4

f(3) = 27 - 27 = 0


比较：max{0, -4, 0} = 0，min{0, -4, 0} = -4

答案：最大值为0，最小值为-4

例题 19：参数取值范围 【提高】

已知函数 f(x) = x³ + ax² + bx 在 x = 1 处取得极值 -2，且 a = -1，求实数 b 的值

解答：

条件1：f(1) = -2

1³ + (-1)×1² + b×1 = -2

1 - 1 + b = -2

b = -2


条件2：在 x = 1 处取得极值，则 f'(1) = 0

f'(x) = 3x² + 2ax + b = 3x² - 2x + b

f'(1) = 3 - 2 + b = 1 + b = 0

b = -1


两个条件得到不同的 b 值，需要同时满足，题目可能有误

若按照题目给定 a = -1，由条件1得 b = -2

验证：f'(1) = 1 + (-2) = -1 ≠ 0，不满足极值条件

答案：题目条件矛盾，无法求解

例题 20：综合应用题 【提高】

已知函数 f(x) = x³ - 3x + 1，求：

(1) 函数的极值

(2) 函数在区间 [-2, 2] 上的最大值和最小值

解答：

(1) 求极值：

f'(x) = 3x² - 3

令 f'(x) = 0：3x² - 3 = 0

x² = 1，x = ±1

  • 当 x = -1 时，f(-1) = (-1)³ - 3(-1) + 1 = -1 + 3 + 1 = 3

    • f'(x) 由正变负，故 x = -1 是极大值点，极大值为 3

  • 当 x = 1 时，f(1) = 1³ - 3×1 + 1 = 1 - 3 + 1 = -1

    • f'(x) 由负变正，故 x = 1 是极小值点，极小值为 -1


(2) 求最值：

在区间 [-2, 2] 内，需要比较端点和极值点的函数值

f(-2) = (-2)³ - 3(-2) + 1 = -8 + 6 + 1 = -1

f(-1) = 3（极大值）

f(1) = -1（极小值）

f(2) = 2³ - 3×2 + 1 = 8 - 6 + 1 = 3


比较：max{-1, 3, -1, 3} = 3，min{-1, 3, -1, 3} = -1

答案：(1) 极大值3，极小值-1 (2) 最大值3，最小值-1
