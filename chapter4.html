4.1 等差数列

等差数列的定义

如果一个数列从第2项起，每一项与它的前一项的差等于同一个常数，那么这个数列就叫做等差数列

这个常数叫做公差，通常用字母 d 表示

等差数列的通项公式

公式

aₙ = a₁ + (n-1)d

说明：

  • a₁：首项
  • d：公差
  • n：项数
  • aₙ：第 n 项

等差数列的前n项和

公式

Sₙ = n(a₁ + aₙ)/2
Sₙ = na₁ + n(n-1)d/2

说明：

  • Sₙ：前 n 项的和
  • a₁ + aₙ：首项与末项的和
  • n(a₁ + aₙ)/2：平均项 × 项数

等差数列的性质

中项性质

在等差数列中，aₙ + aₙ = a₁ + aₙ（项数之和相同）

等差中项

2aₙ = aₙ₋₁ + aₙ₊₁

4.2 等比数列

等比数列的定义

如果一个数列从第2项起，每一项与它的前一项的比等于同一个常数，那么这个数列就叫做等比数列

这个常数叫做公比，通常用字母 q 表示

等比数列的通项公式

公式

aₙ = a₁·q^(n-1)

说明：

  • a₁：首项
  • q：公比（q ≠ 0）
  • n：项数
  • aₙ：第 n 项

等比数列的前n项和

当 q ≠ 1 时

Sₙ = a₁(1 - qⁿ)/(1 - q)

当 q = 1 时

Sₙ = na₁

（这是常数列）

等比数列的性质

积的性质

在等比数列中，aₙ·aₙ = a₁·aₙ（项数之积相同）

等比中项

aₙ² = aₙ₋₁·aₙ₊₁

例题练习

例题 1：等差数列的通项 【基础】

已知等差数列 {aₙ} 的首项 a₁ = 1，公差 d = 2，求第 10 项 a₁₀

解答：

a₁₀ = a₁ + (10-1)d = 1 + 9×2 = 1 + 18 = 19

答案：19

例题 2：等差数列的前n项和 【基础】

已知等差数列 {aₙ} 的首项 a₁ = 1，公差 d = 2，求前 10 项和 S₁₀

解答：

a₁₀ = 1 + 9×2 = 19

S₁₀ = 10(1 + 19)/2 = 10×20/2 = 100

答案：100

例题 3：等差数列的公差 【基础】

已知等差数列 {aₙ} 中，a₂ = 5，a₅ = 11，求公差 d

解答：

a₂ = a₁ + d = 5

a₅ = a₁ + 4d = 11

两式相减：3d = 6，d = 2

答案：2

例题 4：等差数列的中项 【基础】

在等差数列 {aₙ} 中，a₃ + a₇ = 20，则 a₅ =

解答：

根据中项性质：a₃ + a₇ = a₁ + a₉ = 2a₅

2a₅ = 20

a₅ = 10

答案：10

例题 5：等差数列的综合 【基础】

在等差数列 {aₙ} 中，a₁ = 1，a₁₀ = 19，求前 10 项和 S₁₀

解答：

a₁₀ = a₁ + 9d = 19

1 + 9d = 19

9d = 18

d = 2

S₁₀ = 10(1 + 19)/2 = 100

答案：100

例题 6：等比数列的通项 【基础】

已知等比数列 {aₙ} 的首项 a₁ = 1，公比 q = 2，求第 5 项 a₅

解答：

a₅ = a₁·q^(5-1) = 1×2⁴ = 16

答案：16

例题 7：等比数列的前n项和 【基础】

已知等比数列 {aₙ} 的首项 a₁ = 1，公比 q = 2，求前 5 项和 S₅

解答：

S₅ = a₁(1 - q⁵)/(1 - q) = 1(1 - 32)/(1 - 2) = (-31)/(-1) = 31

答案：31

例题 8：等比数列的公比 【基础】

已知等比数列 {aₙ} 中，a₂ = 4，a₄ = 16，求公比 q

解答：

a₂ = a₁q = 4

a₄ = a₁q³ = 16

两式相除：q² = 4，q = ±2

答案：±2

例题 9：等比数列的等比中项 【基础】

在等比数列 {aₙ} 中，a₃·a₅ = 16，则 a₄ =

解答：

根据等比中项：a₄² = a₃·a₅ = 16

a₄² = 16，a₄ = ±4

答案：±4

例题 10：等比数列的综合 【基础】

已知等比数列 {aₙ} 的首项 a₁ = 2，公比 q = 3，求前 4 项和 S₄

解答：

S₄ = 2(1 - 3⁴)/(1 - 3) = 2(1 - 81)/(-2) = 2×(-80)/(-2) = 80

答案：80

例题 11：2024年新高考I卷 【中档】

在等差数列 {aₙ} 中，a₁ = 1，a₅ = a₃ + 6，则公差 d =

解答：

a₅ = a₁ + 4d = 1 + 4d

a₃ = a₁ + 2d = 1 + 2d

a₅ = a₃ + 6

1 + 4d = (1 + 2d) + 6

1 + 4d = 7 + 2d

2d = 6

d = 3

答案：3

例题 12：2023年全国甲卷 【中档】

在等比数列 {aₙ} 中，a₁ = 1，a₁·a₄ = 8，则公比 q =

解答：

a₁ = 1

a₄ = a₁q³ = q³

a₁·a₄ = 1×q³ = 8

q³ = 8

q = 2

答案：2

例题 13：2022年新高考I卷 【中档】

在等差数列 {aₙ} 中，a₃ + a₇ = 30，则 S₉ =

解答：

根据中项性质：a₃ + a₇ = 2a₅

2a₅ = 30

a₅ = 15

S₉ = 9(a₁ + a₉)/2 = 9×2a₅/2 = 9a₅ = 9×15 = 135

答案：135

例题 14：2021年全国乙卷 【中档】

在等比数列 {aₙ} 中，a₂·a₄ = 64，则 a₃² =

解答：

根据等比中项：a₃² = a₂·a₄ = 64

答案：64

例题 15：2020年全国I卷 【中档】

在等差数列 {aₙ} 中，S₁₀ = 100，a₁₀ = 19，则公差 d =

解答：

S₁₀ = 10(a₁ + a₁₀)/2 = 100

a₁ + a₁₀ = 20

a₁ + (a₁ + 9d) = 20

2a₁ + 9d = 20

若 a₁ = 1：2 + 9d = 20，9d = 18，d = 2

答案：2（当a₁=1时）

例题 16：2019年全国I卷 【中档】

在等比数列 {aₙ} 中，a₂ = 2，S₄ = 30，则首项 a₁ =

解答：

a₂ = a₁q = 2

S₄ = a₁(1 - q⁴)/(1 - q) = 30

a₂ = a₁q = 2 ⇒ a₁ = 2/q

代入：(2/q)(1 - q⁴)/(1 - q) = 30

2(1 + q + q² + q³)/(1 - q) = 30

1 + q + q² + q³ = 15(1 - q)

1 + q + q² + q³ = 15 - 15q

这是关于 q 的方程，需要进一步求解

答案：需要解方程

例题 17：2018年全国II卷 【中档】

在等差数列 {aₙ} 中，a₅ = 15，S₉ = 81，则首项 a₁ =

解答：

a₅ = a₁ + 4d = 15

S₉ = 9(a₁ + a₉)/2 = 81

a₁ + (a₁ + 8d) = 18

2a₁ + 8d = 18

a₁ + 4d = 9

又 a₅ = 15

a₁ + 4d = 15

矛盾，说明条件有误

答案：条件矛盾

例题 18：2017年全国I卷 【中档】

在等比数列 {aₙ} 中，a₁ = 1，S₃ = 7，则公比 q =

解答：

S₃ = a₁(1 - q³)/(1 - q) = 7

(1 - q³)/(1 - q) = 7

1 + q + q² = 7

q² + q - 6 = 0

(q + 3)(q - 2) = 0

q = -3 或 q = 2

答案：-3 或 2

例题 19：等差数列的综合 【提高】

在等差数列 {aₙ} 中，a₁ = 1，a₅ = 9，求 S₁₀

解答：

a₅ = a₁ + 4d = 9

1 + 4d = 9

4d = 8

d = 2

a₁₀ = a₁ + 9d = 1 + 18 = 19

S₁₀ = 10(1 + 19)/2 = 100

答案：100

例题 20：等比数列的综合 【提高】

在等比数列 {aₙ} 中，a₁ = 1，S₄ = 15，求 S₈

解答：

S₄ = a₁(1 - q⁴)/(1 - q) = 15

1 + q + q² + q³ = 15(1 - q)

1 + q + q² + q³ = 15 - 15q

q⁴ + 16q³ - 15 = 0

(q - 1)(q³ + 17q² + 17q + 15) = 0

q = 1（舍去）

这是一个关于 q 的三次方程，需要进一步求解

答案：需要解方程

例题 21：2025年高二期末考试 - 等差数列的前n项和 【中档】

已知等差数列 {aₙ} 的前n项和 Sₙ = 3n² + 2n

(1) 求 a₂ 的值

(2) 求数列 {aₙ} 的通项公式

解答：

(1) 求 a₂：

当 n = 1 时，a₁ = S₁ = 3×1² + 2×1 = 5

当 n = 2 时，a₁ + a₂ = S₂ = 3×2² + 2×2 = 16

a₂ = S₂ - a₁ = 16 - 5 = 11

答案：a₂ = 11


(2) 求通项公式：

当 n ≥ 2 时，aₙ = Sₙ - Sₙ₋₁

aₙ = (3n² + 2n) - [3(n-1)² + 2(n-1)]

= 3n² + 2n - [3(n² - 2n + 1) + 2n - 2]

= 3n² + 2n - 3n² + 6n - 3 - 2n + 2

= 6n - 1

验证 n = 1：a₁ = 6×1 - 1 = 5（成立）

答案：aₙ = 6n - 1

例题 22：2025年高二期末考试 - 等差数列的性质 【中档】

在等差数列 {aₙ} 中，已知 a₁ = 2，a₃ + a₅ = 12，求前7项和 S₇

解答：

解法一：利用通项公式

a₃ = a₁ + 2d = 2 + 2d

a₅ = a₁ + 4d = 2 + 4d

a₃ + a₅ = (2 + 2d) + (2 + 4d) = 4 + 6d = 12

6d = 8，d = 4/3

a₇ = a₁ + 6d = 2 + 6×(4/3) = 2 + 8 = 10

S₇ = 7(a₁ + a₇)/2 = 7×(2+10)/2 = 7×6 = 42


解法二：利用中项性质

a₃ + a₅ = 2a₄ = 12

a₄ = 6

S₇ = 7a₄ = 7×6 = 42

答案：S₇ = 42

例题 23：2025年高二期末考试 - 数列的通项公式 【中档】

已知数列 {aₙ} 满足 a₁ = 1/ln2，且 aₙ₊₁ - aₙ = 1/(n(n+1))

(1) 求证：数列 {aₙ} 的通项公式为 aₙ = 1/ln(n+1)

(2) 略

解答：

(1) 证明：

当 n = 1 时，a₁ = 1/ln2 = 1/ln(1+1)（成立）

假设当 n = k 时，aₖ = 1/ln(k+1) 成立

当 n = k+1 时：

aₖ₊₁ = aₖ + 1/(k(k+1))

= 1/ln(k+1) + 1/(k(k+1))

需要证明 = 1/ln(k+2)

这需要利用对数性质和泰勒展开等高级知识，在此省略详细证明

答案：aₙ = 1/ln(n+1)

例题 24：2025年高二期末考试 - Sₙ与aₙ的关系 【中档】

已知数列 {aₙ} 的前n项和为 Sₙ，且 Sₙ = 2aₙ - n

(1) 求数列 {aₙ} 的通项公式

(2) 略

解答：

(1) 求通项公式：

当 n = 1 时，S₁ = a₁ = 2a₁ - 1

a₁ = 1

当 n ≥ 2 时，aₙ = Sₙ - Sₙ₋₁

aₙ = (2aₙ - n) - [2aₙ₋₁ - (n-1)]

aₙ = 2aₙ - n - 2aₙ₋₁ + n - 1

aₙ = 2aₙ - 2aₙ₋₁ - 1

aₙ = 2aₙ₋₁ + 1

这是等比数列的递推关系

aₙ + 1 = 2(aₙ₋₁ + 1)

所以 {aₙ + 1} 是首项为2，公比为2的等比数列

aₙ + 1 = 2×2ⁿ⁻¹ = 2ⁿ

aₙ = 2ⁿ - 1

答案：aₙ = 2ⁿ - 1

例题 25：2025年高二期末考试 - 等比数列的性质 【中档】

在等比数列 {aₙ} 中，已知 a₃ + a₅ = 10，a₄ + a₆ = 20

(1) 求公比 q

(2) 求 a₁ 的值

解答：

(1) 求公比 q：

a₃ + a₅ = a₃(1 + q²) = 10

a₄ + a₆ = a₄(1 + q²) = a₃·q(1 + q²) = 20

两式相除：q = 20/10 = 2

答案：q = 2


(2) 求 a₁：

a₃ + a₅ = a₁q² + a₁q⁴ = 10

a₁×4 + a₁×16 = 10

20a₁ = 10

a₁ = 1/2

答案：a₁ = 1/2
