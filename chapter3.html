3.1 平面向量的线性运算

向量的加法

三角形法则

AB + BC = AC

方法：首尾相接，由起点指向终点

平行四边形法则

OA + OC = OB（OABC为平行四边形）

方法：以同一起点的两个邻边为邻边的平行四边形的对角线

向量的减法

三角形法则

AB - AC = CB

方法：从减向量终点指向被减向量终点

向量的数乘

实数 λ 与向量 a 的积是一个向量，记作 λa

运算性质

|λa| = |λ|·|a|

当 λ > 0 时，λa 与 a 方向相同

当 λ < 0 时，λa 与 a 方向相反

当 λ = 0 时，λa = 0

运算律

  • λ(μa) = (λμ)a
  • (λ + μ)a = λa + μa
  • λ(a + b) = λa + λb

向量共线（平行）的充要条件

a ∥ b ⇔ b = λa (b ≠ 0)

3.2 平面向量的数量积

数量积的定义

已知两个非零向量 a 和 b，它们的夹角为 θ，则数量 |a|·|b|·cosθ 叫做 a 与 b 的数量积，记作 a·b

公式

a·b = |a||b|cosθ

数量积的性质

基本性质

  • a·a = |a|²
  • a·b = b·a
  • (λa)·b = λ(a·b)
  • (a + b)·c = a·c + b·c

数量积的坐标运算

设 a = (x₁, y₁)，b = (x₂, y₂)，则：

数量积

a·b = x₁x₂ + y₁y₂

向量的模

|a| = √(x₁² + y₁²)

向量垂直的充要条件

a ⊥ b ⇔ a·b = 0

即：x₁x₂ + y₁y₂ = 0

例题练习

例题 1：向量加法 【基础】

已知 a = (1, 2)，b = (3, 4)，求 a + b

解答：

a + b = (1 + 3, 2 + 4) = (4, 6)

答案：(4, 6)

例题 2：向量减法 【基础】

已知 a = (5, 3)，b = (2, 1)，求 a - b

解答：

a - b = (5 - 2, 3 - 1) = (3, 2)

答案：(3, 2)

例题 3：向量的数乘 【基础】

已知 a = (2, 3)，求 3a

解答：

3a = 3(2, 3) = (6, 9)

答案：(6, 9)

例题 4：向量的模 【基础】

求向量 a = (3, 4) 的模

解答：

|a| = √(3² + 4²) = √(9 + 16) = √25 = 5

答案：5

例题 5：向量的数量积 【基础】

已知 a = (1, 2)，b = (3, 4)，求 a·b

解答：

a·b = 1×3 + 2×4 = 3 + 8 = 11

答案：11

例题 6：向量垂直 【基础】

判断向量 a = (1, 2) 与 b = (2, -1) 是否垂直

解答：

a·b = 1×2 + 2×(-1) = 2 - 2 = 0

由于 a·b = 0，所以 a 与 b 垂直

答案：垂直

例题 7：向量共线 【基础】

判断向量 a = (1, 2) 与 b = (2, 4) 是否共线

解答：

1×4 - 2×2 = 4 - 4 = 0

由于 x₁y₂ - x₂y₁ = 0，所以 a 与 b 共线

答案：共线

例题 8：2024年新高考I卷 【中档】

已知向量 a = (1, 2)，b = (x, 4)，且 a ⊥ b，则 x =

解答：

由于 a ⊥ b，所以 a·b = 0

1×x + 2×4 = 0

x + 8 = 0

x = -8

答案：-8

例题 9：2023年全国甲卷 【中档】

已知 |a| = 3，|b| = 4，且 a 与 b 的夹角为 60°，求 a·b

解答：

a·b = |a||b|cosθ

= 3 × 4 × cos60°

= 12 × 1/2 = 6

答案：6

例题 10：2022年新高考I卷 【中档】

已知 a = (2, 1)，b = (1, k)，且 a ∥ b，则 k =

解答：

由于 a ∥ b，所以 2k - 1×1 = 0

2k - 1 = 0

k = 1/2

答案：1/2

例题 11：2021年全国乙卷 【中档】

已知向量 a = (1, 2)，b = (3, y)，且 |a| = |b|，则 y =

解答：

|a| = √(1² + 2²) = √5

|b| = √(3² + y²) = √(9 + y²)

由于 |a| = |b|，所以 √5 = √(9 + y²)

5 = 9 + y²

y² = -4

无实数解

答案：无实数解

例题 12：2020年全国I卷 【中档】

已知 a = (1, 0)，b = (0, 1)，则 |a - b| =

解答：

a - b = (1 - 0, 0 - 1) = (1, -1)

|a - b| = √(1² + (-1)²) = √2

答案：√2

例题 13：2019年全国I卷 【中档】

已知向量 a = (2, 1)，b = (1, -2)，则 a·b =

解答：

a·b = 2×1 + 1×(-2) = 2 - 2 = 0

答案：0

例题 14：2018年全国II卷 【中档】

已知 a = (1, 2)，b = (3, 4)，求 |a + b|

解答：

a + b = (4, 6)

|a + b| = √(4² + 6²) = √(16 + 36) = √52 = 2√13

答案：2√13

例题 15：2017年全国I卷 【中档】

已知向量 a = (2, 3)，b = (4, 6)，则 a 与 b 的位置关系是

解答：

2×6 - 3×4 = 12 - 12 = 0

由于 x₁y₂ - x₂y₁ = 0，所以 a 与 b 共线

且 b = 2a，所以 b 与 a 同向

答案：共线且同向

例题 16：向量运算综合 【提高】

已知 a = (1, 2)，b = (3, 1)，求 |2a - b|

解答：

2a = (2, 4)

2a - b = (2 - 3, 4 - 1) = (-1, 3)

|2a - b| = √((-1)² + 3²) = √(1 + 9) = √10

答案：√10

例题 17：向量垂直综合 【提高】

已知 a = (2, 1)，b = (x, 2)，且 a ⊥ b，则 |b| =

解答：

由于 a ⊥ b，所以 a·b = 0

2x + 1×2 = 0

2x + 2 = 0

x = -1

b = (-1, 2)

|b| = √((-1)² + 2²) = √(1 + 4) = √5

答案：√5

例题 18：向量共线综合 【提高】

已知 a = (2, 4)，b = (1, k)，且 a ∥ b，求实数 k

解答：

由于 a ∥ b，所以 2k - 4×1 = 0

2k - 4 = 0

k = 2

答案：2

例题 19：向量夹角 【提高】

已知 a = (1, 0)，b = (1, 1)，求 a 与 b 的夹角

解答：

|a| = √(1² + 0²) = 1

|b| = √(1² + 1²) = √2

a·b = 1×1 + 0×1 = 1

cosθ = a·b / (|a||b|) = 1 / (1×√2) = 1/√2

θ = 45°

答案：45°

例题 20：向量综合 【提高】

已知 |a| = 2，|b| = 3，a·b = 3，求 |a + b|

解答：

|a + b|² = (a + b)·(a + b)

= a·a + 2a·b + b·b

= |a|² + 2a·b + |b|²

= 2² + 2×3 + 3² = 4 + 6 + 9 = 19

|a + b| = √19

答案：√19
