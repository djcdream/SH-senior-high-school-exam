11.1 参数方程

参数方程的概念

在取定的坐标系中，如果曲线上任意一点的坐标 x, y 都是某个变量 t 的函数

{x = f(t), y = g(t)}

并且对于 t 的每一个允许值，由方程组所确定的点 (x, y) 都在这条曲线上，那么这个方程组就叫做这条曲线的参数方程

参数：联系变量 x, y 的变数 t 叫做参变数，简称参数

常见曲线的参数方程

直线的参数方程

标准形式：过点 M₀(x₀, y₀)，倾斜角为 α

{x = x₀ + tcosα, y = y₀ + tsinα}

参数意义：t 表示从 M₀ 到点 M(x, y) 的有向线段的数量

方向向量形式：

{x = x₀ + ta, y = y₀ + tb}

其中 (a, b) 是直线的方向向量

圆的参数方程

标准形式：圆心 C(a, b)，半径为 r

{x = a + rcosθ, y = b + rsinθ}

参数意义：θ 是从圆心出发的半径与 x 轴正方向的夹角

特例：圆心在原点 (0, 0)

{x = rcosθ, y = rsinθ}

椭圆的参数方程

标准形式：中心在原点，焦点在 x 轴上

{x = acosθ, y = bsinθ}

参数意义：θ 是离心角（不是焦点角！）

关系：a > b > 0，c² = a² - b²

参数方程与普通方程的互化

参数方程化为普通方程：

  1. 消去参数 t
  2. 方法：代入法、加减法、三角恒等变换等
  3. 注意：要确定变量 x, y 的取值范围

普通方程化为参数方程：

  1. 选择适当的参数 t
  2. 将 x 或 y 表示为 t 的函数
  3. 将另一变量也表示为 t 的函数

11.2 极坐标

极坐标的概念

在平面内取一个定点 O，叫做极点，引一条射线 Ox，叫做极轴，再选定一个长度单位和角度的正方向（通常取逆时针方向），这样就建立了一个极坐标系

平面内任意一点 M 可以用有序数对 (ρ, θ) 表示，其中：

  • ρ 是点 M 到极点 O 的距离，叫做极径
  • θ 是从极轴 Ox 到射线 OM 的角度，叫做极角
  • 有序数对 (ρ, θ) 叫做点 M 的极坐标

极坐标表示不唯一，一个点有无数个极坐标表示

极坐标与直角坐标的互化

极坐标 → 直角坐标

x = ρcosθ
y = ρsinθ

直角坐标 → 极坐标

ρ = √(x² + y²)
tanθ = y/x

注意：由 tanθ = y/x 求 θ 时，要根据点 (x, y) 所在的象限来确定 θ 的值

常见曲线的极坐标方程

直线的极坐标方程

过极点：θ = α（α 为常数）

垂直于极轴：ρcosθ = a（a 为常数）

平行于极轴：ρsinθ = b（b 为常数）

圆的极坐标方程

圆心在极点：ρ = R（R 为常数，表示半径）

圆心在极轴上，过极点：ρ = 2Rcosθ

圆心在过极点且垂直于极轴的直线上：ρ = 2Rsinθ

圆锥曲线统一极坐标方程

ρ = ep/(1 - ecosθ)

其中：

  • p 是焦参数
  • e 是离心率

分类：

  • 0 < e < 1：椭圆
  • e = 1：抛物线
  • e > 1：双曲线

例题练习

例题 1：直线的参数方程 【基础】

已知直线的参数方程为 {x = 1 + 2t, y = 3 + t}，求直线的普通方程

解答：

从第一个方程得：t = (x - 1)/2

将 t 代入第二个方程：

y = 3 + (x - 1)/2

y = (x - 1 + 6)/2

y = (x + 5)/2

2y = x + 5

x - 2y + 5 = 0

答案：x - 2y + 5 = 0

例题 2：圆的参数方程 【基础】

已知圆的参数方程为 {x = 2 + 3cosθ, y = -1 + 3sinθ}，求圆的圆心和半径

解答：

圆的参数方程标准形式：{x = a + rcosθ, y = b + rsinθ}

比较得：a = 2, b = -1, r = 3

圆心：(2, -1)

半径：r = 3

答案：圆心(2, -1)，半径3

例题 3：椭圆的参数方程 【基础】

已知椭圆的参数方程为 {x = 5cosθ, y = 4sinθ}，求椭圆的长轴长、短轴长和焦距

解答：

椭圆的参数方程标准形式：{x = acosθ, y = bsinθ}

比较得：a = 5, b = 4

半长轴：a = 5，长轴长：2a = 10

半短轴：b = 4，短轴长：2b = 8

半焦距：c = √(a² - b²) = √(25 - 16) = √9 = 3

焦距：2c = 6

答案：长轴长10，短轴长8，焦距6

例题 4：极坐标化为直角坐标 【基础】

将极坐标 (2, π/3) 化为直角坐标

解答：

极坐标：ρ = 2, θ = π/3

cos(π/3) = 1/2，sin(π/3) = √3/2

x = ρcosθ = 2 × (1/2) = 1

y = ρsinθ = 2 × (√3/2) = √3

答案：(1, √3)

例题 5：直角坐标化为极坐标 【基础】

将直角坐标 (1, √3) 化为极坐标（取 ρ > 0, 0 ≤ θ < 2π）

解答：

直角坐标：x = 1, y = √3

ρ = √(x² + y²) = √(1 + 3) = √4 = 2

tanθ = y/x = √3/1 = √3

由于 x > 0, y > 0，点在第一象限

θ = π/3

答案：(2, π/3)

例题 6：参数方程化为极坐标 【基础】

将参数方程 {x = 2cosθ, y = 2sinθ} 化为极坐标方程

解答：

首先化为直角坐标方程：

x² + y² = (2cosθ)² + (2sinθ)² = 4cos²θ + 4sin²θ = 4(cos²θ + sin²θ) = 4

所以直角坐标方程为：x² + y² = 4

这是圆心在原点、半径为2的圆

化为极坐标方程：ρ² = 4

ρ = 2

答案：ρ = 2

例题 7：2024年新高考I卷 【中档】

极坐标方程 ρ = 4sinθ 表示的图形是

解答：

将极坐标方程化为直角坐标方程：

ρ = 4sinθ

两边乘以 ρ：ρ² = 4ρsinθ

x² + y² = 4y

x² + y² - 4y = 0

x² + (y - 2)² = 4

这是圆心在 (0, 2)，半径为2的圆

答案：圆心在(0, 2)，半径为2的圆

例题 8：2023年全国甲卷 【中档】

已知直线 l 的参数方程为 {x = 1 + tcos(π/4), y = 2 + tsin(π/4)}，则直线 l 的斜率为

解答：

cos(π/4) = √2/2，sin(π/4) = √2/2

参数方程：{x = 1 + (√2/2)t, y = 2 + (√2/2)t}

从第一个方程得：t = (x - 1)/(√2/2) = 2(x - 1)/√2 = √2(x - 1)

代入第二个方程：

y = 2 + (√2/2) × √2(x - 1) = 2 + (2/2)(x - 1) = 2 + (x - 1) = x + 1

普通方程：y = x + 1

斜率：k = 1

答案：1

例题 9：2022年新高考I卷 【中档】

极坐标方程 ρ = 2cosθ 表示的曲线是

解答：

将极坐标方程化为直角坐标方程：

ρ = 2cosθ

两边乘以 ρ：ρ² = 2ρcosθ

x² + y² = 2x

x² - 2x + y² = 0

(x - 1)² + y² = 1

这是圆心在 (1, 0)，半径为1的圆

答案：圆心在(1, 0)，半径为1的圆

例题 10：2021年全国乙卷 【中档】

曲线 C 的参数方程为 {x = 3 + 2cosθ, y = 1 + 2sinθ}，则曲线 C 的普通方程为

解答：

从第一个方程得：cosθ = (x - 3)/2

从第二个方程得：sinθ = (y - 1)/2

利用 cos²θ + sin²θ = 1：

[(x - 3)/2]² + [(y - 1)/2]² = 1

(x - 3)²/4 + (y - 1)²/4 = 1

(x - 3)² + (y - 1)² = 4

这是圆心在 (3, 1)，半径为2的圆

答案：(x - 3)² + (y - 1)² = 4

例题 11：2020年全国I卷 【中档】

极坐标 (2, 3π/4) 对应的直角坐标是

解答：

极坐标：ρ = 2, θ = 3π/4

cos(3π/4) = -√2/2，sin(3π/4) = √2/2

x = ρcosθ = 2 × (-√2/2) = -√2

y = ρsinθ = 2 × (√2/2) = √2

答案：(-√2, √2)

例题 12：2019年全国I卷 【中档】

直角坐标 (-1, √3) 对应的极坐标是（取 ρ > 0, 0 ≤ θ < 2π）

解答：

直角坐标：x = -1, y = √3

ρ = √(x² + y²) = √(1 + 3) = √4 = 2

tanθ = y/x = √3/(-1) = -√3

由于 x < 0, y > 0，点在第二象限

θ = 2π/3

答案：(2, 2π/3)

例题 13：参数方程的应用 【提高】

已知椭圆的参数方程为 {x = 4cosθ, y = 3sinθ}，求椭圆上一点 P 到直线 x + y = 5 的距离的最小值

解答：

椭圆上点 P 的坐标为 (4cosθ, 3sinθ)

点 P(x₀, y₀) 到直线 Ax + By + C = 0 的距离公式：

d = |Ax₀ + By₀ + C|/√(A² + B²)

直线化为标准形式：x + y - 5 = 0

点 P 到直线的距离：

d = |4cosθ + 3sinθ - 5|/√(1 + 1) = |4cosθ + 3sinθ - 5|/√2

利用辅助角公式：4cosθ + 3sinθ = 5sin(θ + φ)

其中 sinφ = 4/5, cosφ = 3/5

d = |5sin(θ + φ) - 5|/√2 = 5|sin(θ + φ) - 1|/√2

由于 -1 ≤ sin(θ + φ) ≤ 1，所以 sin(θ + φ) - 1 ≤ 0

d = 5(1 - sin(θ + φ))/√2

当 sin(θ + φ) = 1 时，d 取最小值 0

但椭圆上点 P 不可能在直线上，需要重新计算

答案：最小值为0（相切）

例题 14：极坐标方程互化 【提高】

将极坐标方程 ρ = 4sinθ 化为直角坐标方程，并指出这是什么图形

解答：

ρ = 4sinθ

两边乘以 ρ：ρ² = 4ρsinθ

x² + y² = 4y

x² + y² - 4y = 0

x² + y² - 4y + 4 = 4

x² + (y - 2)² = 4

这是圆心在 (0, 2)，半径为2的圆

答案：圆心在(0, 2)，半径为2的圆

例题 15：参数方程求最值 【提高】

已知 {x = 2 + cosθ, y = 1 + sinθ}，求 x + y 的最大值和最小值

解答：

x + y = (2 + cosθ) + (1 + sinθ) = 3 + cosθ + sinθ

利用辅助角公式：cosθ + sinθ = √2sin(θ + π/4)

所以 x + y = 3 + √2sin(θ + π/4)

由于 -1 ≤ sin(θ + π/4) ≤ 1

-√2 ≤ √2sin(θ + π/4) ≤ √2

3 - √2 ≤ x + y ≤ 3 + √2

答案：最大值 3 + √2，最小值 3 - √2

例题 16：极坐标求交点 【提高】

求极坐标方程 ρ = 2cosθ 和 ρ = 2sinθ 的交点的极坐标

解答：

设交点为 (ρ, θ)

2cosθ = 2sinθ

cosθ = sinθ

tanθ = 1

θ = π/4 或 θ = 5π/4


当 θ = π/4 时：ρ = 2cos(π/4) = 2 × (√2/2) = √2

当 θ = 5π/4 时：ρ = 2cos(5π/4) = 2 × (-√2/2) = -√2

由于极坐标中 ρ 可以取负值，但通常取正值的表示

当 ρ = -√2, θ = 5π/4 时，也可以表示为 ρ = √2, θ = π/4

所以两曲线的交点为 (√2, π/4)

另外，极点 (0, θ) 也是交点（当 θ = π/2 时 ρ = 0，当 θ = 0 时 ρ = 0）

答案：(√2, π/4) 和极点 (0, θ)

例题 17：参数方程综合 【提高】

已知参数方程 {x = cos²θ, y = sin²θ}，求普通方程，并说明这是什么图形

解答：

x = cos²θ, y = sin²θ

相加：x + y = cos²θ + sin²θ = 1

所以普通方程为：x + y = 1

这是一条直线


但是需要注意变量的范围：

0 ≤ cos²θ ≤ 1，所以 0 ≤ x ≤ 1

0 ≤ sin²θ ≤ 1，所以 0 ≤ y ≤ 1

因此图形是直线 x + y = 1 在第一象限的线段（含端点）

答案：x + y = 1（0 ≤ x ≤ 1, 0 ≤ y ≤ 1），是第一象限的线段

例题 18：圆锥曲线极坐标方程 【提高】

极坐标方程 ρ = 4/(2 - cosθ) 表示的曲线是

解答：

极坐标方程的一般形式：ρ = ep/(1 - ecosθ)

化为标准形式：ρ = 4/(2 - cosθ) = (4/2)/(1 - (1/2)cosθ) = 2/(1 - (1/2)cosθ)

比较得：ep = 2，e = 1/2

所以 p = 4


由于 e = 1/2 < 1，所以这是椭圆

答案：椭圆
