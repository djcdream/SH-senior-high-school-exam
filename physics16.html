第十六章 电磁波

知识点1

基本概念

电磁波：由变化的电场和磁场相互激发形成的波，可以在真空中传播

电磁波谱：按频率或波长排列的电磁波，包括无线电波、微波、红外线、可见光、紫外线、X射线、γ射线

电磁波速：c = 3×10⁸m/s，真空中的光速

波长和频率关系：λ = c/f

知识点2

重要规律

电磁波的产生：LC振荡电路可以产生电磁波，频率f = 1/[2π√(LC)]

电磁波的性质：电磁波是横波，电场和磁场相互垂直，且都垂直于传播方向

电磁波的反射和折射：遵循反射定律和折射定律

电磁波的干涉和衍射：电磁波也能发生干涉和衍射现象

电磁波的能量：能流密度S = ½cε₀E² = ½cμ₀H²

例题练习

例题 1：【2024·新课标卷】 电磁波的性质 【基础】

关于电磁波，下列说法正确的是：

(A) 电磁波是纵波

(B) 电磁波可以在真空中传播

(C) 电磁波的传播速度与介质有关

(D) 电磁波的电场和磁场同相位

解答：

（A）错误。电磁波是横波，电场和磁场都垂直于传播方向

（B）正确。电磁波是变化的电场和磁场相互激发形成的，不需要介质，可以在真空中传播

（C）正确。电磁波的传播速度与介质的电磁性质有关，真空中最快

（D）错误。电磁波的电场和磁场相位相同，但空间方向相互垂直

知识点标签：电磁波的性质、横波

答案：B、C

例题 2：【2023·浙江卷】 电磁波谱 【基础】

下列电磁波中，频率由低到高排列的是：

(A) 无线电波、微波、红外线、可见光

(B) γ射线、X射线、紫外线、可见光

(C) 可见光、紫外线、X射线、γ射线

(D) 微波、红外线、可见光、无线电波

解答：

（A）正确。电磁波谱中，频率由低到高的顺序是：无线电波、微波、红外线、可见光、紫外线、X射线、γ射线

（B）错误。这是频率由高到低的顺序

（C）错误。这是可见光及以上，没有无线电波、微波、红外线

（D）错误。顺序不对，且无线电波的频率最低

知识点标签：电磁波谱、频率排列

答案：A

例题 3：【2022·新课标卷】 波长和频率 【中档】

一电磁波的频率为f，在真空中传播的速度为c。求该电磁波的波长。

解答：

（1）波长、频率和波速的关系：c = λf

（2）解得：λ = c/f

（3）电磁波的波长与频率成反比

（4）频率越高，波长越短

知识点标签：电磁波、波长、频率、波速

答案：λ = c/f

例题 4：【2024·河北卷】 LC振荡电路 【中档】

一LC振荡电路，电感为L，电容为C。求该电路产生电磁波的周期和频率。

解答：

（1）LC振荡电路的周期公式：T = 2π√(LC)

（2）频率：f = 1/T = 1/[2π√(LC)]

（3）这是LC电路的固有频率

（4）电磁波的频率与LC电路的固有频率相同

（5）改变L或C可以改变电磁波的频率

知识点标签：LC振荡电路、电磁波的频率

答案：T = 2π√(LC)，f = 1/[2π√(LC)]

例题 5：【2023·全国卷】 电磁波的能量 【中档】

一电磁波在空间中传播，电场强度为E，磁场强度为H。求该电磁波的能流密度（单位时间内通过单位面积的电磁能量）。

解答：

（1）电磁波的能流密度公式：S = EH

（2）电磁波中，电场和磁场的关系：H = √(ε₀/μ₀)·E

（3）所以：S = E·√(ε₀/μ₀)·E = √(ε₀/μ₀)·E²

（4）也可以写成：S = cε₀E² = cμ₀H²，其中c = 1/√(ε₀μ₀)

（5）能流密度描述电磁波的能量传输

知识点标签：电磁波、能流密度、电场和磁场

答案：S = cε₀E² = cμ₀H²

例题 6：【2024·辽宁卷】 电磁波的反射 【中档】

一电磁波垂直射向金属板，若电磁波的频率为f，金属板是否会对电磁波产生反射？说明理由。

解答：

（1）电磁波垂直射向金属板时，会与金属板中的自由电子相互作用

（2）电磁波的电场会驱动金属板中的自由电子运动

（3）运动中的电子会产生电磁波，形成反射

（4）反射波的频率与入射波相同，但相位相反

（5）这就是电磁波的反射现象，类似于光波的反射

（6）反射波的强度取决于金属板的导电性和入射角

知识点标签：电磁波、反射、金属板

答案：会反射，金属板中的自由电子与电磁波相互作用产生反射波

例题 7：【2022·浙江卷】 电磁波的折射 【中档】

一电磁波从空气射入水中，入射角为i。若空气中的电磁波速为c₁，水中的电磁波速为c₂（c₁ > c₂），求折射角r。

解答：

（1）根据折射定律：sin i/sin r = c₁/c₂

（2）解得：sin r = (c₂/c₁)·sin i

（3）折射角：r = arcsin[(c₂/c₁)·sin i]

（4）由于c₁ > c₂，所以折射角r小于入射角i

（5）电磁波从光疏介质进入光密介质时，向法线折射

知识点标签：电磁波、折射、折射定律

答案：r = arcsin[(c₂/c₁)·sin i]

例题 8：【2023·河北卷】 电磁波的干涉 【高档】

两频率相同的电磁波在某点相遇，若两电磁波的振幅相同，相位差为Δφ，求该点合成电磁波的振幅。

解答：

（1）设两电磁波的振幅分别为A₁ = A和A₂ = A

（2）合成电磁波的振幅：A合 = √(A₁² + A₂² + 2A₁A₂·cosΔφ)

（3）代入A₁ = A₂ = A：A合 = √(A² + A² + 2A²·cosΔφ) = √[2A²(1 + cosΔφ)]

（4）利用三角函数公式：1 + cosΔφ = 2cos²(Δφ/2)

（5）所以：A合 = √[2A²·2cos²(Δφ/2)] = √[4A²cos²(Δφ/2)] = 2A|cos(Δφ/2)|

（6）当Δφ = 0时，A合 = 2A（相长干涉）；当Δφ = π时，A合 = 0（相消干涉）

知识点标签：电磁波、干涉、振幅合成

答案：A合 = 2A|cos(Δφ/2)|

例题 9：【2024·新课标Ⅱ卷】 电磁波的衍射 【中档】

一电磁波通过宽缝，缝宽为d，电磁波的波长为λ。若缝宽d与波长λ的关系满足d = λ，在缝后多远处能观察到明显的衍射现象？

解答：

（1）电磁波的衍射现象与缝宽d和波长λ的比值有关

（2）当d ≈ λ时，衍射现象最明显

（3）当d >> λ时，衍射现象不明显，近似沿直线传播

（4）当d << λ时，衍射现象非常明显，电磁波可以绕过缝向各个方向传播

（5）本题中d = λ，属于"缝宽与波长相当"的情况，衍射现象比较明显

（6）在缝后一定距离（远大于缝宽）处都能观察到明显的衍射现象

知识点标签：电磁波、衍射、缝宽、波长

答案：当d = λ时，在缝后较远处都能观察到明显的衍射现象

例题 10：【2022·全国卷】 无线电波 【基础】

一无线电波的频率为f，在空气中传播的速度为c。求该无线电波的波长和周期。

解答：

（1）波长：λ = c/f

（2）周期：T = 1/f

（3）频率越高，波长越短，周期越短

（4）无线电波的频率范围很广，从长波、中波到短波

（5）不同频率的无线电波有不同的传播特性

知识点标签：无线电波、波长、周期、频率

答案：λ = c/f，T = 1/f

例题 11：【2023·江苏卷】 电磁波的偏振 【中档】

一电磁波的偏振方向是竖直方向。若要使该电磁波通过偏振片，偏振片的透振方向应如何放置？

解答：

（1）偏振片只允许与其透振方向平行的电磁波分量通过

（2）为了使竖直偏振的电磁波完全通过，偏振片的透振方向应设置为竖直方向

（3）此时，电磁波的电场振动方向与偏振片的透振方向平行，完全通过

（4）若偏振片的透振方向设置为水平方向，则电磁波完全被阻挡

（5）若偏振片的透振方向与竖直方向成θ角，则通过的电磁波分量为E' = E·cosθ

知识点标签：电磁波、偏振、偏振片

答案：偏振片的透振方向应设置为竖直方向

例题 12：【2024·江西卷】 电磁波的多普勒效应 【中档】

一电磁波源以速度v₁向静止的接收者运动，电磁波的频率为f₀。若电磁波的速度为c，求接收者接收到的频率f'。

解答：

（1）电磁波源向接收者运动时，波长缩短：λ' = (c - v₁)/f₀

（2）接收者接收到的频率：f' = c/λ'

（3）代入λ'得：f' = c/[(c - v₁)/f₀] = c·f₀/(c - v₁)

（4）由于c - v₁ < c，所以f' > f₀，接收到的频率升高

（5）这称为电磁波的多普勒效应，应用于雷达测速、天体速度测量等

知识点标签：电磁波、多普勒效应、频率变化

答案：f' = c·f₀/(c - v₁)

例题 13：【2022·河北卷】 可见光 【基础】

红光的波长约为700nm，紫光的波长约为400nm。求红光和紫光的频率之比。

解答：

（1）红光的波长：λ红 = 700nm = 700×10⁻⁹m

（2）紫光的波长：λ紫 = 400nm = 400×10⁻⁹m

（3）红光的频率：f红 = c/λ红 = c/(700×10⁻⁹)

（4）紫光的频率：f紫 = c/λ紫 = c/(400×10⁻⁹)

（5）频率之比：f红/f紫 = [c/(700×10⁻⁹)]/[c/(400×10⁻⁹)] = 400/700 = 4/7

（6）红光的频率约为紫光的4/7

（7）波长越短，频率越高

知识点标签：可见光、波长、频率

答案：f红/f紫 = 4/7

例题 14：【2023·新课标卷】 电磁波的强度 【中档】

一电磁波在介质中传播，若将电场强度增大为原来的2倍，其他条件不变，则电磁波的强度变为原来的几倍？

解答：

（1）电磁波的强度（能流密度）：S = cε₀E²

（2）原强度：S₁ = cε₀E₁²

（3）电场强度增大为原来的2倍后：E₂ = 2E₁

（4）新强度：S₂ = cε₀E₂² = cε₀(2E₁)² = 4cε₀E₁²

（5）强度之比：S₂/S₁ = 4cε₀E₁²/(cε₀E₁²) = 4

（6）电磁波的强度与电场强度的平方成正比

知识点标签：电磁波、强度、电场强度

答案：4倍

例题 15：【2024·全国卷】 电磁波的应用 【高档】

一电磁波用于通信，发射功率为P，天线向各个方向均匀发射。求距离发射天线r处电磁波的强度（忽略空气吸收）。

解答：

（1）电磁波向各个方向均匀发射，传播到距离r处时，电磁波分布在半径为r的球面上

（2）球面的面积：A = 4πr²

（3）距离r处电磁波的强度：I = P/A = P/(4πr²)

（4）强度与距离的平方成反比

（5）距离越远，强度越小

（6）这是电磁波在自由空间中的传播规律

（7）实际通信中，还需考虑天线增益、大气吸收等因素

知识点标签：电磁波、强度、球面传播、通信

答案：I = P/(4πr²)
