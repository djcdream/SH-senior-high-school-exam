第九章 机械振动

← 返回物理列表 知识点1 知识点2 例题练习

知识点1

基本概念

机械振动：物体在平衡位置附近所做的往复运动

简谐运动：物体在跟偏离平衡位置的位移大小成正比，并且总是指向平衡位置的回复力作用下的振动

回复力：F = -kx，其中k是常数，负号表示回复力方向与位移方向相反

振幅：A，物体离开平衡位置的最大距离

周期：T，物体完成一次全振动所用的时间，单位是秒(s)

频率：f，单位时间内完成全振动的次数，单位是赫兹(Hz)，f = 1/T

知识点2

重要公式

简谐运动的周期公式：T = 2π√(m/k)

单摆的周期公式：T = 2π√(l/g)

弹簧振子的周期：T = 2π√(m/k)

简谐运动的能量：E = ½kA²（振子系统的机械能守恒）

受迫振动：物体在周期性驱动力作用下的振动

共振：驱动力的频率等于物体的固有频率时，振幅最大

例题练习

例题 1：【2024·新课标卷】 简谐运动判别 【基础】

以下运动中属于简谐运动的是：

(A) 弹簧振子在光滑水平面上的振动

(B) 小球在半径很大的光滑凹形球面最低点附近的摆动

(C) 小球在半球形光滑碗底附近的小角度摆动

(D) 以上都是

解答：

（A）正确。弹簧振子的回复力F = -kx，满足简谐运动的条件

（B）正确。小球在球面最低点附近摆动，回复力F = -mgθ ≈ -mg·x/R = -(mg/R)x，满足简谐运动条件

（C）正确。与（B）类似，回复力满足简谐运动条件

知识点标签：简谐运动、回复力

答案：D

例题 2：【2023·浙江卷】 弹簧振子周期 【中档】

一质量为m的弹簧振子，劲度系数为k，在光滑水平面上做简谐运动。若将振子的质量增大为原来的4倍，其他条件不变，则振子的周期变为原来的几倍？

解答：

（1）弹簧振子的周期公式：T = 2π√(m/k)

（2）原周期：T₁ = 2π√(m/k)

（3）质量变为4m后，新周期：T₂ = 2π√(4m/k) = 2π·2√(m/k) = 2T₁

（4）周期之比：T₂/T₁ = 2

知识点标签：简谐运动、弹簧振子、周期

答案：2倍

例题 3：【2022·新课标卷】 单摆周期 【中档】

一单摆的摆长为l，摆球质量为m。若将摆长缩短为原来的1/4，则单摆的周期变为原来的几倍？

解答：

（1）单摆的周期公式：T = 2π√(l/g)

（2）原周期：T₁ = 2π√(l/g)

（3）摆长变为l/4后，新周期：T₂ = 2π√[(l/4)/g] = 2π·(1/2)√(l/g) = T₁/2

（4）周期之比：T₂/T₁ = 1/2

（5）摆长缩短，周期变短

知识点标签：单摆、周期、摆长

答案：1/2倍

例题 4：【2024·河北卷】 简谐运动能量 【中档】

一弹簧振子在光滑水平面上做简谐运动，振幅为A，劲度系数为k。求振子在平衡位置时的动能和最大位移处的弹性势能。

解答：

（1）振子的总能量（机械能守恒）：E = ½kA²

（2）在最大位移处，全部能量为弹性势能：Ep_max = ½kA²，动能为零

（3）在平衡位置处，全部能量为动能：Ek_max = ½kA²，弹性势能为零

（4）平衡位置时的动能：Ek_max = ½kA²

（5）最大位移处的弹性势能：Ep_max = ½kA²

知识点标签：简谐运动、能量守恒、机械能

答案：平衡位置动能 = ½kA²，最大位移处弹性势能 = ½kA²

例题 5：【2023·全国卷】 单摆重力加速度测量 【高档】

用单摆测量某地的重力加速度，测量摆长为l，单摆完成n次全振动的时间为t。求重力加速度g的表达式。

解答：

（1）单摆的周期：T = t/n

（2）根据单摆周期公式：T = 2π√(l/g)

（3）由t/n = 2π√(l/g)，平方得：(t/n)² = 4π²l/g

（4）解得：g = 4π²ln²/t²

（5）这是测量重力加速度的一种方法，称为单摆法

知识点标签：单摆、周期、重力加速度测量

答案：g = 4π²ln²/t²

例题 6：【2024·辽宁卷】 受迫振动与共振 【中档】

一物体在周期性驱动力作用下做受迫振动。当驱动力的频率等于物体的固有频率时，下列说法正确的是：

(A) 物体振动的频率等于驱动力的频率

(B) 物体振动的振幅最大，发生共振

(C) 物体振动的振幅最小

(D) 物体振动的周期等于驱动力的周期

解答：

（A）正确。受迫振动的频率等于驱动力的频率，与固有频率无关

（B）正确。当驱动力的频率等于物体的固有频率时，发生共振，振幅最大

（C）错误。共振时振幅最大，不是最小

（D）正确。受迫振动的周期等于驱动力的周期

知识点标签：受迫振动、共振、固有频率

答案：A、B、D

例题 7：【2022·浙江卷】 弹簧振子速度 【中档】

一弹簧振子在光滑水平面上做简谐运动，振幅为A，劲度系数为k，振子质量为m。求振子通过平衡位置时的速度。

解答：

（1）振子的总能量：E = ½kA²

（2）在平衡位置处，全部能量为动能：Ek = ½mv²

（3）根据能量守恒：½mv² = ½kA²

（4）解得：v = √(k/m)·A = A√(k/m)

（5）平衡位置处速度最大，为振幅与角速度的乘积

知识点标签：简谐运动、能量守恒、速度

答案：v = A√(k/m)

例题 8：【2023·河北卷】 单摆摆长 【中档】

一单摆的周期为T。若将摆长增大为原来的2倍，则单摆的周期变为原来的几倍？

解答：

（1）单摆的周期公式：T = 2π√(l/g)

（2）原周期：T₁ = 2π√(l/g)

（3）摆长变为2l后，新周期：T₂ = 2π√(2l/g) = 2π·√2√(l/g) = √2·T₁

（4）周期之比：T₂/T₁ = √2 ≈ 1.414

知识点标签：单摆、周期、摆长

答案：√2倍（约1.414倍）

例题 9：【2024·新课标Ⅱ卷】 简谐运动加速度 【中档】

一弹簧振子做简谐运动，质量为m，劲度系数为k。当振子偏离平衡位置的位移为x时，求振子的加速度。

解答：

（1）简谐运动的回复力：F = -kx

（2）根据牛顿第二定律：F = ma

（3）由-ma = -kx，得：a = -kx/m

（4）加速度方向与位移方向相反，指向平衡位置

（5）加速度大小与位移成正比：a = (k/m)|x|

知识点标签：简谐运动、回复力、加速度

答案：a = -kx/m，大小为a = (k/m)|x|

例题 10：【2022·全国卷】 单摆频率 【基础】

一单摆的摆长为l，重力加速度为g。求单摆的频率f。

解答：

（1）单摆的周期：T = 2π√(l/g)

（2）频率与周期的关系：f = 1/T

（3）所以：f = 1/[2π√(l/g)] = g^(1/2)/[2πl^(1/2)] = √(g/l)/(2π)

（4）频率的单位是赫兹(Hz)，表示每秒振动的次数

知识点标签：单摆、周期、频率

答案：f = 1/[2π√(l/g)] = √(g/l)/(2π)

例题 11：【2023·江苏卷】 阻尼振动 【中档】

关于阻尼振动，下列说法正确的是：

(A) 阻尼振动的振幅保持不变

(B) 阻尼振动的周期比无阻尼时大

(C) 阻尼振动的振幅逐渐减小

(D) 阻尼振动是机械能守恒的

解答：

（A）错误。阻尼振动中由于阻力作用，振幅逐渐减小

（B）正确。阻尼振动的周期比无阻尼时略有增大

（C）正确。阻尼振动中，振幅随时间逐渐减小

（D）错误。阻尼振动中，机械能不守恒，有能量转化为热能等

知识点标签：阻尼振动、振幅、机械能

答案：B、C

例题 12：【2024·江西卷】 相位差 【中档】

两个弹簧振子，做简谐运动的周期相同，振幅相同。当其中一个振子经过平衡位置向上运动时，另一个振子刚好在最大位移处。求两个振子的相位差。

解答：

（1）从平衡位置向上运动到最大位移处，振子经历了1/4个周期

（2）一个周期T对应相位变化为2π

（3）1/4个周期对应的相位差：Δφ = (1/4)·2π = π/2

（4）第二个振子比第一个振子落后π/2的相位

（5）相位差的单位是弧度(rad)，也可以用角度表示，即90°

知识点标签：简谐运动、相位、相位差

答案：π/2（90°）

例题 13：【2022·河北卷】 共振条件 【中档】

一物体在做受迫振动，其固有频率为f₀。若要使物体的振幅最大，驱动力的频率f应满足什么条件？

解答：

（1）受迫振动的频率等于驱动力的频率：f振 = f

（2）当驱动力的频率等于物体的固有频率时，发生共振

（3）共振时，物体的振幅达到最大值

（4）共振条件：f = f₀

（5）若驱动力的频率略大于或略小于固有频率，振幅也会较大，但不是最大

知识点标签：受迫振动、共振、固有频率

答案：f = f₀

例题 14：【2023·浙江卷】 简谐运动位移 【中档】

一弹簧振子做简谐运动，振幅为A，周期为T。求振子从最大位移处运动到平衡位置所需的最短时间。

解答：

（1）简谐运动从最大位移处到平衡位置，相当于1/4个周期

（2）所以最短时间：t = T/4

（3）也可以从平衡位置到最大位移处，时间也是T/4

（4）若从最大位移到另一侧最大位移，时间为T/2

（5）完成一次全振动，时间为T

知识点标签：简谐运动、周期、位移

答案：t = T/4

例题 15：【2024·全国卷】 简谐运动图像 【高档】

一简谐运动的位移-时间图像为正弦曲线，振幅为A，周期为T。写出位移x随时间t的变化关系式（t=0时，x=0且向正方向运动）。

解答：

（1）简谐运动的位移-时间关系可以用正弦函数或余弦函数表示

（2）角频率：ω = 2π/T

（3）由于t=0时，x=0且向正方向运动，所以用正弦函数：x = Asin(ωt)

（4）也可以写成：x = Asin(2πt/T)

（5）若用余弦函数，则为：x = Acos(ωt - π/2)

（6）振幅A描述位移的最大值，周期T描述振动的快慢

知识点标签：简谐运动、位移-时间图像、三角函数

答案：x = Asin(2πt/T)
