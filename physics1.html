1.1 参考系

参考系的定义

为了描述物体的运动而选作参考的物体叫做参考系

参考系的选择是任意的，但选择不同的参考系，对物体运动的描述可能不同

参考系的选择原则

  • 选择使问题简化的参考系
  • 通常选地面为参考系
  • 研究地球公转时，选太阳为参考系
  • 研究火车上物体的运动，可选火车为参考系

运动和静止的相对性

运动是绝对的，静止是相对的

同一个物体相对于不同的参考系，运动状态可能不同

1.2 时间和位移

时刻和时间间隔

时刻

指某一瞬时，在时间轴上用点表示

例如：第2秒末、3秒初

时间间隔

指两时刻之间的间隔，在时间轴上用线段表示

例如：2秒内、第2秒内

位移和路程

位移

定义：初位置指向末位置的有向线段

矢量：既有大小又有方向

取决于：只取决于初末位置

路程

定义：物体运动轨迹的实际长度

标量：只有大小，没有方向

取决于：取决于实际运动路径

重要结论

  • 单向直线运动中，位移大小等于路程
  • 往返运动或曲线运动中，位移大小小于路程
  • 位移可以为零，路程只有在物体回到原点时为零

1.3 速度

速度的定义

速度是描述物体运动快慢和方向的物理量

速度是矢量，既有大小又有方向

平均速度和瞬时速度

平均速度

v = Δx/Δt

说明：

  • 粗略描述物体在某段时间内的运动快慢
  • 矢量，方向与位移方向相同
  • 计算：位移与所用时间的比值

瞬时速度

v = lim(Δt→0) Δx/Δt

说明：

  • 精确描述物体在某一时刻的运动快慢
  • 矢量，方向沿物体运动轨迹的切线方向
  • 大小等于瞬时速率

速度和速率的区别

物理量 速度 速率
类型 矢量 标量
定义 位移与时间的比值 路程与时间的比值
方向 有方向 无方向

1.4 加速度

加速度的定义

加速度是描述速度变化快慢的物理量

加速度是矢量，既有大小又有方向

加速度公式

a = Δv/Δt = (v₂ - v₁)/(t₂ - t₁)

单位：m/s²（米每二次方秒）

说明：

  • 加速度等于速度的变化量与所用时间的比值
  • 加速度的方向与速度变化量的方向相同
  • 加速度大表示速度变化快，不表示速度大

速度、速度变化量、加速度的区别

物理量 速度 v 速度变化量 Δv 加速度 a
物理意义 描述运动快慢 描述速度变化 描述速度变化快慢
定义 v = Δx/Δt Δv = v₂ - v₁ a = Δv/Δt
单位 m/s m/s m/s²

加速度方向的判断

  • 加速运动：加速度方向与速度方向相同
  • 减速运动：加速度方向与速度方向相反
  • 匀速运动：加速度为零

例题练习

例题 1：参考系的选择 【基础】

坐在行驶火车中的乘客，以火车为参考系，看到路边的树木向什么方向运动？

解答：

以火车为参考系，假设火车不动，则路边的树木相对于火车向后运动

答案：向后运动（与火车运动方向相反）

例题 2：时间和时刻 【基础】

关于时间和时刻，下列说法正确的是：

(A) 第2秒内是2秒的时间间隔

(B) 第2秒末是2秒的时间间隔

(C) 2秒内是2秒的时间间隔

(D) 2秒末是2秒的时间间隔

解答：

(A) 错误。第2秒内是1秒的时间间隔（第1秒末到第2秒末）

(B) 错误。第2秒末是一个时刻

(C) 正确。2秒内是从0时刻到2秒末，时间间隔为2秒

(D) 错误。2秒末是一个时刻

答案：C

例题 3：位移和路程 【基础】

某物体沿半径为R的圆周运动一周，则物体的位移和路程分别为多少？

解答：

位移：物体回到原点，初位置和末位置相同，位移为0

路程：物体沿圆周运动一周，路程为圆的周长 2πR

答案：位移为0，路程为2πR

例题 4：平均速度的计算 【基础】

某物体沿直线运动，前一半时间速度为4m/s，后一半时间速度为6m/s，求全程的平均速度

解答：

设总时间为2t

前一半时间位移：x₁ = 4t

后一半时间位移：x₂ = 6t

总位移：x = x₁ + x₂ = 4t + 6t = 10t

平均速度：v = x/(2t) = 10t/(2t) = 5m/s

答案：5m/s

例题 5：加速度的计算 【基础】

某汽车从静止开始加速，经过10s速度达到20m/s，求汽车的加速度

解答：

初速度：v₁ = 0

末速度：v₂ = 20m/s

时间：Δt = 10s

加速度：a = (v₂ - v₁)/Δt = (20 - 0)/10 = 2m/s²

答案：2m/s²

例题 6：加速度方向的判断 【中档】

汽车以20m/s的速度行驶，刹车后经5s停下，设刹车过程为匀减速运动，求加速度的大小和方向

解答：

初速度：v₁ = 20m/s

末速度：v₂ = 0

时间：Δt = 5s

速度变化量：Δv = v₂ - v₁ = 0 - 20 = -20m/s

加速度：a = Δv/Δt = -20/5 = -4m/s²

加速度为负值，说明加速度方向与速度方向相反

答案：加速度大小为4m/s²，方向与运动方向相反

例题 7：位移和路程的计算 【中档】

查看解答

解答：

路程：沿实际路径计算

s = 40m + 30m = 70m

位移：从起点指向终点的有向线段

根据勾股定理：x = √(40² + 30²) = √(1600 + 900) = √2500 = 50m

方向：东偏北某个角度（tanθ = 30/40 = 0.75，θ ≈ 37°）

答案：位移为50m，方向东偏北37°；路程为70m

例题 8：平均速度的多种计算 【中档】

某物体沿直线运动，前一半位移速度为4m/s，后一半位移速度为6m/s，求全程的平均速度

解答：

设总位移为2x

前一半位移时间：t₁ = x/4

后一半位移时间：t₂ = x/6

总时间：t = t₁ + t₂ = x/4 + x/6 = (3x + 2x)/12 = 5x/12

平均速度：v = 2x/t = 2x/(5x/12) = 2x × 12/(5x) = 24/5 = 4.8m/s

答案：4.8m/s

例题 9：2024年高考真题 【中档】

甲乙两车在同一直线上运动，甲车速度为10m/s，乙车速度为15m/s，若以甲车为参考系，则乙车的速度为多少？方向如何？

解答：

设甲车速度方向为正方向

v甲 = 10m/s，v乙 = 15m/s

以甲车为参考系，乙车的相对速度：

v乙对甲 = v乙 - v甲 = 15 - 10 = 5m/s

相对速度为正，说明方向与甲车运动方向相同

答案：5m/s，方向与甲车运动方向相同

例题 10：2023年高考真题 【中档】

物体从A点运动到B点，前一半时间平均速度为v₁，后一半时间平均速度为v₂，则全程平均速度为多少？

解答：

设总时间为2t

前一半时间位移：x₁ = v₁t

后一半时间位移：x₂ = v₂t

总位移：x = x₁ + x₂ = v₁t + v₂t = (v₁ + v₂)t

平均速度：v = x/(2t) = (v₁ + v₂)t/(2t) = (v₁ + v₂)/2

答案：(v₁ + v₂)/2
