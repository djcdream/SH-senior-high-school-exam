匀变速直线运动

定义

物体沿直线运动，且加速度恒定的运动叫做匀变速直线运动

加速度恒定：大小和方向都不变

分类

  • 匀加速直线运动：加速度方向与速度方向相同
  • 匀减速直线运动：加速度方向与速度方向相反

匀变速直线运动的公式

速度公式

v = v₀ + at

v：末速度，v₀：初速度，a：加速度，t：时间

位移公式

x = v₀t + ½at²

x：位移

速度位移公式

v² - v₀² = 2ax

重要推论

平均速度公式

v̄ = (v₀ + v)/2

中间时刻速度

v中 = v̄ = (v₀ + v)/2

中间位置速度

v中位置 = √[(v₀² + v²)/2]

自由落体运动

定义

物体只在重力作用下从静止开始下落的运动叫做自由落体运动

特点

  • 初速度 v₀ = 0
  • 加速度 a = g ≈ 9.8m/s²（重力加速度）
  • 只受重力作用

自由落体运动公式

v = gt
h = ½gt²
v² = 2gh

例题练习

例题 1：匀加速运动 【基础】

汽车从静止开始以2m/s²的加速度加速，求5秒后的速度和位移

解答：

v₀ = 0, a = 2m/s², t = 5s

速度：v = v₀ + at = 0 + 2×5 = 10m/s

位移：x = v₀t + ½at² = 0 + ½×2×25 = 25m

答案：速度为10m/s，位移为25m

例题 2：匀减速运动 【基础】

汽车以20m/s的速度刹车，加速度大小为4m/s²，求5秒后的速度和位移

解答：

先求停车时间：v = v₀ + at = 0

0 = 20 + (-4)t，t = 5s

恰好5秒停下

位移：x = v₀t + ½at² = 20×5 + ½×(-4)×25 = 100 - 50 = 50m

答案：速度为0，位移为50m

例题 3：自由落体运动 【基础】

物体自由下落2秒，求下落高度和落地速度（取g=10m/s²）

解答：

t = 2s, g = 10m/s²

高度：h = ½gt² = ½×10×4 = 20m

速度：v = gt = 10×2 = 20m/s

答案：下落高度20m，落地速度20m/s
