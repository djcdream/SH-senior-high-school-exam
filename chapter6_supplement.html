例题 35：圆的综合 【提高】

已知圆 C 过点 A(1, 3)、B(5, 1)，且圆心在直线 y = x 上，求圆的方程

解答：

设圆心 C(a, a)（因为圆心在 y = x 上）

由于圆过 A、B，所以 |CA| = |CB|

√[(1 - a)² + (3 - a)²] = √[(5 - a)² + (1 - a)²]

两边平方：(1 - a)² + (3 - a)² = (5 - a)² + (1 - a)²

(3 - a)² = (5 - a)²

9 - 6a + a² = 25 - 10a + a²

4a = 16

a = 4

所以圆心 C(4, 4)

半径 r = √[(1 - 4)² + (3 - 4)²] = √(9 + 1) = √10

圆的方程：(x - 4)² + (y - 4)² = 10

答案：(x - 4)² + (y - 4)² = 10

3x/25 + (12/5)y/16 = 1

两边乘以100：12x + 15y = 100

化简：4x + 5y = 100/3

例题 38：两圆位置关系 【提高】

圆 C₁：x² + y² = 1 与圆 C₂：(x - 3)² + (y - 4)² = 16 的位置关系是

解答：

圆 C₁：圆心 O₁(0, 0)，半径 r₁ = 1

圆 C₂：圆心 O₂(3, 4)，半径 r₂ = 4

半径和：r₁ + r₂ = 1 + 4 = 5

例题 39：圆与椭圆相切 【提高】

圆 x² + y² = 1 与椭圆 x²/4 + y²/1 = 1 在第一象限相切，求切点坐标

解答：

由于相切，圆心与椭圆中心连线通过切点

椭圆长轴在 x 轴上，a = 2, b = 1

第一象限的切点在椭圆上：(2cosθ, sinθ)

由于切点也在圆上：(2cosθ, sinθ) 满足 x² + y² = 1

(2cosθ)² + (sinθ)² = 1

4cos²θ + sin²θ = 1

4cos²θ + (1 - cos²θ) = 1

3cos²θ = 0

cosθ = 0

θ = π/2

切点：(2cos(π/2), sin(π/2)) = (0, 1)

答案：(0, 1)

例题 40：直线与椭圆相交 【提高】

直线 y = x + 1 截椭圆 x²/4 + y² = 1 所得的弦长是

解答：

将直线代入椭圆方程：x²/4 + (x + 1)² = 1

x²/4 + x² + 2x + 1 = 1

两边乘以4：x² + 4x² + 8x + 4 = 4

5x² + 8x = 0

x(5x + 8) = 0

x = 0 或 x = -8/5

当 x = 0 时，y = 1

当 x = -8/5 时，y = -8/5 + 1 = -3/5

弦长：√[(-8/5 - 0)² + (-3/5 - 1)²] = √(64/25 + 64/25) = √(128/25) = 8√2/5

答案：8√2/5

例题 41：圆与抛物线相交 【提高】

圆 x² + y² = 4 与抛物线 y² = 4x 的交点个数是

解答：

抛物线：y² = 4x

代入圆方程：x² + 4x = 4

x² + 4x - 4 = 0

x = [-4 ± √(16 + 16)]/2 = [-4 ± √32]/2 = -2 ± 2√2

x₁ = -2 - 2√2 < 0（舍去，因为抛物线 x ≥ 0）

x₂ = -2 + 2√2 > 0

只有 x₂ 有效

当 x = -2 + 2√2 时，y² = 4(-2 + 2√2)

= -8 + 8√2

由于 -8 + 8√2 ≈ -8 + 11.31 = 3.31 > 0，所以 y 有两个值

因此有 2 个交点

答案：2个

例题 42：椭圆焦点弦 【提高】

例题 43：双曲线渐近线夹角 【提高】

双曲线 x²/9 - y²/16 = 1 的两条渐近线的夹角是

解答：

双曲线：a = 3, b = 4

渐近线：y = ±(b/a)x = ±(4/3)x

设两条渐近线的斜率分别为 k₁ = 4/3，k₂ = -4/3

夹角 α 满足：tanα = |(k₂ - k₁)/(1 + k₁k₂)|

= |(-4/3 - 4/3)/(1 + (-16/9))|

= |(-8/3)/(-7/9)| = |-8/3 × (-9/7)| = |24/7| = 24/7

α = arctan(24/7)

答案：arctan(24/7)

例题 44：抛物线动点 【提高】

抛物线 y² = 4x 上的一点 P 到焦点 F 的距离等于到准线 l 的距离，求点 P 的坐标

解答：

抛物线：y² = 4x

2p = 4，p = 2

焦点：F(p/2, 0) = (1, 0)

准线：x = -p/2 = -1

设点 P(x, y)

点 P 到焦点 F 的距离：d₁ = √[(x - 1)² + y²]

点 P 到准线的距离：d₂ = |x + 1|

由于 d₁ = d₂，所以 √[(x - 1)² + y²] = |x + 1|

平方：(x - 1)² + y² = (x + 1)²

x² - 2x + 1 + y² = x² + 2x + 1

y² = 4x

这与抛物线方程一致

由于点 P 在抛物线上，所以满足这个条件

点 P 的坐标：(t², 2t)，其中 t 为参数

答案：(t², 2t)，t为参数

例题 45：椭圆面积最大 【提高】

点 P 在椭圆 x²/4 + y²/3 = 1 上移动，求 △POA 面积的最大值（其中 O 为原点，A 为右顶点）

解答：

椭圆：a = 2, b = √3

右顶点：A(2, 0)

椭圆上点：P(2cosθ, √3sinθ)

△POA 的面积：S = (1/2)×|OA|×|y_P| = (1/2)×2×|√3sinθ| = √3|sinθ|

当 sinθ = ±1 时，面积最大

S_max = √3

此时 θ = π/2 或 θ = 3π/2

点 P：(0, √3) 或 (0, -√3)

答案：最大值√3

例题 46：圆与直线最值 【提高】

圆 x² + y² = 4 上的动点 P 到直线 y = x + 2 的距离的最大值是

例题 49：抛物线范围 【提高】

抛物线 y² = 4x 上的点 P 到点 Q(2, 2) 的距离的最小值是

解答：

设抛物线上点 P(t², 2t)

距离平方：d² = (t² - 2)² + (2t - 2)²

= (t² - 2)² + 4(t - 1)²

设 u = t - 1

d² = ((u + 1)² - 3)² + 4u²

= (u² + 2u - 2)² + 4u²

= u⁴ + 4u³ + 4u² - 4u² - 8u + 4 + 4u²

= u⁴ + 4u³ + 4u² - 8u + 4

求导：d'(u²) = 2u³ + 12u² + 8u - 8

令 d'(u²) = 0：2u³ + 12u² + 8u - 8 = 0

u³ + 6u² + 4u - 4 = 0

(u - 1)(u² + 7u + 4) = 0

u = 1

t = u + 1 = 2

点 P(4, 4)

d_min = √[(4 - 2)² + (4 - 2)²] = √(4 + 4) = √8 = 2√2

答案：2√2

例题 50：圆锥曲线综合 【提高】

已知椭圆 C₁ 和双曲线 C₂ 有相同的焦点 F₁(±√3, 0) 和 F₂(±√3, 0)，且椭圆的离心率 e₁ = √3/2，双曲线的离心率 e₂ = 2/√3，则椭圆 C₁ 和双曲线 C₂ 的标准方程分别是

解答：

对于椭圆：e₁ = c/a = √3/2

c = √3，a = c/e₁ = √3 × 2/√3 = 2

c² = a² - b²

3 = 4 - b²

b² = 1，b = 1

椭圆方程：x²/4 + y²/1 = 1


对于双曲线：e₂ = c/a = 2/√3

c = √3，a = c/e₂ = √3 × √3/2 = 3/2

c² = a² + b²

3 = 9/4 + b²

b² = 3 - 9/4 = 3/4，b = √3/2

双曲线方程：x²/(9/4) - y²/(3/4) = 1

即：4x²/9 - 4y²/3 = 1

答案：x²/4 + y²/1 = 1 和 4x²/9 - 4y²/3 = 1
