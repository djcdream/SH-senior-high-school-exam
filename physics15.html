高中物理

第十五章 交变电流

← 返回物理列表 知识点1 知识点2 例题练习

知识点1

基本概念

交变电流：大小和方向随时间做周期性变化的电流

正弦交变电流：i = I_m·sin(ωt)，其中I_m是峰值，ω是角频率

有效值：I = I_m/√2，表示交变电流的等效直流电流

周期：T = 2π/ω，单位是秒(s)

知识点2

重要规律

理想变压器：原副线圈电压之比等于匝数之比：U₁/U₂ = n₁/n₂

变压器功率：P₁ = P₂，原线圈输入功率等于副线圈输出功率

远距离输电：采用高压输电，减小输电电流，减小线路损耗

三相交流电：三个相位相差120°的交变电流，供电更稳定

例题练习

例题 1：【2024·新课标卷】 有效值 【基础】

一正弦交变电流的峰值为10A。求该交变电流的有效值和通过电阻R的电功率。

解答：

（1）有效值：I = I_m/√2 = 10/√2 ≈ 7.07A

（2）电功率：P = I²R = (10/√2)²R = 50R

（3）也可以用峰值计算：P = ½I_m²R = ½×10²R = 50R

知识点标签：交变电流、有效值、电功率

答案：I = 7.07A，P = 50R

例题 2：【2023·浙江卷】 理想变压器 【中档】

一理想变压器，原线圈匝数为1000，副线圈匝数为200，原线圈电压为220V，副线圈接一电阻R。求副线圈电压和电流。

解答：

（1）根据理想变压器电压关系：U₁/U₂ = n₁/n₂

（2）副线圈电压：U₂ = U₁·n₂/n₁ = 220×200/1000 = 44V

（3）根据功率关系：U₁I₁ = U₂I₂

（4）副线圈电流：I₂ = U₂/R = 44/R

（5）原线圈电流：I₁ = U₂I₂/U₁ = (44/R)×44/220 = 8.8/R

知识点标签：理想变压器、匝数比、功率关系

答案：U₂ = 44V，I₂ = 44/R

例题 3：【2022·新课标卷】 交变电流的表达式 【基础】

一交变电流的周期为T，有效值为I。写出该交变电流的瞬时值表达式。

解答：

（1）角频率：ω = 2π/T

（2）峰值：I_m = √2·I

（3）瞬时值表达式：i = I_m·sin(ωt) = √2·I·sin(2πt/T)

（4）也可以写成余弦形式：i = I_m·cos(ωt)

（5）有效值I是电流的方均根值

知识点标签：交变电流、瞬时值、周期、有效值

答案：i = √2·I·sin(2πt/T)

例题 4：【2024·河北卷】 交变电流的峰值 【中档】

一交变电流的有效值为I，通过电阻R，在一个周期内产生的热量为Q。求该交变电流的峰值。

解答：

（1）交变电流在一个周期内产生的热量：Q = I²RT

（2）根据有效值定义：I²R = ½I_m²R（因为i²的平均值为½I_m²）

（3）所以：I = I_m/√2

（4）热量：Q = (I_m/√2)²RT = ½I_m²RT

（5）峰值：I_m = √(2Q/(RT))

（6）峰值是有效值的√2倍

知识点标签：交变电流、有效值、峰值、热量

答案：I_m = √(2Q/(RT))

例题 5：【2023·全国卷】 变压器的效率 【中档】

一变压器的原线圈电压为U₁，电流为I₁；副线圈电压为U₂，电流为I₂。若变压器的效率为η，求原线圈的输入功率和副线圈的输出功率。

解答：

（1）副线圈输出功率：P₂ = U₂I₂

（2）原线圈输入功率：P₁ = U₁I₁

（3）效率：η = P₂/P₁ = (U₂I₂)/(U₁I₁)

（4）对于理想变压器，η = 100%；对于实际变压器，η < 100%

（5）功率损失：P损 = P₁ - P₂ = (1 - η)P₁

（6）效率越高，功率损失越小

知识点标签：变压器、效率、功率

答案：P₂ = U₂I₂，P₁ = U₂I₂/η

例题 6：【2024·辽宁卷】 远距离输电 【中档】

一发电站输出功率为P，输出电压为U₁，通过电阻为R的输电线向用户供电。若将输出电压提高到U₂，其他条件不变，求输电线上的功率损失之比。

解答：

（1）初始情况下，输电线上的电流：I₁ = P/U₁

（2）初始功率损失：P损₁ = I₁²R = (P/U₁)²R

（3）提高电压后，输电线上的电流：I₂ = P/U₂

（4）提高电压后功率损失：P损₂ = I₂²R = (P/U₂)²R

（5）功率损失之比：P损₁/P损₂ = (P/U₁)²R/[(P/U₂)²R] = U₂²/U₁²

（6）若U₂ = kU₁（电压提高到k倍），则P损₂ = P损₁/k²

（7）电压越高，输电损失越小

知识点标签：远距离输电、功率损失、电压升高

答案：P损₁/P损₂ = U₂²/U₁²

例题 7：【2022·浙江卷】 三相交流电 【中档】

三相交流电的三个相电压分别为u₁ = U_m·sin(ωt)，u₂ = U_m·sin(ωt - 120°)，u₃ = U_m·sin(ωt + 120°)。求三相电压之和。

解答：

（1）三相电压之和：u₁ + u₂ + u₃

（2）= U_m[sin(ωt) + sin(ωt - 120°) + sin(ωt + 120°)]

（3）利用三角函数公式：sin(α - β) + sin(α + β) = 2sinα·cosβ

（4）= U_m[sin(ωt) + 2sin(ωt)·cos120°]

（5）由于cos120° = -1/2，所以：= U_m[sin(ωt) + 2sin(ωt)(-1/2)] = U_m[sin(ωt) - sin(ωt)] = 0

（6）三相电压之和为零，这是三相交流电的特点

知识点标签：三相交流电、相位差、三角函数

答案：u₁ + u₂ + u₃ = 0

例题 8：【2023·河北卷】 交变电流的功率 【中档】

一电阻R接在电压为u = U_m·sin(ωt)的交变电压上。求该电阻在一个周期内消耗的电能和平均功率。

解答：

（1）瞬时功率：p = u²/R = [U_m·sin(ωt)]²/R = U_m²sin²(ωt)/R

（2）平均功率：P平均 = (1/T)∫[0到T] p·dt = (1/T)∫[0到T] U_m²sin²(ωt)/R·dt

（3）由于sin²(ωt)的平均值为1/2，所以：P平均 = U_m²/(2R)

（4）一个周期内消耗的电能：W = P平均·T = U_m²T/(2R)

（5）也可以用有效值计算：U = U_m/√2，所以P平均 = U²/R

（6）平均功率等于电压有效值的平方除以电阻

知识点标签：交变电流、瞬时功率、平均功率

答案：P平均 = U_m²/(2R)，W = U_m²T/(2R)

例题 9：【2024·新课标Ⅱ卷】 交变电流的频率 【基础】

一交变电流的瞬时值表达式为i = 10·sin(100πt)A。求该交变电流的频率、周期和有效值。

解答：

（1）瞬时值表达式：i = I_m·sin(ωt) = 10·sin(100πt)A

（2）峰值：I_m = 10A

（3）角频率：ω = 100π rad/s

（4）频率：f = ω/(2π) = 100π/(2π) = 50Hz

（5）周期：T = 1/f = 1/50 = 0.02s

（6）有效值：I = I_m/√2 = 10/√2 ≈ 7.07A

（7）这是工频交流电的标准频率

知识点标签：交变电流、频率、周期、有效值

答案：f = 50Hz，T = 0.02s，I = 7.07A

例题 10：【2022·全国卷】 变压器的匝数比 【中档】

一理想变压器，原线圈匝数为n₁，副线圈匝数为n₂。若原线圈电压为U₁，副线圈电压为U₂，求匝数比n₁:n₂。

解答：

（1）根据理想变压器电压关系：U₁/U₂ = n₁/n₂

（2）所以匝数比：n₁:n₂ = U₁:U₂

（3）若U₁ > U₂，则为降压变压器；若U₁ < U₂，则为升压变压器

（4）匝数比决定了变压器的升降压特性

（5）对于降压变压器，n₁ > n₂；对于升压变压器，n₁ < n₂

知识点标签：理想变压器、匝数比、升降压

答案：n₁:n₂ = U₁:U₂

例题 11：【2023·江苏卷】 交变电流的平均值 【中档】

一正弦交变电流在半个周期内的平均值为I_half。求该交变电流的有效值与平均值的比值。

解答：

（1）半个周期内的平均值：I_half = (2/π)I_m

（2）有效值：I = I_m/√2

（3）比值：I/I_half = (I_m/√2)/[(2/π)I_m] = π/(2√2) ≈ 1.11

（4）有效值大于半个周期内的平均值

（5）平均值的定义取决于选取的时间区间

（6）对于正弦交变电流，平均值为零（正负抵消），半周期平均值为2I_m/π

知识点标签：交变电流、平均值、有效值

答案：I/I_half = π/(2√2) ≈ 1.11

例题 12：【2024·江西卷】 交变电流的相位 【中档】

一交变电流的瞬时值表达式为i = I_m·sin(ωt + φ)，另一交变电流的瞬时值表达式为i' = I_m·sin(ωt)。求两交变电流的相位差和它们合成的瞬时值。

解答：

（1）第一个电流的相位：φ₁ = ωt + φ

（2）第二个电流的相位：φ₂ = ωt

（3）相位差：Δφ = φ₁ - φ₂ = φ

（4）合成瞬时值：i合 = i + i' = I_m·sin(ωt + φ) + I_m·sin(ωt)

（5）利用三角函数公式：sin(α + β) + sinα = 2sin(α + β/2)·cos(β/2)

（6）= 2I_m·sin(ωt + φ/2)·cos(φ/2)

（7）合成电流的峰值：I合_m = 2I_m·|cos(φ/2)|

（8）合成电流的相位：ωt + φ/2

知识点标签：交变电流、相位差、合成

答案：Δφ = φ，i合 = 2I_m·sin(ωt + φ/2)·cos(φ/2)

例题 13：【2022·河北卷】 交变电流的热效应 【中档】

一电阻R，分别通过峰值相等的直流电流和正弦交变电流。若通电时间相同，求两种情况下产生的热量之比。

解答：

（1）直流电流：I直流 = I_m（峰值相等）

（2）直流产生的热量：Q直流 = I_m²RT

（3）交变电流的有效值：I交流 = I_m/√2

（4）交变产生的热量：Q交流 = I交流²RT = (I_m/√2)²RT = ½I_m²RT

（5）热量之比：Q直流/Q交流 = I_m²RT/[½I_m²RT] = 2

（6）直流产生的热量是交变的两倍（峰值相等时）

（7）若直流等于交变的有效值，则产生的热量相同

知识点标签：交变电流、直流、热量、有效值

答案：Q直流/Q交流 = 2

例题 14：【2023·新课标卷】 交变电流的峰值 【中档】

一交变电流通过电阻R，在一个周期内产生的热量为Q。若要保持产生的热量不变，但将电流的峰值减半，需要如何改变电阻R？

解答：

（1）初始情况下，热量：Q = I_m²RT/2

（2）峰值减半后：I'_m = I_m/2

（3）峰值减半后产生的热量：Q' = (I'_m)²RT'/2 = (I_m/2)²RT'/2 = I_m²RT'/8

（4）由于Q' = Q，所以：I_m²RT'/8 = I_m²RT/2

（5）解得：T' = 4T（周期不变）或R' = 4R

（6）所以需要将电阻增大为原来的4倍

（7）峰值减半，要产生相同的热量，需要增大电阻

知识点标签：交变电流、峰值、热量、电阻

答案：R' = 4R

例题 15：【2024·全国卷】 交变电流的综合 【高档】

一理想变压器，原线圈匝数为n₁，副线圈匝数为n₂，原线圈电压为u₁ = U_m·sin(ωt)，副线圈接电阻R。求副线圈电压的有效值和电流的有效值，以及原线圈的电流。

解答：

（1）原线圈电压的有效值：U₁ = U_m/√2

（2）根据理想变压器电压关系：U₁/U₂ = n₁/n₂

（3）副线圈电压的有效值：U₂ = U₁·n₂/n₁ = (U_m/√2)·n₂/n₁

（4）副线圈电流的有效值：I₂ = U₂/R = (U_m/√2)·n₂/(n₁R)

（5）根据功率关系：U₁I₁ = U₂I₂

（6）原线圈电流的有效值：I₁ = U₂I₂/U₁ = I₂

（7）所以：I₁ = (U_m/√2)·n₂/(n₁R)

（8）原线圈电流的有效值等于副线圈电流的有效值乘以匝数比的倒数

知识点标签：理想变压器、有效值、功率关系、电流

答案：U₂ = (U_m/√2)·n₂/n₁，I₂ = (U_m/√2)·n₂/(n₁R)，I₁ = (U_m/√2)·n₂/(n₁R)
